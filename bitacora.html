PROYECTO FINAL DE FUNDAMENTOS DE SOFTWARE

Abstracción de Datos.

La abstracción de datos permite que los desarrolladores y los administradores muestren solo los datos necesarios a los usuarios front-end, ya que no necesitan tener acceso a un silo de datos completo.

Acceso Condicional.

El acceso condicional es un marco de seguridad que garantiza que solo los usuarios autorizados puedan acceder a recursos confidenciales, equilibrando una seguridad sólida con la experiencia del usuario.

Acceso Remoto.

El acceso remoto es la capacidad de los usuarios de conectarse desde cualquier sitio a un dispositivo o una red.

Acceso Remoto Seguro.

El acceso remoto seguro es la capacidad de conectarse a una red, ordenador o sistema desde una ubicación remota garantizando al mismo tiempo la privacidad de los datos, la autenticación del usuario y la protección contra amenazas cibernéticas.

Access Point (Punto de Acceso).

De forma técnica, los Access Point o Wireless Access Point son dispositivos que se utilizan para realizar una conexión inalámbrica entre equipos y crear una red inalámbrica externa para interconectar dispositivos móviles o tarjetas de red inalámbricas.

Aceleración por Hardware.

La aceleración de hardware se define como el proceso que transfiere parte del trabajo de procesamiento de la aplicación desde el software que se ejecuta en la unidad central de procesamiento (CPU) a un recurso de hardware inactivo.

Acelerómetro.

El acelerómetro es un sensor que mide aceleración en una o varias direcciones, usado en móviles, consolas y dispositivos IoT.

ACL (Access Control List).

Una lista de control de acceso, a menudo abreviada como ACL, es una lista que puede definirse como un conjunto de reglas. Estas reglas están diseñadas para proporcionar un cierto nivel de control sobre el acceso a una red o sistema.

Acoplamiento de Software.

El acoplamiento de software se refiere al grado de interdependencia entre los módulos o componentes de un sistema.

Active Directory (AD).

Active Directory (AD) es una base de datos y un conjunto de servicios que conectan a los usuarios con los recursos de red que necesitan para realizar su trabajo.

Ad Hoc Testing.

Las pruebas ad hoc se distinguen por ser un enfoque informal para las pruebas de software, que se distingue por la ausencia de trabajo preparatorio o planes predefinidos.

Adaptadores Wi-Fi.

Un adaptador Wi-Fi es un dispositivo que permite conectar un ordenador a una red inalámbrica y le permite tener acceso de manera remota.

Adaptive Streaming.

El Adaptive Streaming es una tecnología que ajusta automáticamente la calidad del video en tiempo real para adaptarse a las condiciones cambiantes de la conexión a Internet y las capacidades del dispositivo del espectador.

Address Resolution Protocol (ARP).

El protocolo de resolución de direcciones (Address Resolution Protocol, ARP) es un protocolo o procedimiento que conecta una dirección de protocolo de Internet (IP) en constante cambio a una dirección de máquina física fija.

Adobe photoshop.

Photoshop es un programa de edición de imágenes desarrollado por Adobe. Se utiliza principalmente para retocar fotografías, crear gráficos y diseñar contenido visual de manera profesional y detallada.

Adware.

El adware es un programa que muestra anuncios emergentes no deseados (y a veces irritantes) que pueden aparecer en tu computadora o dispositivo móvil.

Agile Coach.

Un Agile Coach es un profesional especializado en facilitar y promover la adopción de metodologías ágiles dentro de una organización.

Algoritmo.

Un algoritmo es en realidad un procedimiento por etapas. Es un conjunto de reglas que hay que seguir para realizar una tarea o resolver un problema.

Algoritmo A*.

El algoritmo A* es un algoritmo potente y ampliamente utilizado para recorrer grafos y encontrar caminos. Encuentra el camino más corto entre un nodo inicial y un nodo objetivo en un grafo ponderado.

Algoritmo de Compresión.

Los algoritmos de compresión de datos toman un archivo y lo codifican en un estado comprimido.

Algoritmo de Encriptación.

Un algoritmo de encriptación es el método utilizado para transformar los datos en texto encriptado.

Algoritmo de Hashing.

Un algoritmo de hashing es una función matemática que toma un conjunto de datos de entrada de cualquier tamaño y lo transforma en una cadena alfanumérica de longitud fija, llamada "hash" o "huella digital".

Algoritmo de Ordenamiento.

Los algoritmos de ordenación permiten ordenar una lista de elementos para que sea más utilizable.

Algoritmo Genético.

Un algoritmo genético es una técnica de búsqueda evolutiva utilizada en informática para encontrar soluciones verdaderas o aproximadas a problemas de optimización.

Alias en Programación.

Un aliaImg/1-100/IMG24_Algoritmo Genético.pngs es un nombre o etiqueta que usas como acceso directo en sistemas informáticos para simplificar comandos largos, nombres complicados o rutas frecuentes.

Alta Disponibilidad (HA).

Alta disponibilidad (HA) es un término que se refiere a la capacidad de un sistema de ser accesible y confiable cerca del 100% del tiempo.

Almacenamiento en la Nube.

El almacenamiento en la nube es un modelo de computación en la nube que permite almacenar datos y archivos en Internet a través de un proveedor de computación en la nube.

Altavoces.

Los altavoces son dispositivos que permiten la amplificación del sonido.

Amazon Web Services (AWS).

Amazon Web Services es un proveedor de servicios en la nube.

Análisis de Datos.

El análisis de datos es el proceso sistemático de examinar, limpiar, transformar e interpretar datos sin procesar para obtener información valiosa, identificar patrones y tendencias, y así facilitar la toma de decisiones informadas y estratégicas en diversos campos.

Análisis de Sentimiento.

El análisis de sentimiento, también conocido como minería de opiniones, es el proceso de analizar grandes volúmenes de texto para determinar si expresa un sentimiento positivo, un sentimiento negativo o un sentimiento neutro.

Análisis Forense Digital.

El análisis forense digital es el proceso de recopilar y analizar pruebas digitales de forma que se mantenga su integridad y admisibilidad en los tribunales.

Análisis Predictivo.

El análImg/1-100/IMG32_Análisis Forense Digital.jpgisis predictivo es una rama de análisis avanzados que hace predicciones sobre resultados futuros utilizando datos históricos combinados con modelado estadístico, técnicas de extracción de datos y aprendizaje automático.

Ancho de Banda.

El ancho de banda se mide como la cantidad de datos que se pueden transferir entre dos puntos de una red en un tiempo específico. Normalmente, el ancho de banda se mide en bits por segundo (bps) y se expresa como una tasa de bits.

Android.

Android es un sistema operativo para móviles diseñado por la compañía estadounidense Google, basado en Linux.

Android Auto.

Android Auto ofrece una experiencia de apps optimizadas para conductores a usuarios que cuentan con teléfonos Android con la app de Android Auto y un vehículo o un sistema estéreo compatible.

Android Studio.

Android Studio es el entorno de desarrollo integrado (IDE) oficial que se usa en el desarrollo de apps para Android.

Angular.

Angular es una plataforma de desarrollo, construida sobre TypeScript. Es un framework basado en componentes para crear aplicaciones web escalables.

Anonimización de Datos.

En ciencia de datos, la anonimización de datos se refiere al proceso de modificar un conjunto de datos de forma que resulte imposible o muy difícil identificar a los individuos basándose en los datos disponibles.

Antena 5G.

Las antenas 5G son componentes clave de la tecnología de comunicación inalámbrica 5G y se utilizan para enviar y recibir señales inalámbricas, lo que permite que los dispositivos se conecten a las redes 5G.

Antivirus.

Originalmente, un programa antivirus era un software que detectaba y en ocasiones eliminaba virus informáticos de los dispositivos infectados, y por lo tanto también contribuía a detener la propagación del contenido malicioso.

API (Application Programming Interface).

Una API (application programming interface), o interfaz de programación de aplicaciones, es un conjunto de reglas o protocolos que permiten que las aplicaciones de software se comuniquen entre sí para intercambiar datos, características y funcionalidades.

API Gateway.

API gateway es una herramienta de API management que actúa como intermediario entre un cliente de API (la aplicación en el dispositivo de un usuario) y los servicios de backend (ubicados en un servidor).

API GraphQL.

GraphQL es un lenguaje de consulta de código abierto para API y un entorno de ejecución del lado del servidor.

API Management.

La gestión de API es el proceso escalable de creación, publicación y gestión de conexiones de interfaces de programación de aplicaciones (API), compartirlas, controlar el acceso, realizar un seguimiento de su uso y aplicar políticas de seguridad, dentro de un entorno empresarial y multinube.

API RESTful.

Una API REST es una interfaz de programación de aplicaciones (API) que se ajusta a los principios de diseño del estilo arquitectónico de transferencia de estado representacional (REST), un estilo utilizado para conectar sistemas de hipermedia distribuidos.

API SOAP.

SOAP es un protocolo utilizado para intercambiar información estructurada en el ámbito de los servicios web.

Aplicación Híbrida.

Es aquella que se puede descargar e instalar en múltiples plataformas móviles como Android e iOS. Está construida de manera que permite a los desarrolladores usar el mismo código para todos los sistemas operativos.

Aplicación Nativa.

Se llaman aplicaciones nativas debido a que se desarrollan para el sistema operativo nativo de cada dispositivo.

Aplicaciones de Comunicación.

Las aplicaciones de comunicación son herramientas tecnológicas diseñadas para el intercambio de información entre individuos, equipos y organizaciones, utilizando medios como texto, voz, video, datos y archivos.

Aplicaciones de Diseño.

Las aplicaciones de diseño son programas informáticos o herramientas de software que permiten a los usuarios crear y modificar contenido visual, como imágenes, ilustraciones, logotipos, maquetaciones de documentos y diseños de interfaz de usuario.

App Store.

App Store es la plataforma de descarga de aplicaciones móviles o apps de los dispositivos con sistema operativo iOS y Mac Os de la empresa Apple.

Apple CarPlay.

Apple CarPlay es la forma más cómoda y segura de usar un iPhone en el coche.

Apple ID.

Un Apple ID (también llamada “cuenta de Apple”) está compuesto de una dirección de correo y una contraseña. En muchas ubicaciones, puedes usar un número telefónico en lugar de una dirección de correo.

Apple Pay.

Apple Pay es la cartera digital de Apple. En las tiendas, Apple Pay te permite pagar sosteniendo tu teléfono sobre un datáfono que acepte esta plataforma de pago.

Apple Silicon.

Apple Silicon es una serie de procesadores de sistema en chip (SoC) y sistema en paquete (SiP) diseñados por Apple Inc., principalmente con arquitectura ARM.

AppleScript.

AppleScript es un lenguaje de scripting creado por Apple. Permite a los usuarios controlar directamente las aplicaciones de Macintosh que admiten scripts, así como partes del propio macOS.

Aplicación Web Progresiva (PWA).

Una app web progresiva (AWP) es una app web que usa mejoras progresivas para proporcionar a los usuarios una experiencia más confiable, utiliza capacidades nuevas para proporcionar una experiencia más integrada y se puede instalar.

Application Lifecycle Management (ALM).

La gestión del ciclo de vida de las aplicaciones (ALM) es la gestión de las aplicaciones de software desde la cuna hasta la tumba: desde la concepción y el desarrollo hasta la implementación, las revisiones, el mantenimiento y, finalmente, el retiro.

Application Server (Servidor de Aplicaciones).

Un servidor de aplicaciones, también conocido como servidor de aplicaciones o servidor de aplicaciones web, es un tipo de servidor que aloja software utilizado para aplicaciones comerciales.

Aprendizaje Automático (Machine Learning).

El aprendizaje automático es un subconjunto de la inteligencia artificial que permite que un sistema aprenda y mejore de forma autónoma con redes neuronales y aprendizaje profundo, sin necesidad de una programación explícita, a través del análisis de grandes cantidades de datos.

Aprendizaje por Refuerzo.

En el aprendizaje por refuerzo, un agente autónomo aprende a realizar una tarea por ensayo y error en ausencia de cualquier orientación por parte de un usuario humano.

Aprendizaje Supervisado.

El aprendizaje supervisado es una técnica de machine learning que utiliza conjuntos de datos de entrada y salida etiquetados por humanos para entrenar modelos de inteligencia artificial.

Aprendizaje No Supervisado.

El aprendizaje no supervisado, también conocido como machine learning no supervisado, utiliza algoritmos de machine learning (ML) para analizar y agrupar conjuntos de datos no etiquetados. Estos algoritmos descubren patrones ocultos o agrupaciones de datos sin necesidad de intervención humana.

Árbol de decisión.

Un árbol de decisión es un algoritmo de aprendizaje supervisado no paramétrico, que se utiliza tanto para tareas de clasificación como de regresión.

Arquitectura Cliente-Servidor.

Cliente-Servidor es uno de los estilos arquitectónicos distribuidos más conocidos, el cual está compuesto por dos componentes, el proveedor y el consumidor.

Arquitectura de Microservicios.

Una arquitectura de microservicios divide una aplicación en una serie de servicios implementables de forma independiente que se comunican a través de API.

Arquitectura de Red.

El concepto de arquitectura de red hace alusión al diseño de una red de comunicaciones.

Arquitectura de Software.

La arquitectura de software se refiere a la estructura y diseño de un sistema de software. Es una representación de alto nivel que define cómo los componentes del software interactúan entre sí, cómo se organizan y cómo cumplen con los requisitos funcionales y no funcionales del sistema.

Arquitectura Hexagonal.

La Arquitectura Hexagonal, o de Puertos y Adaptadores, es un patrón de diseño de software que busca aislar la lógica de negocio del núcleo de la aplicación, de las dependencias externas como bases de datos o interfaces de usuario.

Arquitectura Limpia.

La Arquitectura Limpia (Clean Architecture) es un principio de diseño de software que busca crear aplicaciones mantenibles, escalables y flexibles, mediante la separación estricta de responsabilidades en capas bien definidas.

Arquitectura Monolítica.

La arquitectura monolítica es un modelo tradicional de desarrollo de software en el que una única base de código ejecuta múltiples funciones empresariales. En un sistema operativo monolítico, el núcleo dirige toda la funcionalidad.

Arquitectura Orientada a Servicios (SOA).

La arquitectura orientada a los servicios (SOA) es un tipo de diseño de software que permite reutilizar sus elementos gracias a las interfaces de servicios que se comunican a través de una red con un lenguaje común.

Archivos Binarios.

Un archivo binario es un archivo cuyo contenido está en formato binario y consiste en una serie de bytes secuenciales, cada uno de ocho bits. El contenido debe ser interpretado por un programa o un procesador de hardware que comprenda de antemano exactamente cómo está formateado y cómo leer los datos.

Archivos de Configuración.

El software utiliza un archivo de configuración. Postulaciones para establecer parámetros y configuraciones iniciales que controlan ciertos aspectos del comportamiento del programa.

Archivos Temporales.

Un archivo temporal es un archivo creado por un programa para almacenar datos de manera provisional mientras este se ejecuta.

Archivos XML.

La abreviatura XML significa "Lenguaje de Marcado Extensible" . XML, similar a HTML, es un lenguaje de programación para estructurar y presentar datos en un formato de texto formateado.

Artificial General Intelligence (AGI).

La inteligencia artificial general (AGI) se refiere a la inteligencia hipotética de una máquina que posee la capacidad de comprender o aprender cualquier tarea intelectual que pueda realizar un ser humano. Es un tipo de inteligencia artificial (IA) cuyo objetivo es imitar las habilidades cognitivas del cerebro humano.

Archivo ASCII.

Un archivo ASCII es un archivo simple Archivo de texto que codifica caracteres utilizando el Código estándar americano para el intercambio de información (ASCII)Este sistema de codificación representa el texto asignando un valor numérico, que va de 0 a 127, a cada carácter, símbolo o comando de control.

Archivo CSV (Comma Separated Values).

Un archivo CSV (Comma-Separated Values o Valores Separados por Comas) es un formato de texto plano que almacena datos tabulares, donde cada línea representa un registro (fila) y los campos (columnas) de ese registro se separan mediante comas u otros delimitadores.

Archivo JSON.

Los archivos JSON son una de las principales herramientas que tenemos en el desarrollo actual para compartir datos entre sistemas o entre las distintas capas de una aplicación web.

Archivo Log.

Un archivo log es un registro detallado y cronológico de eventos, actividades y operaciones de un sistema informático, aplicación o dispositivo.

Archivo PDF (Portable Document Format).

PDF es una abreviatura que significa formato de documento portátil (Portable Document Format, en inglés). Es un formato versátil que creamos en Adobe y que ofrece una forma sencilla y segura de presentar e intercambiar documentos con independencia del software, el hardware o el sistema operativo que utilice quien los consulte.

Argumento en Programación.

Un argumento es un valor que se pasa a una función cuando se llama. Estos valores permiten que la función realice cálculos o procesos personalizados según los datos proporcionados.

Arduino.

Arduino es una plataforma de creación de electrónica de código abierto, la cual está basada en hardware y software libre, flexible y fácil de utilizar para los creadores y desarrolladores.

Array (Arreglo).

Los arrays son objetos similares a una lista cuyo prototipo proporciona métodos para efectuar operaciones de recorrido y de mutación. Tanto la longitud como el tipo de los elementos de un array son variables.

Asistente Virtual.

Un asistente virtual es un profesional que ofrece servicios de administración y soporte a otro emprendedor o empresa en remoto. Facilita la gestión de las tareas diarias de un negocio empleando, para ello, apps y herramientas digitales y teniendo la posibilidad de trabajar en remoto.

Asset Digital.

Un digital asset es cualquier tipo de dato que existe en forma digital y que se posee, comparte y almacena en un sistema de digital asset management (DAM), en un dispositivo digital o en la red.

ASP.NET.

.NET es una plataforma para desarrolladores formada por herramientas, lenguajes de programación y bibliotecas para crear una gran variedad de aplicaciones.

Assembly Language.

Assembly language es un lenguaje de programación de bajo nivel que utiliza códigos mnemotécnicos para representar las instrucciones de la máquina.

ATA (Advanced Technology Attachment).

ATA (Advanced Technology Attachment) es una interfaz y protocolo estándar para la conexión de dispositivos de almacenamiento masivo, como discos duros y unidades ópticas, con una computadora.

Ataque DDoS (Distributed Denial of Service).

Un ataque de denegación de servicio distribuido (DDoS) es un intento malintencionado de interrumpir el tráfico normal de un servidor, servicio o red determinada, sobrecargando el objetivo o su infraestructura asociada con una avalancha de tráfico de Internet.

Ataque de Fuerza Bruta.

Un ataque de fuerza bruta es un método de piratería informática que utiliza pruebas y errores para descifrar contraseñas, credenciales de inicio de sesión y claves de cifrado.

Ataque de Ingeniería Social.

Los ataques de ingeniería social manipulan a las personas para que compartan información que no deberían compartir, descarguen software que no deberían descargar, visiten sitios web que no deberían visitar, envíen dinero a delincuentes o cometan otros errores que comprometan su seguridad personal u organizacional.

Ataque de Replay.

Un ataque de repetición ocurre cuando un ciberdelincuente intercepta una comunicación de red segura y la retrasa o reenvía fraudulentamente para engañar al receptor y que haga lo que el hacker desea.

Ataque Man-in-the-Middle (MITM).

Un ataque Mitm (man-in-the-middle) es un ciberataque en el que un pirata informático roba información confidencial al escuchar a escondidas las comunicaciones entre dos objetivos en línea, como un usuario y una aplicación web.

Ataque Phishing.

"Phishing" se refiere a un intento de robar información confidencial, normalmente nombres de usuario, contraseñas, números de tarjetas de crédito, información de cuentas bancarias u otros datos importantes para su uso o venta.

Ataque Ransomware.

Un ataque de ransomware es un ciberataque en el que un tipo de software malicioso, llamado ransomware, restringe el acceso a los datos o al sistema infectado y exige un rescate para restaurar dicho acceso.

Atomicidad en Bases de Datos.

La atomicidad es un concepto básico en los sistemas de bases de datos. Significa que una transacción debe tener éxito por completo o no ejecutarse en absoluto. Si algo falla a medio camino, todo se vuelve atrás como si nunca hubiera ocurrido.

Auditoría de Sistemas.

La auditoría de sistemas es aquel examen destinado a revisar y evaluar los controles y sistemas de informática existentes en la entidad, así como su utilización, eficiencia y seguridad.

Auditoría Informática.

Su objetivo principal es verificar que tu infraestructura tecnológica sea segura, eficiente, cumpla con las regulaciones y esté alineada con los objetivos del negocio.

Augmented Analytics.

La analítica aumentada es aquella que se "aumenta" con tecnologías de inteligencia artificial (IA), incluyendo aprendizaje automático y procesamiento del lenguaje natural (PLN).

Auriculares.

Los auriculares son un dispositivo que consta de uno o dos auriculares con una diadema para sujetarlos sobre las orejas y con un micrófono adjunto, utilizado para llamadas telefónicas u otras comunicaciones habladas a distancia.

Autenticación Biométrica.

La autenticación biométrica utiliza las características físicas de una persona (como los rasgos faciales, el escaneo del iris o las huellas dactilares) para verificar su identidad antes de conceder el acceso a datos o sistemas confidenciales.

Autenticación en Dos Pasos.

La autenticación en dos pasos, es una medida de seguridad que ofrecen prácticamente todas las plataformas y apps de servicios online y que es fundamental activar para proteger las cuentas online de accesos indebidos.

Autenticación Multifactor (MFA).

La autenticación multifactor (MFA) es un proceso de registro en varios pasos que requiere que los usuarios ingresen algo más de información que simplemente una contraseña.

AutoCAD.

AutoCAD es el programa por defecto que muchos arquitectos usan para diseñar bocetos, dibujos, planos, estructuras y piezas que deben cumplir con ciertos parámetros solicitados por los clientes.

AutoML (Automated Machine Learning).

El aprendizaje automático automatizado (AutoML) es la práctica de automatizar el desarrollo integral de modelos de aprendizaje automático (modelos de aprendizaje automático (ML)).

Automatización de Procesos Robóticos (RPA).

La automatización de procesos robóticos (RPA), también conocida como robótica de software, utiliza tecnologías de automatización inteligente para realizar tareas de oficina repetitivas de trabajadores humanos, como extraer datos, rellenar formularios, mover archivos y más.

Automatización de Pruebas de Software.

La automatización de pruebas de software es el proceso de usar herramientas y scripts para ejecutar automáticamente las pruebas de una aplicación, comparando los resultados obtenidos con los esperados para verificar su funcionamiento correcto.

Autorización.

La autorización es el proceso para dar a alguien la capacidad de acceder a un recurso.

Autocompletado.

El autocompletado es una función presente en muchos programas de software y aplicaciones que predice el resto de una palabra o frase mientras escribes. Este sistema acelera tu escritura y ayuda a reducir errores.

Autoescalado (Autoscaling).

El autoescalado, a veces denominado "escalado automático", es una característica del cloud computing que asigna automáticamente recursos informáticos en función de la demanda del sistema.

Avatar Digital.

Un avatar digital combina inteligencia artificial y otras tecnologías avanzadas para crear imágenes realistas y dinámicas de personas y otras figuras. Aunque algunos avatares digitales con IA solo repiten textos y hacen gestos, muchos pueden interactuar con personas en tiempo real.

Availability en Cloud Computing.

La disponibilidad se refiere al tiempo que se mantiene operativo un sistema o componente. Esto se mide en porcentajes, como ejemplo: si alojas en la nube una aplicación y el servidor presenta interrupciones por una falla eléctrica, la disponibilidad ya no sería del 100%.

AWS Lambda.

AWS Lambda puede ejecutar código sin aprovisionar ni administrar servidores. Lambda ejecuta el código en una infraestructura informática de alta disponibilidad y administra todos los recursos informáticos, incluidos el mantenimiento del servidor y del sistema operativo, el aprovisionamiento de capacidad, el escalado automático y el registro.

Azure Active Directory (Azure AD).

Azure Active Directory (AD) es un servicio de administración de identidades y acceso basado en la nube.

Azure DevOps.

Azure DevOps es una plataforma basada en la nube que proporciona herramientas integradas para los equipos de desarrollo de software. Incluye todo lo que necesita para planear el trabajo, colaborar en código, compilar aplicaciones, probar funcionalidad e implementar en producción.

BaaS (Backend as a Service).

Modelo de servicio en la nube que proporciona a los desarrolladores herramientas listas para gestionar bases de datos, autenticación, almacenamiento y notificaciones sin necesidad de configurar un backend desde cero.

Backbone de Red.

Es la parte central de una red de comunicaciones que proporciona el camino principal para el tráfico de datos y conecta diferentes segmentos o subredes.

Backdoor (Puerta Trasera).

Vulnerabilidad o acceso oculto a un sistema informático que permite a atacantes ingresar sin pasar por los mecanismos de autenticación normales. Puede ser creado de forma maliciosa o accidental.

Backup Completo.

Copia de seguridad que incluye todos los datos de un sistema, permitiendo una recuperación completa en caso de pérdida de información.

Backup Diferencial.

Respaldo que copia únicamente los datos modificados desde el último backup completo, reduciendo espacio y tiempo de almacenamiento.

Backup en la Nube.

Servicio que permite almacenar copias de seguridad en servidores remotos a través de internet, garantizando accesibilidad y protección frente a desastres locales.

Backup Incremental.

Copia que guarda solo los archivos modificados desde el último backup (ya sea completo o incremental), optimizando espacio y velocidad.

Backup Snapshot.

Copia de seguridad puntual de un sistema o volumen que captura su estado exacto en un momento específico. Es muy usada en entornos virtualizados y servicios en la nube.

Balanced Scorecard en TI.

Herramienta de gestión estratégica que mide el desempeño de tecnologías de la información mediante indicadores financieros, de procesos internos, innovación y satisfacción del cliente.

Banda Ancha.

Tecnología de transmisión de datos a alta velocidad que permite la conexión simultánea de múltiples servicios como voz, video e internet.

Bare Metal Server.

Servidor físico dedicado a un solo cliente o aplicación, sin capas de virtualización, lo que permite un mayor control, rendimiento y seguridad.

Barra de Navegación.

Elemento visual en aplicaciones o páginas web que organiza y facilita el acceso a las secciones principales del sitio.

Base de Conocimiento (Knowledge Base).

Repositorio centralizado donde se recopila información, procedimientos y soluciones para el soporte técnico y la toma de decisiones.

Base de Datos.

Conjunto organizado de datos que se almacenan electrónicamente y permiten acceso, gestión y actualización eficientes.

Base de Datos Distribuida.

Tipo de base de datos en la cual la información está repartida en múltiples ubicaciones físicas, pero se administra como un solo sistema.

Base de Datos Relacional.

Modelo de base de datos que organiza la información en tablas relacionadas mediante claves primarias y foráneas.

Baseline de Proyecto.

Conjunto aprobado de parámetros de alcance, tiempo y costo que sirve como referencia para medir el desempeño y evolución de un proyecto.

Bash (Bourne Again Shell).

Intérprete de comandos para sistemas Unix/Linux que permite ejecutar scripts, automatizar procesos y administrar sistemas de manera eficiente.

Batch File (Archivo por Lotes).

Archivo de texto que contiene una serie de comandos que son ejecutados secuencialmente por el sistema operativo, principalmente en Windows.

Batch Processing (Procesamiento por Lotes).

Método de ejecución de programas o tareas en bloques sin intervención manual durante su procesamiento.

Baud Rate (Velocidad en Baudios).

Medida de la cantidad de símbolos transmitidos por segundo en una señal de comunicación digital. Se utiliza en telecomunicaciones para evaluar la velocidad de transmisión.

Beacon en IoT.

Dispositivo que utiliza tecnología inalámbrica de corto alcance para emitir señales periódicas, facilitando la geolocalización y la comunicación contextual.

Behavioral Biometrics (Biometría de Comportamiento).

Tecnología de autenticación que analiza patrones de comportamiento humano, como escritura, uso del teclado o movimiento del ratón, para verificar identidades.

Benchmark de Rendimiento.

Prueba estandarizada que mide la eficiencia de hardware o software en comparación con otros sistemas similares, evaluando velocidad, capacidad y estabilidad.

Benchmarking Tecnológico.

Práctica de comparar tecnologías, procesos o sistemas de una organización frente a estándares de la industria para detectar mejoras.

Berkeley DB.

Sistema de base de datos de código abierto desarrollado por Oracle, diseñado para ser ligero, embebido y de alto rendimiento en aplicaciones que requieren gestión eficiente de datos.

BI Developer (Desarrollador BI).

Profesional especializado en diseñar, implementar y mantener soluciones de inteligencia de negocios, incluyendo integración de datos, creación de dashboards y análisis.

BI Publisher.

Herramienta de Oracle que permite diseñar, generar y distribuir informes en diferentes formatos (PDF, Excel, HTML) a partir de múltiples fuentes de datos.

BI Suite.

Conjunto de herramientas integradas de inteligencia de negocios que permiten análisis de datos, generación de reportes y visualización interactiva para empresas.

Big Data.

Conjunto de volúmenes masivos de datos que no pueden ser gestionados con sistemas tradicionales, requiriendo técnicas avanzadas de almacenamiento y análisis.

Big Data Analytics.

Conjunto de técnicas que permiten examinar grandes volúmenes de datos para descubrir patrones, correlaciones y apoyar la toma de decisiones.

BigQuery.

Servicio en la nube de Google que permite el análisis rápido de grandes volúmenes de datos mediante SQL y capacidades de machine learning.

Binary Search (Búsqueda Binaria).

Algoritmo eficiente que localiza un elemento en una lista ordenada dividiendo el conjunto en mitades sucesivas hasta encontrar la coincidencia.

BIND (Berkeley Internet Name Domain).

Software de código abierto que implementa el sistema de nombres de dominio (DNS), ampliamente utilizado en servidores para resolver nombres en direcciones IP.

BIOS Firmware Password.

Medida de seguridad que bloquea el acceso a la configuración del BIOS/UEFI y evita que usuarios no autorizados modifiquen la secuencia de arranque o ajustes críticos.

BIOS/UEFI.

Firmware que inicializa el hardware de un dispositivo al arrancar y carga el sistema operativo. UEFI es la evolución moderna del BIOS con más seguridad y compatibilidad.

Bit.

Unidad mínima de información digital que puede tomar el valor 0 o 1. Es la base del sistema binario en el que funcionan las computadoras y se utiliza para representar datos, ejecutar instrucciones y transmitir información.

Bitácora de Proyectos.

Documento o sistema que registra actividades, avances y cambios de un proyecto. Permite dar seguimiento, identificar retrasos y facilitar auditorías.

BitLocker.

Herramienta de cifrado de Microsoft que protege los datos en discos duros mediante encriptación completa, evitando accesos no autorizados en caso de pérdida o robo del dispositivo.

Bitmap.

Formato de imagen digital que representa gráficos mediante una matriz de píxeles. Su principal ventaja es la fidelidad visual, pero los archivos suelen ser de gran tamaño.

Bitrate.

Cantidad de bits transmitidos o procesados por segundo en una señal digital, comúnmente usada en video, audio y streaming para medir calidad y velocidad.

BITS (Background Intelligent Transfer Service).

Servicio de Windows que permite transferir archivos en segundo plano, con tolerancia a interrupciones. Es ampliamente usado por Microsoft Update y otras aplicaciones para optimizar el ancho de banda y garantizar la entrega de archivos.

Black Box Testing (Pruebas de Caja Negra).

Método de prueba de software en el que se evalúa la funcionalidad de un sistema sin considerar su código interno, enfocándose en entradas y salidas.

Black Hat Hacker.

Cibercriminal que utiliza sus conocimientos técnicos para explotar vulnerabilidades con fines maliciosos, como robo de datos, sabotaje o fraude.

Blacklisting (Lista Negra).

Técnica de ciberseguridad que bloquea direcciones IP, dominios, aplicaciones o usuarios sospechosos para evitar accesos no autorizados o propagación de malware.

Bloatware.

Conjunto de programas preinstalados en dispositivos que ocupan recursos innecesarios y pueden ralentizar el rendimiento. Muchos usuarios optan por desinstalarlos para optimizar el sistema.

Blockchain.

Tecnología de registro distribuido que permite almacenar transacciones en bloques encadenados de manera inmutable, descentralizada y transparente. Su aplicación más conocida es en criptomonedas, pero también se usa en contratos inteligentes, trazabilidad y ciberseguridad.

Blockchain Explorer.

Herramienta en línea que permite a los usuarios consultar transacciones, bloques y direcciones en una red blockchain. Facilita la transparencia y auditoría de operaciones en tiempo real.

Blockchain Fork.

Situación en la que una blockchain se divide en dos cadenas diferentes debido a cambios en las reglas de consenso o actualizaciones de software. Pueden ser “hard forks” (cambios incompatibles) o “soft forks” (compatibles).

Blockchain Híbrido.

Modelo que combina características de blockchain público y privado, permitiendo acceso abierto en algunos procesos y control restringido en otros, para equilibrar transparencia y seguridad.

Blockchain Privado.

Red blockchain cerrada en la que solo usuarios autorizados pueden validar transacciones. Suele aplicarse en entornos corporativos que requieren confidencialidad y eficiencia.

Blockchain Público.

Red blockchain abierta donde cualquier usuario puede participar en la validación y verificación de transacciones, garantizando descentralización y transparencia.

Blue Team.

Equipo de ciberseguridad enfocado en la defensa de sistemas informáticos. Su tarea principal es detectar, responder y mitigar amenazas para proteger la infraestructura tecnológica frente a ataques.

Bluetooth.

Estándar de comunicación inalámbrica de corto alcance que permite la transmisión de datos entre dispositivos electrónicos. Es usado en periféricos, IoT y comunicación móvil.

Boot Sector.

Sector inicial de un disco que contiene información esencial para arrancar el sistema operativo. Los virus de sector de arranque fueron comunes en los años 90.

Bootloader.

Programa que se ejecuta al encender un dispositivo y carga el sistema operativo en la memoria. Sin el bootloader, el hardware no podría iniciar el software necesario para funcionar.

Borland C++ Builder.

Entorno de desarrollo integrado (IDE) lanzado en los años 90 para programar aplicaciones en C++ con soporte visual y rápido desarrollo de interfaces gráficas.

Borrado Seguro de Datos.

Técnica de eliminación de información que sobrescribe múltiples veces los sectores de un disco, impidiendo la recuperación de los datos originales.

Bot de Chat.

Aplicación basada en inteligencia artificial o reglas predefinidas que simula conversaciones humanas para asistir usuarios en procesos como soporte técnico o atención al cliente.

Bot Detection (Detección de Bots).

Conjunto de técnicas y herramientas que identifican y bloquean el tráfico automatizado malicioso en aplicaciones web, como bots de spam, scrapers o ataques de fuerza bruta.

Botnet.

Red de dispositivos infectados con malware que son controlados de manera remota por ciberdelincuentes para ejecutar ataques como spam, DDoS o robo de datos.

Botón de Pánico Digital.

Función en aplicaciones de seguridad que, al activarse, envía alertas inmediatas o borra información crítica para proteger al usuario en situaciones de riesgo.

Bounce Rate (Tasa de Rebote).

Métrica de analítica web que mide el porcentaje de usuarios que abandonan un sitio tras visitar una sola página, indicando posibles problemas de contenido o usabilidad.

Branch en Git.

Rama de desarrollo independiente dentro de un repositorio Git, que permite trabajar en nuevas funcionalidades o correcciones sin afectar la rama principal hasta su integración.

Branding Digital.

Estrategia de gestión de marca en entornos digitales que busca posicionar productos o servicios mediante redes sociales, sitios web, SEO y marketing de contenidos.

Breadcrumbs (Migas de Pan).

Elemento de navegación en interfaces digitales que muestra la ruta de páginas visitadas o la jerarquía del sitio, ayudando al usuario a ubicarse y volver a secciones anteriores.

Bridge de Red.

Dispositivo que conecta dos segmentos de red y filtra el tráfico para optimizar el rendimiento. Funciona en la capa de enlace de datos del modelo OSI.

Broadband over Power Line (BPL).

Tecnología que transmite datos de internet a través de la red eléctrica, permitiendo acceso en áreas donde no llegan otros servicios de banda ancha.

Broadcast en Redes.

Método de transmisión en el que un mensaje se envía a todos los dispositivos de una red de manera simultánea. Puede ser útil para anuncios, pero genera tráfico adicional.

Browser Cache.

Almacenamiento temporal en el navegador web que guarda recursos como imágenes, scripts y páginas HTML, acelerando la carga de sitios visitados con frecuencia.

Buffer Overflow (Desbordamiento de Búfer).

Vulnerabilidad de seguridad que ocurre cuando un programa escribe más datos de los que un búfer puede almacenar, sobrescribiendo memoria adyacente. Puede ser explotada para ejecutar código malicioso.

Bug (Error de Software).

Fallo en el código de un programa que produce comportamientos inesperados o incorrectos. Los bugs pueden variar desde errores menores hasta vulnerabilidades críticas.

Build en Desarrollo de Software.

Proceso de compilación y ensamblaje del código fuente para generar una versión ejecutable o instalable de un programa.

Business Continuity Plan (BCP).

Estrategia que asegura la continuidad de las operaciones críticas de TI en caso de interrupciones, desastres naturales, fallos de infraestructura o ciberataques.

Business Analyst (BA).

Profesional que actúa como enlace entre las necesidades del negocio y las soluciones tecnológicas, identificando requisitos, documentando procesos y apoyando en la gestión de proyectos.

Business Intelligence (BI).

Conjunto de tecnologías, procesos y metodologías que convierten datos en información útil para la toma de decisiones empresariales. Incluye análisis, reportes y visualización de datos.

Business Model Canvas (BMC).

Herramienta visual de gestión que describe y analiza modelos de negocio mediante nueve bloques clave: propuesta de valor, clientes, canales, ingresos, costos, recursos, actividades, alianzas y relaciones.

Business Process Management (BPM).

Disciplina que combina métodos, herramientas y tecnologías para diseñar, analizar, ejecutar y optimizar procesos de negocio, mejorando eficiencia y productividad.

Business Rules Engine (BRE).

Software que permite definir, ejecutar y administrar reglas de negocio de forma flexible, separándolas del código para facilitar cambios rápidos en los procesos.

Business-to-Business (B2B).

Modelo comercial en el que una empresa ofrece productos o servicios directamente a otra empresa, en lugar de al consumidor final. Es común en sectores como software corporativo, proveedores de insumos industriales y servicios en la nube. Su enfoque está en relaciones a largo plazo, volumen de ventas y personalización de soluciones.

Business-to-Consumer (B2C).

Modelo de negocio en el que una empresa vende productos o servicios directamente al consumidor final. Es común en el comercio electrónico, aplicaciones móviles, retail y servicios digitales. Se enfoca en la experiencia del cliente, el marketing masivo y la rapidez en las transacciones.

BYOD (Bring Your Own Device).

Política corporativa que permite a los empleados usar dispositivos personales en el trabajo. Aporta flexibilidad, pero implica riesgos de seguridad que deben gestionarse con controles y normativas claras.

C#.

Lenguaje de programación desarrollado por Microsoft dentro de la plataforma .NET. Está orientado a objetos y se usa ampliamente en el desarrollo de aplicaciones de escritorio, móviles, web y videojuegos con Unity. Es robusto, tipado y combina características de C++ y Java.

C++.

Lenguaje de programación de propósito general que extiende al C, incorporando programación orientada a objetos. Es muy utilizado en sistemas operativos, videojuegos, controladores y aplicaciones de alto rendimiento. Se caracteriza por su eficiencia y control sobre la memoria.

C++ Builder.

Entorno de desarrollo rápido (RAD) diseñado para programar en C++. Permite crear aplicaciones de escritorio y móviles con interfaces gráficas, integrando herramientas de diseño visual y bases de datos. Se destaca por su productividad y soporte multiplataforma.

CaaS (Containers as a Service).

Modelo de servicio en la nube que facilita la gestión de contenedores. Permite desplegar, escalar y administrar aplicaciones en entornos virtualizados mediante plataformas como Kubernetes o Docker. Es clave para la automatización y portabilidad en DevOps.

CAD (Computer-Aided Design).

Uso de software para crear diseños técnicos en ingeniería, arquitectura y manufactura. Permite elaborar planos detallados en 2D y 3D, optimizando procesos de diseño y producción. Ejemplos conocidos son AutoCAD y SolidWorks.

CAD 3D.

Extensión del CAD que trabaja directamente en tres dimensiones. Se emplea en modelado de piezas, simulación de movimientos y prototipado digital. Favorece la visualización realista de productos y reduce errores de diseño antes de la fabricación.

Caja Blanca (White Box Testing).

Técnica de prueba de software en la que el evaluador tiene acceso al código fuente y a la estructura interna del sistema. Se centra en verificar flujos de datos, condiciones lógicas y seguridad. Garantiza la calidad del software desde su núcleo.

Caja Gris (Gray Box Testing).

Método de prueba de software que combina aspectos de la caja blanca y la caja negra. El tester tiene conocimiento parcial de la lógica interna del sistema. Se usa en auditorías de seguridad y validación funcional.

Cable de Fibra Óptica.

Medio de transmisión de datos que utiliza pulsos de luz a través de hilos de vidrio o plástico. Ofrece gran ancho de banda, alta velocidad y resistencia a interferencias electromagnéticas. Es clave en telecomunicaciones, internet de alta velocidad y redes de larga distancia.

Cable UTP (Unshielded Twisted Pair).

Tipo de cable de red compuesto por pares de hilos de cobre trenzados sin apantallamiento. Es el más común en redes Ethernet debido a su bajo costo y facilidad de instalación. Se clasifica en categorías (Cat5e, Cat6, Cat7) según su capacidad de transmisión.

Cache Distribuida.

Sistema de almacenamiento temporal de datos que funciona en múltiples servidores interconectados. Permite mejorar el rendimiento de aplicaciones al reducir la latencia y balancear la carga. Es utilizado en grandes sistemas web y entornos de Big Data.

CalDAV (Calendaring Extensions to WebDAV).

Protocolo basado en WebDAV que permite la gestión y sincronización de calendarios en internet. Es utilizado por aplicaciones como Google Calendar y Apple Calendar. Facilita la interoperabilidad entre clientes y servidores de calendarios.

Call Center en la Nube.

Solución tecnológica que permite a las empresas gestionar centros de atención al cliente mediante infraestructura basada en la nube. Ofrece flexibilidad, escalabilidad y reducción de costos, ya que elimina la necesidad de hardware físico. Además, integra herramientas como chat, voz y análisis en tiempo real.

Call Stack.

Estructura de datos que almacena información sobre las funciones activas de un programa. Cada vez que se llama a una función, se crea un marco (stack frame) con sus parámetros, variables y dirección de retorno. Es esencial para la depuración y control de flujo en la programación.

Call to Action (CTA).

Elemento en marketing digital que invita al usuario a realizar una acción específica, como registrarse, descargar o comprar. Generalmente se presenta como un botón o enlace destacado. Un buen CTA incrementa la conversión y guía la experiencia del usuario en sitios web y aplicaciones.

Callback en Programación.

Función que se pasa como argumento a otra función y que se ejecuta posteriormente, en general después de que se complete una operación. Es ampliamente usada en programación asincrónica, como en JavaScript y Python. Facilita el control de flujos no secuenciales.

Cámara Web.

Una webcam es una cámara digital que captura datos de vídeo y audio y los transmite en tiempo real a través de Internet. Se suele utilizar para videoconferencias, retransmisiones en directo, reuniones en línea y grabación de vídeos.

Canal de Comunicación Seguro.

Medio de transmisión de datos protegido mediante cifrado y protocolos de seguridad. Garantiza confidencialidad, integridad y autenticidad de la información transmitida entre emisor y receptor. Es fundamental en banca en línea, comercio electrónico y redes privadas.

Canva.

Plataforma en línea de diseño gráfico que permite crear presentaciones, publicaciones y materiales visuales sin necesidad de conocimientos avanzados. Ofrece plantillas prediseñadas, herramientas colaborativas y opciones de exportación. Es ampliamente utilizada en marketing y educación.

Captcha.

Prueba de seguridad que distingue entre humanos y bots en aplicaciones web. Normalmente consiste en resolver acertijos visuales o de texto que las máquinas no pueden interpretar fácilmente. Se utiliza para prevenir fraudes, registros automáticos y ataques de fuerza bruta.

Card Sorting.

Técnica de investigación en experiencia de usuario (UX) que organiza información en categorías según la perspectiva de los usuarios. Ayuda a estructurar menús, sitios web o aplicaciones de forma intuitiva. Se realiza de manera física (con tarjetas) o digital con herramientas especializadas.

Cassandra.

Base de datos NoSQL distribuida, diseñada para manejar grandes volúmenes de datos en múltiples servidores. Es altamente escalable, tolerante a fallos y ofrece replicación en tiempo real. Es usada por empresas como Netflix y Facebook en sistemas críticos.

CDN (Content Delivery Network).

Red de servidores distribuidos geográficamente que entrega contenido web al usuario desde el servidor más cercano. Reduce la latencia, mejora la velocidad de carga y protege contra ataques DDoS. Es clave en streaming, comercio electrónico y aplicaciones globales.

Celda en Redes Móviles.

Área geográfica cubierta por una antena de telecomunicaciones (estación base). Cada celda se conecta a un sistema de red móvil, permitiendo la comunicación de voz y datos de los dispositivos. La cobertura de múltiples celdas permite la movilidad del usuario sin perder conexión.

Centro de Datos (Data Center).

Infraestructura física que alberga servidores, sistemas de almacenamiento y redes de telecomunicaciones. Está diseñado para garantizar disponibilidad, seguridad y redundancia en servicios digitales. Es esencial para empresas que requieren alta capacidad de procesamiento y almacenamiento.

Certificado Digital.

Archivo electrónico que autentica la identidad de una persona, organización o sitio web en internet. Se basa en criptografía de clave pública y garantiza confidencialidad y confianza en las transacciones. Es esencial en HTTPS, firmas electrónicas y comunicaciones seguras.

Certificación CompTIA.

Conjunto de certificaciones internacionales en tecnologías de la información. Abarcan áreas como soporte técnico (A+), redes (Network+), seguridad (Security+) y nube (Cloud+). Son reconocidas globalmente como estándares de conocimiento en TI.

Certificación ITIL.

Marco de mejores prácticas para la gestión de servicios de TI. La certificación ITIL valida el conocimiento en procesos como gestión de incidentes, cambios y disponibilidad. Es altamente demandada en empresas que buscan eficiencia y alineación tecnológica con el negocio.

Certificación PMP.

Credencial otorgada por el Project Management Institute (PMI) a profesionales en gestión de proyectos. Reconocida a nivel mundial, valida habilidades en planificación, ejecución y cierre de proyectos bajo metodologías tradicionales y ágiles.

Certificación Scrum Master.

Acreditación en metodologías ágiles que reconoce a profesionales capaces de facilitar y guiar equipos Scrum. Certificaciones como CSM o PSM son muy demandadas en proyectos de desarrollo de software y transformación digital.

Chip.

Componente electrónico fabricado en silicio que contiene circuitos integrados. Es la base del hardware moderno y puede ejecutar funciones de procesamiento, memoria o control. Se utiliza en computadores, teléfonos móviles, automóviles y prácticamente todos los dispositivos electrónicos actuales.

Chrome OS.

Sistema operativo desarrollado por Google basado en el navegador Chrome y en Linux. Está diseñado para funcionar principalmente con aplicaciones web y servicios en la nube. Se utiliza en dispositivos ligeros como Chromebooks, optimizando velocidad, seguridad y simplicidad.

CI/CD (Continuous Integration / Continuous Delivery).

Conjunto de prácticas en desarrollo de software que permiten integrar, probar y desplegar código de manera automática y continua. Mejora la calidad, acelera las entregas y reduce errores humanos. Es ampliamente utilizado en metodologías ágiles y DevOps.

Ciberataque.

Acción maliciosa realizada contra sistemas informáticos con el fin de robar, alterar o destruir información. Puede ejecutarse mediante malware, phishing, ransomware o ataques de denegación de servicio. Los ciberataques representan una de las principales amenazas para empresas y gobiernos.

Ciberdefensa.

Conjunto de medidas, técnicas y estrategias diseñadas para proteger sistemas informáticos frente a ciberataques. Incluye monitorización, análisis de amenazas y respuestas ante incidentes.

Ciberdelito.

Delito cometido utilizando tecnologías digitales o redes informáticas. Entre los más comunes están el fraude electrónico, el robo de identidad y el acceso no autorizado a sistemas.

Ciberespacio.

Entorno virtual creado por la interconexión de sistemas digitales a través de internet y otras redes. Incluye la comunicación, el almacenamiento de datos y las interacciones sociales o económicas realizadas en línea. Es un espacio clave para el comercio electrónico y la ciberseguridad.

Ciberinteligencia.

Uso de herramientas y análisis de datos digitales para anticipar, prevenir y responder a amenazas cibernéticas. Ayuda a las organizaciones a conocer mejor a sus atacantes y minimizar riesgos.

Ciberseguridad.

Conjunto de prácticas, tecnologías y procesos diseñados para proteger sistemas informáticos, redes y datos contra ataques, accesos no autorizados y daños. Abarca desde seguridad en dispositivos hasta protección de infraestructuras críticas. Es fundamental en un mundo digital interconectado.

Cibervigilancia.

Monitoreo continuo de redes, sistemas y actividades digitales para detectar posibles amenazas o comportamientos sospechosos. Se utiliza en entornos empresariales y gubernamentales para prevenir ataques.

Cifrado.

Proceso criptográfico que transforma información legible en un formato ilegible para protegerla contra accesos no autorizados. Puede ser simétrico (misma clave para cifrar y descifrar) o asimétrico (claves pública y privada). Se utiliza en banca digital, comunicaciones seguras y almacenamiento de datos.

Cifrado Asimétrico.

Método de cifrado que utiliza un par de claves: una pública para cifrar y una privada para descifrar. Es la base de tecnologías como SSL/TLS, blockchain y firmas digitales. Ofrece mayor seguridad en el intercambio de información en entornos no confiables.

Cifrado Simétrico.

Técnica criptográfica donde se utiliza la misma clave para cifrar y descifrar los datos. Aunque es rápido y eficiente, requiere una gestión segura de la clave para evitar accesos no autorizados. Se aplica en almacenamiento de archivos, bases de datos y comunicaciones internas.

Circuito Integrado.

Dispositivo electrónico que contiene miles o millones de componentes (transistores, resistencias, condensadores) en un solo chip de silicio. Es esencial en la fabricación de computadoras y dispositivos móviles.

Cloud Computing.

Modelo de prestación de servicios de TI a través de internet que permite acceso bajo demanda a servidores, almacenamiento, bases de datos y aplicaciones. Ofrece flexibilidad, escalabilidad y reducción de costos operativos. Existen modelos públicos, privados e híbridos.

Cloud Híbrida.

Infraestructura de TI que combina servicios de nube pública y privada. Permite a las organizaciones mantener datos sensibles en entornos privados mientras aprovechan la escalabilidad y economía de la nube pública. Favorece la flexibilidad y la seguridad.

Cloud Nativa (Computación Nativa en la Nube).

Enfoque de desarrollo de aplicaciones específicamente diseñadas para aprovechar los servicios y la arquitectura de la nube, utilizando microservicios, contenedores y DevOps.

Cloud Privada.

Modelo de nube informática que ofrece recursos exclusivos a una sola organización. Puede estar ubicada en las instalaciones de la empresa o en centros de datos de terceros. Brinda mayor control, personalización y seguridad que la nube pública.

Cloud Pública.

Servicio de nube que proporciona recursos a través de proveedores externos como AWS, Azure o Google Cloud. Es compartida entre múltiples usuarios y se paga bajo un modelo de consumo. Destaca por su escalabilidad, bajo costo y rapidez de implementación.

Cloud Storage.

Servicio de almacenamiento de datos en servidores remotos accesibles por internet. Permite guardar, sincronizar y compartir archivos sin necesidad de hardware físico local. Ejemplos comunes incluyen Google Drive, Dropbox y OneDrive.

Clúster.

Conjunto de computadoras o servidores que trabajan de manera conjunta como si fueran un único sistema. Se utiliza para aumentar la disponibilidad, el rendimiento y la capacidad de procesamiento en aplicaciones críticas.

Clúster de Bases de Datos.

Conjunto de servidores que gestionan de forma distribuida una base de datos, permitiendo mayor velocidad de consulta, tolerancia a fallos y escalabilidad horizontal.

Clúster de Computación.

Grupo de computadoras conectadas que trabajan en paralelo para realizar tareas complejas como simulaciones científicas, minería de datos o inteligencia artificial.

Clúster de Servidores.

Conjunto de servidores interconectados que funcionan como una sola unidad para mejorar rendimiento, escalabilidad y disponibilidad. Se usan en bases de datos, aplicaciones web y procesamiento intensivo.

CMS (Content Management System).

Plataforma que permite crear, gestionar y publicar contenido digital sin necesidad de conocimientos avanzados en programación. Ejemplos populares son WordPress, Joomla y Drupal. Facilita la administración de sitios web y comercio electrónico.

Cobol.

Lenguaje de programación orientado a negocios, creado en los años 60. Se utiliza ampliamente en sistemas bancarios, financieros y administrativos debido a su robustez y permanencia.

Código Abierto (Open Source).

Modelo de desarrollo de software en el que el código fuente es accesible y puede ser modificado, distribuido o mejorado por cualquier persona. Promueve la innovación colaborativa y la transparencia. Ejemplos destacados son Linux, Apache y MySQL.

Código Binario.

Sistema de numeración en base 2 que utiliza únicamente los dígitos 0 y 1. Es el lenguaje fundamental de las computadoras, ya que representa instrucciones y datos mediante estados eléctricos (encendido/apagado). Constituye la base de todo procesamiento digital.

Código Fuente.

Conjunto de instrucciones escritas en un lenguaje de programación comprensible para los humanos. Es el punto de partida para que un compilador o intérprete genere un programa ejecutable. Su claridad y documentación son claves para el mantenimiento de software.

Código QR.

Tipo de código de barras bidimensional que almacena información en módulos negros y blancos. Puede contener enlaces, texto, contactos o accesos rápidos a aplicaciones, y se escanea fácilmente con dispositivos móviles. Es muy utilizado en publicidad y pagos digitales.

Cognitive Computing (Computación Cognitiva).

Rama de la informática que desarrolla sistemas capaces de simular procesos de razonamiento humano, como el aprendizaje, la interpretación del lenguaje natural y la toma de decisiones.

Cold Backup.

Respaldo de datos que se realiza cuando el sistema está completamente detenido. Garantiza consistencia en la información, aunque implica mayor tiempo de inactividad.

Cold Data.

Datos que se consultan con poca frecuencia y suelen almacenarse en medios de bajo costo. Son útiles para archivado a largo plazo y no requieren acceso inmediato.

Colocation de Servidores.

ervicio que permite a las empresas alojar sus propios servidores en un centro de datos de un tercero, aprovechando seguridad, energía redundante y conectividad de alto nivel.

Column-Oriented Database (Base de Datos Orientada a Columnas).

Tipo de base de datos que organiza y almacena la información por columnas en lugar de filas. Esto permite optimizar consultas analíticas, compresión de datos y procesamiento masivo.

COM (Component Object Model).

Modelo de Microsoft para permitir que distintos componentes de software interactúen entre sí de manera independiente del lenguaje de programación en el que fueron creados.

Comando en Línea (CLI Command Line Interface).

Interfaz basada en texto que permite al usuario interactuar con un sistema operativo o programa mediante comandos escritos. Ofrece más control y rapidez que las interfaces gráficas.

Comercio Electrónico C2B (Consumer to Business)./h2>

Modelo en el que los consumidores ofrecen productos o servicios a empresas, como ocurre con plataformas de freelancers o marketing de contenidos.

Comercio Electrónico C2C (Consumer to Consumer).

Modelo de negocio en línea que conecta a consumidores para la compra y venta de bienes o servicios, generalmente mediante plataformas intermediarias como eBay.

Compilación Cruzada.

Proceso de generar código ejecutable para una plataforma distinta a aquella en la que se realiza la compilación. Es común en el desarrollo de sistemas embebidos y dispositivos móviles, donde se programa en un equipo, pero se ejecuta en otro con diferente arquitectura.

Compilación Just-In-Time (JIT).

Técnica de ejecución en la que el código intermedio de un programa se traduce a código máquina en tiempo real. Esto mejora el rendimiento al optimizar el programa según el hardware y el uso específico. Se emplea en plataformas como Java y .NET.

Compilador.

Programa que traduce código fuente escrito en un lenguaje de alto nivel a lenguaje máquina ejecutable. Permite que el hardware interprete y ejecute instrucciones. Ejemplos de compiladores son GCC para C/C++ y javac para Java.

Componente de Software.

Unidad modular reutilizable que encapsula funcionalidades específicas en un sistema. Facilita el desarrollo ágil, la escalabilidad y la integración de aplicaciones. Ejemplos comunes son bibliotecas, APIs y microservicios.

Computación Autónoma.

Enfoque en el que los sistemas informáticos se autogestionan, corrigen errores y optimizan recursos sin intervención humana. Se basa en inteligencia artificial y machine learning para reducir la complejidad administrativa.

Computación Cognitiva.

Rama de la informática que combina inteligencia artificial, procesamiento de lenguaje natural y aprendizaje automático para simular procesos de pensamiento humano. Se utiliza en chatbots, sistemas de recomendación y análisis predictivo.

Computación Cuántica.

Campo de la informática que aprovecha principios de la mecánica cuántica como la superposición y el entrelazamiento. Permite resolver problemas complejos mucho más rápido que las computadoras clásicas. Es aplicada en criptografía, optimización y simulación molecular.

Computación Cuántica en la Nube.

Servicio que permite acceder de forma remota a procesadores cuánticos mediante plataformas en la nube. Facilita a investigadores y empresas experimentar con algoritmos cuánticos sin necesidad de disponer de hardware especializado.

Computación Distribuida.

Modelo en el que múltiples computadoras interconectadas trabajan de manera conjunta como un único sistema. Permite compartir recursos y dividir tareas para optimizar el rendimiento.

Computación en Borde (Edge Computing).

Enfoque de procesamiento de datos que se realiza cerca de la fuente de generación, en lugar de depender de un centro de datos remoto. Reduce la latencia y mejora la eficiencia en aplicaciones como IoT, vehículos autónomos y dispositivos médicos.

Computación en la Nube Híbrida.

Combinación de nube pública y privada que permite a las organizaciones equilibrar seguridad y escalabilidad según sus necesidades.

Computación Paralela.

Técnica que permite la ejecución simultánea de múltiples procesos dividiendo tareas entre diferentes procesadores o núcleos. Se utiliza en supercomputadoras, análisis de datos y gráficos 3D.

Computación Perimetral (Edge Computing).

Consiste en procesar datos en dispositivos cercanos al lugar donde se generan, en lugar de enviarlos a un servidor central. Esto mejora la velocidad y reduce la latencia en aplicaciones críticas.

Computación Ubicua.

Paradigma que busca integrar la informática en la vida cotidiana de forma invisible, donde los dispositivos y sistemas interactúan automáticamente con el entorno del usuario.

Computadora de Sobremesa.

Equipo de cómputo diseñado para uso fijo en oficinas u hogares. Generalmente cuenta con monitor, teclado, ratón y torre, y ofrece mayor capacidad de personalización que los portátiles.

Computadora Portátil (Laptop).

Dispositivo de cómputo transportable que integra pantalla, teclado, batería recargable y componentes internos en un solo equipo. Está diseñado para brindar movilidad al usuario.

Computadora Vestible (Wearable Computer).

Dispositivo tecnológico que se lleva puesto, como relojes inteligentes o gafas de realidad aumentada, y que permite acceder a información o funciones digitales en movimiento.

Conexión Full-Duplex.

Tipo de comunicación en redes que permite la transmisión y recepción de datos de manera simultánea. Es más eficiente que half-duplex y se utiliza en Ethernet y telecomunicaciones.

Conexión Half-Duplex.

Modo de comunicación en el que los datos pueden transmitirse en ambas direcciones, pero no al mismo tiempo. Ejemplo: las radios de dos vías.

Conexión Inalámbrica (Wi-Fi).

Tecnología de comunicación que utiliza ondas de radio para conectar dispositivos a una red local sin necesidad de cables. Es ampliamente usada en hogares, oficinas y espacios públicos.

Conexión Simplex.

Forma de comunicación en la que los datos fluyen en una sola dirección, sin retorno. Un ejemplo clásico es la transmisión televisiva.

Conmutación de Circuitos.

Método de comunicación en el que se establece un canal dedicado entre dos puntos antes de transmitir datos. Fue ampliamente utilizado en redes telefónicas tradicionales, garantizando una conexión continua.

Conmutación de Paquetes.

Técnica de transmisión de datos en la que la información se divide en pequeños paquetes independientes que viajan por distintas rutas hasta llegar al destino. Es la base del funcionamiento de Internet.

Conmutador (Switch).

Dispositivo de red encargado de recibir, procesar y enviar datos entre dispositivos conectados a una red local. A diferencia de un hub, dirige los paquetes de datos únicamente al destinatario correcto, mejorando la eficiencia y seguridad.

Consola.

Dispositivo o interfaz de entrada/salida usada en informática y videojuegos. En programación, se refiere a la ventana donde se ejecutan comandos; en videojuegos, es un equipo especializado en correr títulos de entretenimiento.

Consola de Administración.

Interfaz centralizada utilizada por administradores de TI para monitorear, configurar y gestionar sistemas, redes y aplicaciones. Simplifica tareas de mantenimiento y control de seguridad.

Contenedor (Container).

Unidad ligera de software que empaqueta aplicaciones y todas sus dependencias necesarias para ejecutarse de manera uniforme en cualquier entorno. Tecnologías como Docker popularizaron su uso en la nube y DevOps.

Contenedor de Software.

Similar al concepto de contenedor, se refiere a la encapsulación de aplicaciones y bibliotecas en un entorno aislado, lo que asegura portabilidad, escalabilidad y consistencia en el despliegue.

Container Orchestration (Orquestación de Contenedores).

Conjunto de herramientas y procesos para automatizar la gestión de contenedores, incluyendo despliegue, escalado, balanceo de carga y supervisión. Kubernetes es el sistema más utilizado.

Context-Aware Computing (Computación Consciente del Contexto).

Tecnología que adapta el comportamiento de sistemas y aplicaciones según el entorno y situación del usuario. Se aplica en dispositivos móviles, IoT y sistemas inteligentes.

Control de Acceso.

Conjunto de mecanismos que regulan quién puede acceder a sistemas, redes o aplicaciones. Se basa en credenciales como contraseñas, tarjetas inteligentes o autenticación biométrica, garantizando la seguridad de los recursos.

Control de Acceso Basado en Atributos (ABAC).

Modelo de seguridad que otorga permisos en función de atributos del usuario, recurso o entorno (ejemplo: cargo, ubicación, hora). Proporciona mayor flexibilidad que los métodos tradicionales y se emplea en entornos críticos.

Control de Acceso Basado en Roles (RBAC).

Modelo de control en el que los permisos se asignan según los roles dentro de una organización. Simplifica la administración al permitir que múltiples usuarios compartan permisos al ocupar el mismo rol.

Control de Versiones.

Sistema que gestiona los cambios en el código fuente de un proyecto, permitiendo colaboración y seguimiento histórico. Git y Subversión son ejemplos populares que facilitan la integración en equipos de desarrollo.

Control de Versiones Centralizado.

Modelo en el que un servidor único almacena el código y gestiona las versiones. Los usuarios trabajan conectados a este repositorio central, como ocurre en CVS o Subversión. Tiene limitaciones frente a sistemas distribuidos.

Control de Versiones Distribuido.

Modelo en el que cada usuario posee una copia completa del repositorio, lo que permite trabajar sin conexión y fusionar cambios después. Git es el ejemplo más utilizado a nivel mundial.

Controladores (Drivers).

Programas que permiten que el sistema operativo se comunique con el hardware. Actúan como traductores entre los dispositivos físicos (impresoras, tarjetas gráficas, etc.) y las aplicaciones de software.

Controlador Lógico Programable (PLC).

Dispositivo electrónico usado en automatización industrial para controlar maquinaria y procesos. Puede recibir entradas de sensores, ejecutar lógica predefinida y activar salidas como motores o válvulas.

Cookie.

Archivo pequeño que los sitios web almacenan en el navegador del usuario. Guarda información como preferencias, historial de navegación o datos de sesión. Aunque mejora la experiencia del usuario, también plantea retos de privacidad.

Cookie de Sesión.

Tipo de cookie temporal que se elimina al cerrar el navegador. Se usa para mantener activa la sesión de un usuario mientras navega por un sitio web, garantizando la continuidad de procesos como compras en línea o inicios de sesión.

Cookie Persistente.

Archivo que permanece en el dispositivo del usuario incluso después de cerrar el navegador. Almacena información de largo plazo, como credenciales recordadas o configuraciones personalizadas de un sitio.

Copia de Seguridad en Caliente (Hot Backup).

Respaldo de datos realizado mientras el sistema sigue en funcionamiento. Es útil en entornos que requieren alta disponibilidad, como bases de datos empresariales, aunque suele ser más complejo de gestionar.

Correo Electrónico.

Servicio digital que permite el intercambio de mensajes a través de redes informáticas. Es uno de los medios de comunicación más utilizados en el mundo, tanto en contextos personales como profesionales.

Correo Electrónico Corporativo.

Servicio de email personalizado para empresas, que incluye dominios propios y mayores niveles de seguridad. Facilita la identidad de marca y la gestión de usuarios dentro de una organización.

Cortafuegos de Aplicación Web (WAF).

Sistema de seguridad especializado en proteger aplicaciones web contra ataques como inyecciones SQL, cross-site scripting (XSS) o tráfico malicioso. Opera filtrando y monitoreando las solicitudes HTTP/HTTPS.

Cortafuegos de Red.

Dispositivo o software que controla el tráfico de red entrante y saliente según políticas de seguridad predefinidas. Protege los sistemas de accesos no autorizados y ataques externos.

CPU (Unidad Central de Procesamiento).

Principal componente de un sistema informático encargado de ejecutar instrucciones y procesar datos. Es considerado el “cerebro” del computador, compuesto por la unidad de control y la unidad aritmeticológica.

CPU Virtual.

Recurso de procesamiento creado en entornos de virtualización. Simula una CPU física y permite que múltiples máquinas virtuales compartan la misma infraestructura de hardware. Se utiliza en nubes públicas y privadas para optimizar recursos.

Criptografía Asimétrica.

Método de cifrado que utiliza un par de claves: una pública para encriptar y una privada para desencriptar. Es la base de la seguridad en internet, empleada en certificados digitales, firmas electrónicas y conexiones HTTPS.

Criptografía Cuántica.

Campo de la criptografía que aplica principios de la mecánica cuántica, como el entrelazamiento y la superposición, para garantizar la seguridad de las comunicaciones. Su técnica más conocida es la distribución cuántica de claves (QKD).

Criptografía de Clave Simétrica.

Método de cifrado en el que la misma clave se utiliza tanto para encriptar como para desencriptar los datos. Es más rápido que la criptografía asimétrica, pero requiere un canal seguro para compartir la clave.

Criptografía Homomórfica.

Técnica que permite realizar operaciones sobre datos cifrados sin necesidad de descifrarlos primero. Esto protege la privacidad y es útil en aplicaciones como el análisis de datos en la nube.

Criptominería.

Proceso de validación de transacciones y creación de nuevas unidades de criptomonedas mediante la resolución de complejos problemas matemáticos. Requiere alta capacidad de cómputo y un gran consumo energético.

Criptomoneda.

Activo digital basado en criptografía y tecnología blockchain que funciona como medio de intercambio. Ejemplos conocidos son Bitcoin, Ethereum y Litecoin. Su valor fluctúa según la oferta, la demanda y la confianza de los usuarios.

CRM (Customer Relationship Management).

Sistema de gestión que centraliza la información de clientes para mejorar las relaciones comerciales. Permite seguimiento de ventas, marketing personalizado y soporte al cliente, aumentando la eficiencia empresarial.

Cross-Platform Development.

Práctica de desarrollo de software que permite que una aplicación funcione en múltiples sistemas operativos (Windows, macOS, Linux, Android, iOS). Frameworks como Flutter o Xamarin facilitan este enfoque.

Cross-Site Request Forgery (CSRF).

Tipo de ataque en el que un usuario autenticado en una aplicación es forzado a ejecutar acciones no deseadas sin su consentimiento. Se previene mediante tokens CSRF y validaciones en el lado del servidor.

Crowdfunding Tecnológico.

Modelo de financiamiento colectivo donde múltiples personas contribuyen económicamente a proyectos tecnológicos. Plataformas como Kickstarter e Indiegogo permiten que startups obtengan capital inicial para innovaciones.

Crowdsourcing.

Modelo de colaboración que aprovecha la participación masiva de personas, generalmente a través de internet, para resolver problemas, generar ideas o desarrollar productos. Ha sido clave en proyectos de innovación abierta.

CRUD.

Conjunto de operaciones básicas que se aplican en bases de datos y sistemas de gestión de información. Representa la funcionalidad mínima que una aplicación debe ofrecer para manejar datos de forma eficiente.

Crust de Seguridad en TI.

Término usado para describir una capa protectora alrededor de los sistemas informáticos. Incluye políticas, procedimientos y herramientas diseñadas para resguardar activos digitales de accesos no autorizados.

CSIRT (Computer Security Incident Response Team).

Equipo especializado en gestionar incidentes de seguridad informática. Su misión es detectar, analizar y mitigar ataques cibernéticos, además de coordinar la respuesta y la recuperación en organizaciones.

CSS (Cascading Style Sheets).

Lenguaje utilizado para definir el diseño visual de páginas web. Permite controlar colores, fuentes, espaciado y disposición de elementos, separando la presentación del contenido HTML.

CSV (Comma-Separated Values).

Formato de archivo simple que almacena datos tabulares en texto plano, separando los valores por comas. Es ampliamente usado para importar y exportar información entre sistemas y bases de datos.

DALL-E.

Modelo de inteligencia artificial creado por OpenAI que genera imágenes a partir de descripciones en lenguaje natural. Se basa en redes neuronales y técnicas avanzadas de aprendizaje profundo. Ha sido entrenado con grandes volúmenes de datos para producir imágenes creativas, originales y con alto nivel de detalle. Su uso se extiende en diseño, publicidad, prototipado y arte digital.

Dashboard Empresarial.

Es una interfaz visual que integra y muestra en tiempo real indicadores clave de rendimiento (KPIs). Permite a los directivos y equipos de trabajo tomar decisiones rápidas basadas en datos. Incluye gráficos, tablas y métricas de diferentes áreas como ventas, finanzas o logística. Su principal ventaja es transformar información compleja en un formato fácil de entender.

Data Cleansing.

Proceso que consiste en identificar, depurar y corregir datos erróneos, duplicados o incompletos en una base de datos. Garantiza que la información utilizada en análisis y reportes sea confiable y precisa. Implica técnicas de normalización, validación y estandarización. Es fundamental en proyectos de business intelligence y data science.

Data Governance.

Conjunto de normas, políticas y procesos que regulan la gestión y uso de los datos dentro de una organización. Su objetivo es asegurar la calidad, seguridad y disponibilidad de la información. Involucra la asignación de responsabilidades sobre la administración de datos. Es clave en entornos donde los datos son considerados un activo estratégico.

Dataflow Programming.

Paradigma de programación donde el flujo de datos determina el orden de ejecución. Los programas se representan como grafos de nodos conectados por canales de datos. Facilita el procesamiento paralelo y distribuido. Es común en aplicaciones de big data, procesamiento multimedia y sistemas de señal.

Data Lake.

Es un repositorio que almacena grandes cantidades de datos en bruto sin necesidad de estructurarlos previamente. Permite integrar datos de múltiples fuentes, estructurados o no estructurados. Es muy utilizado en big data y análisis predictivo. Proporciona flexibilidad para procesar la información según las necesidades posteriores.

Data Leakage (Fuga de Datos).

Situación en la que información sensible o confidencial se expone, transmite o accede de forma no autorizada. Puede ocurrir por errores humanos, configuraciones incorrectas, ataques cibernéticos o uso indebido de aplicaciones. Es un riesgo crítico en seguridad informática y cumplimiento normativo. Para prevenirla se aplican cifrado, monitoreo y políticas de control de acceso.

Data Mart.

Subconjunto de un data warehouse diseñado para un área o departamento específico de la organización. Su objetivo es ofrecer consultas más rápidas y personalizadas. Se utilizan en marketing, ventas, finanzas, entre otros. Facilitan el acceso a la información relevante sin necesidad de procesar toda la base de datos empresarial.

Data Mining.

Disciplina que utiliza algoritmos y técnicas estadísticas para descubrir patrones, relaciones y tendencias en grandes volúmenes de datos. Es clave en la toma de decisiones estratégicas en empresas. Se aplica en marketing, predicción de fraudes, segmentación de clientes y bioinformática. Transforma datos en conocimiento útil.

Data Science.

Campo interdisciplinario que combina programación, estadística, matemáticas y aprendizaje automático para analizar e interpretar datos. Los científicos de datos emplean lenguajes como Python, R o SQL. Se utiliza en la predicción de tendencias, optimización de procesos y desarrollo de modelos de inteligencia artificial. Es uno de los pilares de la transformación digital.

Data Virtualization.

Tecnología que permite acceder y gestionar datos en tiempo real sin importar en qué sistema o ubicación se encuentren almacenados. Crea una vista unificada de la información para facilitar su consulta y análisis. Reduce la necesidad de duplicar o mover datos entre plataformas. Es útil en entornos donde los datos provienen de múltiples fuentes.

Data Warehouse.

Almacén centralizado que integra datos de diferentes sistemas en un formato estructurado y optimizado para análisis. Es la base de la inteligencia de negocios (BI) en las organizaciones. Permite ejecutar consultas complejas y generar reportes históricos y predictivos. Se diferencia de los data lakes en que los datos ya están procesados y organizados.

Database Sharding.

Estrategia de escalabilidad para bases de datos que consiste en dividirlas en fragmentos independientes llamados shards. Cada fragmento se aloja en un servidor distinto, lo que mejora el rendimiento y la disponibilidad. Es común en aplicaciones con millones de usuarios como redes sociales o sistemas financieros. Reduce la sobrecarga de consultas y distribuye el tráfico de manera eficiente.

DataOps.

Es una metodología que combina data engineering, data management y DevOps para optimizar el ciclo de vida de los datos. Su objetivo es automatizar y mejorar la calidad de los procesos de integración, transformación y análisis de información. Facilita la colaboración entre equipos de datos y TI. Se usa mucho en entornos donde los datos cambian rápidamente.

DataOps Pipeline.

Conjunto de prácticas y herramientas que automatizan el ciclo de vida de los datos, desde la ingesta hasta el análisis. Integra principios de DevOps en el ámbito de datos, promoviendo calidad, velocidad y colaboración. Es clave para proyectos de machine learning y analítica avanzada.

Debugging.

Proceso de identificar, aislar y corregir errores o defectos en el software. Involucra técnicas como logging, breakpoints y análisis de memoria. Es una de las tareas más comunes en el desarrollo de software. Un buen debugging reduce costos y mejora la calidad del producto final.

Decision Support System (DSS).

Sistema de información que ayuda a los directivos a tomar decisiones basadas en datos. Combina bases de datos, modelos analíticos y herramientas de visualización. Se usa en finanzas, salud, logística y negocios. Proporciona apoyo, pero no sustituye el juicio humano.

Deep Learning.

Subcampo de la inteligencia artificial que utiliza redes neuronales profundas con múltiples capas de procesamiento. Es capaz de reconocer patrones complejos en datos como imágenes, sonidos y texto. Se aplica en asistentes virtuales, autos autónomos, traducción automática y generación de contenido. Es una de las tecnologías más avanzadas en IA moderna.

Deepfake.

Contenido multimedia manipulado mediante inteligencia artificial, en el que se sustituyen rostros, voces o escenas. Se logra con redes generativas adversarias (GANs). Aunque puede usarse en entretenimiento, también plantea riesgos éticos y de seguridad, como la desinformación. Hoy existen herramientas de detección para combatir su mal uso.

Defensa en Profundidad.

Estrategia de ciberseguridad que aplica múltiples capas de defensa para proteger sistemas y datos. Incluye firewalls, antivirus, cifrado, autenticación y monitoreo continuo. Su principio es que, si una capa falla, las demás mantengan la seguridad. Es ampliamente utilizada en empresas y gobiernos para reducir riesgos de ciberataques.

Delay en Redes.

Es la latencia que experimenta un paquete de datos al viajar desde el origen hasta el destino. Puede deberse a factores como transmisión, procesamiento, encolamiento o propagación. Un alto delay afecta negativamente aplicaciones sensibles al tiempo, como videollamadas o videojuegos en línea. Reducirlo es un desafío clave en telecomunicaciones.

Demilitarized Zone (DMZ).

Subred intermedia en una arquitectura de red que protege los recursos internos de accesos externos no autorizados. Se utiliza para alojar servidores accesibles desde internet, como páginas web o correos electrónicos. Sirve como “zona de contención” ante posibles ataques. Es una práctica de seguridad común en infraestructuras críticas.

Densidad de Píxeles (PPI).

Métrica que indica la cantidad de píxeles contenidos en una pulgada de pantalla. Cuanto mayor es el PPI, más nítida y detallada se ve la imagen. Es clave en dispositivos móviles, monitores y televisores. Influye directamente en la experiencia de usuario en tareas como lectura, diseño gráfico o realidad virtual.

Dependencia de Software.

Se refiere a los módulos, librerías o frameworks externos que un programa necesita para funcionar. Una gestión inadecuada de dependencias puede ocasionar vulnerabilidades de seguridad o incompatibilidades. Existen gestores como npm, pip o Maven que facilitan este control. Es un aspecto crucial en el desarrollo y mantenimiento de software.

Deployment Automatizado.

Práctica que consiste en implementar aplicaciones en entornos de prueba o producción sin intervención manual. Usa herramientas de automatización como Jenkins, Ansible o GitHub Actions. Aumenta la velocidad de entrega, reduce errores humanos y asegura consistencia en los despliegues. Es parte esencial de la integración y entrega continua (CI/CD).

Desarrollo Ágil.

Metodología de gestión de proyectos de software que se basa en la colaboración, la entrega rápida de valor y la adaptación a cambios. Está sustentada en el Manifiesto Ágil y emplea marcos como Scrum, Kanban o XP. Promueve la comunicación constante entre clientes, desarrolladores y equipos de negocio. Es la forma más usada hoy en día en desarrollo de software.

Desarrollo Backend.

Área del desarrollo enfocada en la lógica, la base de datos y los procesos internos que permiten que una aplicación funcione. Se encarga de procesar solicitudes, gestionar información y mantener la seguridad de los datos. Lenguajes comunes son Java, Python, PHP, Ruby y Node.js. Su buen diseño garantiza escalabilidad y eficiencia en las aplicaciones.

Desarrollo Frontend.

Área del desarrollo encargada de crear la interfaz gráfica y la experiencia del usuario en aplicaciones web o móviles. Utiliza tecnologías como HTML, CSS, JavaScript y frameworks como React, Angular o Vue.js. Conecta el diseño con la lógica del backend. Su objetivo principal es ofrecer accesibilidad, usabilidad y diseño responsivo.

Desarrollo Full Stack.

Se refiere a la capacidad de un desarrollador o equipo para trabajar tanto en el frontend como en el backend de una aplicación. Requiere dominio de múltiples tecnologías, APIs, servidores y bases de datos. Es un rol muy valorado porque permite una visión integral del proyecto. Aporta versatilidad y eficiencia en equipos pequeños o medianos.

Desarrollo Nativo Móvil.

Creación de aplicaciones diseñadas específicamente para un sistema operativo como Android o iOS. Emplea lenguajes propios, como Kotlin/Java en Android y Swift/Objective-C en iOS. Estas aplicaciones tienen mejor rendimiento, acceso completo al hardware y una experiencia optimizada para el usuario. Son ideales para proyectos que requieren alta calidad.

Desarrollo Web.

Proceso de creación y mantenimiento de sitios y aplicaciones en internet. Incluye aspectos como programación, diseño, bases de datos, servidores y seguridad. Puede ir desde simples páginas estáticas hasta complejas plataformas interactivas. Es una de las áreas con mayor demanda en el mundo de la tecnología.

Desarrollo Web Progresivo (PWA).

Tipo de aplicación web que combina lo mejor de las páginas tradicionales y las aplicaciones móviles. Funcionan en el navegador, pero ofrecen características como notificaciones push, acceso offline y diseño responsivo. Son ligeras, rápidas y multiplataforma. Permiten brindar experiencias cercanas a las apps nativas sin necesidad de descarga.

Desfragmentación de Disco.

Proceso que reorganiza los fragmentos de archivos dispersos en un disco duro para que queden almacenados de manera contigua. Mejora la velocidad de acceso y el rendimiento del sistema. Es especialmente útil en discos mecánicos (HDD), ya que en los SSD no se recomienda. Contribuye a prolongar la vida útil de un disco.

Digital Forensics (Informática Forense).

Disciplina que aplica técnicas científicas para investigar incidentes digitales. Permite recuperar, analizar y preservar evidencias en dispositivos electrónicos. Es crucial en ciberseguridad, investigaciones policiales y auditorías. Requiere cumplir normas legales para garantizar la validez de las pruebas.

Digital Twin (Gemelo Digital).

Representación virtual de un objeto, proceso o sistema físico. Se actualiza en tiempo real mediante sensores y datos. Permite simulaciones, predicciones y mejoras en mantenimiento. Es utilizado en industria, salud, energía y ciudades inteligentes. Es un pilar en la transformación digital y la Industria 4.0.

Digital Signature (Firma Digital).

Mecanismo criptográfico que garantiza la autenticidad e integridad de un mensaje o documento electrónico. Utiliza claves públicas y privadas dentro de sistemas de criptografía asimétrica. Asegura que el contenido no ha sido alterado y confirma la identidad del emisor. Es ampliamente utilizada en comercio electrónico, contratos digitales y certificados SSL/TLS.

Despliegue Azul-Verde.

Estrategia de implementación de software que utiliza dos entornos paralelos: uno en producción (azul) y otro preparado para la nueva versión (verde). El cambio entre ambos es casi instantáneo, lo que reduce riesgos de interrupciones. Permite hacer rollbacks rápidos en caso de fallas. Es muy usado en entornos DevOps.

Despliegue Canary.

Técnica de implementación progresiva en la que una nueva versión de software se lanza primero a un grupo reducido de usuarios. Permite detectar errores o problemas antes de extender el despliegue al total de la base. Minimiza riesgos y asegura estabilidad. Es común en grandes plataformas tecnológicas.

Despliegue Continuo.

Práctica de DevOps que automatiza la entrega de software en producción tras superar pruebas de calidad. Usa pipelines de integración y entrega continua (CI/CD). Favorece ciclos de liberación más cortos, reduce errores y acelera la innovación. Es un paso más allá de la integración continua.

Diagrama de Casos de Uso.

Modelo UML que muestra las interacciones entre los actores (usuarios u otros sistemas) y el sistema en desarrollo. Define los requerimientos funcionales de una aplicación desde la perspectiva del usuario. Es una herramienta clave en la etapa de análisis de sistemas, ayudando a entender el alcance y las funcionalidades esperadas.

Diagrama de Clases UML.

Representación gráfica utilizada en la metodología UML (Unified Modeling Language) que describe la estructura de un sistema mediante clases, atributos, métodos y relaciones. Permite modelar la arquitectura de software de manera clara y organizada. Es fundamental en el diseño orientado a objetos y facilita la comunicación entre analistas y desarrolladores.

Diagrama de Flujo.

Representación visual de un proceso o algoritmo mediante símbolos estandarizados (óvalos, rectángulos, rombos, flechas). Permite comprender y comunicar cómo fluye la información o las actividades dentro de un sistema. Se utiliza en programación, procesos empresariales y resolución de problemas. Su claridad lo convierte en una herramienta didáctica y práctica.

Diagrama Entidad-Relación (DER).

Modelo gráfico usado en bases de datos para representar entidades, atributos y relaciones. Facilita el diseño lógico de bases de datos relacionales, asegurando integridad y consistencia. Se compone de entidades (rectángulos), relaciones (rombos) y atributos (óvalos). Es ampliamente empleado en la ingeniería de software y administración de datos.

Digitalización.

Proceso de convertir información analógica (documentos, imágenes, sonidos) en formato digital. Permite almacenar, procesar y transmitir datos de manera más eficiente. Es una base de la transformación digital y ha revolucionado sectores como la educación, la medicina y la gestión empresarial. Impulsa la accesibilidad y preservación de la información.

Diplomado en TIC.

Programa académico de corta o mediana duración enfocado en las Tecnologías de la Información y la Comunicación (TIC). Busca actualizar y profundizar conocimientos en áreas como redes, programación, ciberseguridad, gestión de proyectos y transformación digital. Está dirigido a profesionales o estudiantes que buscan mejorar su perfil en el ámbito tecnológico.

Directiva Europea de Protección de Datos (GDPR).

Normativa legal de la Unión Europea implementada en 2018 para garantizar la privacidad y protección de los datos personales. Obliga a las organizaciones a obtener consentimiento explícito, permitir el acceso, rectificación y eliminación de los datos. Ha marcado un estándar global en derechos digitales y sanciona fuertemente su incumplimiento.

DirectX.

Conjunto de APIs (Interfaces de Programación de Aplicaciones) desarrolladas por Microsoft para manejar tareas relacionadas con gráficos, sonido y multimedia en Windows. Es ampliamente utilizada en videojuegos y aplicaciones que requieren alto rendimiento gráfico. Incluye componentes como Direct3D, DirectDraw y DirectSound. Su propósito es optimizar la interacción con hardware gráfico.

Disco Duro.

Dispositivo de almacenamiento que guarda información de manera permanente, ya sea en formato magnético (HDD) o electrónico (SSD). Es esencial en computadoras y servidores para almacenar sistemas operativos, programas y archivos. Los SSD ofrecen mayor velocidad y durabilidad. Su capacidad puede ir desde pocos gigabytes hasta varios terabytes.

Disco de Estado Sólido (SSD).

Dispositivo de almacenamiento que utiliza memoria flash NAND en lugar de discos magnéticos como los HDD. Ofrece mayor velocidad de lectura/escritura, menor consumo energético y resistencia a golpes. Ha reemplazado progresivamente a los discos duros tradicionales en laptops y servidores. Es clave para mejorar el rendimiento general de un sistema.

Dispositivos de Almacenamiento.

Son equipos o soportes que permiten guardar datos de forma temporal o permanente. Incluyen discos duros, memorias USB, CD/DVD, tarjetas SD y almacenamiento en la nube. Varían en capacidad, velocidad y portabilidad. Son fundamentales para la preservación y acceso a la información digital.

Dispositivos de Comunicación.

Son equipos de hardware que permiten la transmisión y recepción de información entre sistemas. Ejemplos incluyen módems, routers, switches y tarjetas de red. Facilitan la conexión entre computadoras, redes locales e internet. Son indispensables en telecomunicaciones y en la infraestructura de redes modernas.

Dispositivos de Entrada.

Son herramientas que permiten al usuario introducir datos en un sistema informático. Entre los más comunes están el teclado, el ratón, el escáner y el micrófono. Transforman acciones humanas o señales físicas en información procesable por la computadora. Son el punto de interacción inicial entre el usuario y la máquina.

Dispositivos de Procesamiento.

Componentes internos de un sistema que transforman y manipulan datos para generar resultados. El principal ejemplo es la Unidad Central de Procesamiento (CPU), junto con coprocesadores y tarjetas gráficas (GPU). Determinan la velocidad y eficiencia de los cálculos. Son considerados el “cerebro” del computador.

Dispositivos de Salida.

Son aquellos que muestran, transmiten o reproducen la información procesada por un sistema. Incluyen monitores, impresoras, altavoces y proyectores. Permiten al usuario recibir resultados en diferentes formatos: visuales, sonoros o físicos. Representan la etapa final del ciclo de procesamiento de datos.

Dispositivos Portátiles.

Equipos electrónicos diseñados para facilitar la movilidad y el acceso a la información en cualquier lugar. Ejemplos: laptops, tablets y smartphones. Combinan portabilidad con potencia de procesamiento. Son fundamentales en la era digital, permitiendo trabajar, comunicarse y entretenerse en movimiento.

Dispositivos Smart.

Aparatos que integran conectividad a internet, sensores e inteligencia artificial para realizar tareas automatizadas. Ejemplos: televisores inteligentes, asistentes virtuales y sistemas de domótica. Mejoran la experiencia del usuario al permitir control remoto y personalización. Son parte del ecosistema del Internet de las Cosas (IoT).

Dispositivos VR/AR.

Tecnologías que permiten experiencias inmersivas de realidad virtual (VR) o aumentada (AR). Incluyen visores, guantes hápticos y gafas inteligentes. La VR sumerge al usuario en entornos digitales, mientras que la AR combina el mundo físico con información digital. Se aplican en videojuegos, educación, medicina y entrenamiento industrial.

Dispositivos Wearables.

Tecnología vestible que se integra en objetos de uso diario, como relojes inteligentes, pulseras de actividad o gafas AR. Recopilan datos biométricos y ofrecen funciones de conectividad. Su uso es común en salud, deporte y estilo de vida. Representan la convergencia entre moda y tecnología.

Distributed Ledger Technology (DLT).

Tecnología que permite registrar transacciones de manera descentralizada y segura en múltiples nodos. La blockchain es su forma más conocida. Elimina la necesidad de intermediarios, aumentando transparencia y confianza. Se aplica en criptomonedas, trazabilidad de cadenas de suministro y contratos inteligentes.

DNS (Domain Name System).

Sistema que traduce nombres de dominio legibles (como www.ejemplo.com) en direcciones IP que las computadoras entienden. Es fundamental para la navegación en internet. Funciona de manera jerárquica y distribuida a través de servidores DNS. Sin este sistema, los usuarios tendrían que memorizar largas secuencias numéricas.

Documento Digital.

Archivo electrónico que contiene información en formato digital, como PDF, DOCX o imágenes escaneadas. Puede ser generado directamente en computadoras o mediante procesos de digitalización. Su validez legal depende de normas de firma electrónica y certificación. Facilita la gestión documental, reduciendo costos y mejorando la accesibilidad.

Docker.

Plataforma de software que permite crear, desplegar y ejecutar aplicaciones en contenedores. Estos contenedores aíslan las aplicaciones junto con sus dependencias, garantizando portabilidad entre distintos entornos. Facilita la automatización, escalabilidad y consistencia en el desarrollo de software. Es ampliamente usado en DevOps y cloud computing.

DOCS (Google Docs).

Aplicación en línea de Google que permite la creación y edición colaborativa de documentos de texto. Ofrece almacenamiento en la nube mediante Google Drive y soporte para edición en tiempo real. Facilita la productividad en equipo, con historial de cambios y compatibilidad con múltiples formatos. Es parte de Google Workspace.

Documentación Técnica.

Conjunto de documentos que describen el diseño, uso y mantenimiento de un sistema o software. Incluye manuales de usuario, guías de instalación, diagramas de arquitectura y especificaciones técnicas. Es esencial para garantizar comprensión, soporte y escalabilidad. Facilita la comunicación entre desarrolladores, usuarios y administradores.

Dominio de Internet.

Nombre único que identifica una dirección en la red, facilitando el acceso a sitios web sin necesidad de recordar direcciones IP. Se compone de niveles jerárquicos, como el dominio de nivel superior (.com, .org) y el nombre elegido por el usuario. Es gestionado por organizaciones como ICANN.

Dominio Web.

Nombre único que identifica un sitio en internet y que se traduce a una dirección IP a través del DNS. Se compone de niveles jerárquicos, como el dominio de nivel superior (.com, .org, .net) y el nombre elegido por el propietario. Facilita el acceso a páginas sin necesidad de recordar números IP. Su gestión está regulada por organizaciones como ICANN.

Driver de Dispositivo.

Programa que permite la comunicación entre el sistema operativo y un hardware específico (como impresoras, tarjetas gráficas o cámaras). Actúa como traductor, interpretando las instrucciones del software para que el hardware las ejecute correctamente. Es esencial para el funcionamiento de cualquier periférico y debe estar actualizado para garantizar estabilidad y seguridad.

DSL (Digital Subscriber Line).

Tecnología que utiliza las líneas telefónicas de cobre para transmitir datos digitales a alta velocidad sin interferir con las llamadas de voz. Fue uno de los primeros métodos de acceso a internet de banda ancha. Ofrece velocidades superiores al dial-up tradicional, aunque inferiores a la fibra óptica. Sigue siendo usada en áreas donde no llega otra infraestructura.

DTD (Document Type Definition).

Especificación usada en XML para definir la estructura y reglas de un documento. Permite validar que un archivo cumpla con un formato estándar, definiendo elementos, atributos y jerarquías. Es fundamental en la interoperabilidad de sistemas que intercambian información estructurada. Fue ampliamente usado antes de la adopción de XML Schema.

Dual Boot.

Configuración que permite instalar y ejecutar más de un sistema operativo en el mismo equipo. El usuario elige cuál arrancar al inicio. Se utiliza comúnmente para combinar Windows con distribuciones Linux. Es útil para pruebas, desarrollo o aprovechar las ventajas de distintos sistemas en un solo dispositivo.

Dual Channel.

Tecnología de memoria RAM que mejora el rendimiento al permitir que dos módulos funcionen en paralelo. Duplica el ancho de banda de transferencia de datos entre la memoria y el procesador. Se activa al instalar pares de módulos idénticos en las ranuras correspondientes. Es común en PCs de alto rendimiento y gaming.

Dump de Memoria.

Copia o volcado del contenido de la memoria de un sistema en un momento específico. Se utiliza para depuración y análisis forense, ya que permite revisar el estado de procesos y errores. Es fundamental en la detección de fallos críticos de software y ciberseguridad. Puede ser parcial o completo según la necesidad.

Dysk Encryption (Cifrado de Disco).

Método de seguridad que protege los datos de un dispositivo de almacenamiento mediante algoritmos criptográficos. Solo los usuarios autorizados con la clave correcta pueden acceder a la información. Previene fugas de datos en caso de robo o pérdida. Ejemplos: BitLocker, VeraCrypt y FileVault.

Dynamic Host Configuration Protocol (DHCP).

Protocolo de red que asigna direcciones IP automáticamente a los dispositivos conectados. Simplifica la administración de redes al evitar configuraciones manuales. Los servidores DHCP entregan parámetros como IP, máscara de subred y puerta de enlace. Es un estándar esencial en redes locales e internet.

Dynamic Link Library (DLL).

Archivos en Windows que contienen funciones y rutinas que pueden ser utilizadas por múltiples programas. Facilitan la modularidad y la reutilización de código. Mejoran la eficiencia al compartir recursos, pero también son un vector común de errores y vulnerabilidades. Son una parte clave del ecosistema de Windows.

Dynamic Programming (Programación Dinámica).

Técnica de diseño de algoritmos que resuelve problemas dividiéndolos en subproblemas más pequeños, almacenando sus soluciones para evitar cálculos repetitivos. Se aplica en optimización, grafos y secuencias. Ejemplos clásicos: el problema de la mochila, Fibonacci y rutas más cortas. Mejora la eficiencia reduciendo la complejidad computacional.

Dynamic Query Optimization.

Proceso en el que un motor de bases de datos ajusta dinámicamente la ejecución de consultas para mejorar el rendimiento. Analiza estadísticas de los datos y el estado del sistema en tiempo real. Permite adaptarse a cambios en las condiciones de la base de datos. Es clave en sistemas con grandes volúmenes de información.

Dynamic Routing.

Método de enrutamiento en redes que utiliza protocolos para ajustar automáticamente las rutas de los paquetes según la topología de la red. Protocolos como OSPF, RIP y BGP lo implementan. Aumenta la resiliencia y escalabilidad frente a cambios o fallos en la red. Contrasta con el enrutamiento estático.

Dynamic Testing.

Tipo de prueba de software que se realiza ejecutando el código en un entorno real o simulado. Evalúa funcionalidad, rendimiento y seguridad. Incluye pruebas unitarias, de integración, de sistema y de aceptación. A diferencia de las pruebas estáticas, detecta errores en la ejecución real del programa.

DynDNS (Dynamic DNS).

Servicio que actualiza automáticamente los registros DNS de un dominio cuando cambia la dirección IP pública del usuario. Es útil para conexiones domésticas con IP dinámica y acceso remoto a servidores. Permite mantener servicios accesibles sin necesidad de contratar IP fija. Ejemplo: No-IP, DuckDNS.

E-Business.

Actividad empresarial que aprovecha los medios digitales y las TIC para gestionar procesos internos, operaciones comerciales y relaciones con clientes o proveedores. Va más allá del e-commerce, pues incluye marketing, logística, servicio al cliente y gestión organizacional en entornos digitales.

E-Commerce.

Modalidad de comercio electrónico que permite la compra y venta de bienes o servicios a través de internet. Utiliza plataformas digitales y medios de pago electrónicos. Puede ser B2C (empresa a consumidor), B2B (empresa a empresa) o C2C (consumidor a consumidor).

E-Learning.

Modelo de educación a distancia que emplea internet y tecnologías digitales para impartir clases, cursos o capacitaciones. Permite el aprendizaje asincrónico o sincrónico mediante plataformas como Moodle, Blackboard o Google Classroom.

Edge AI.

Aplicación de inteligencia artificial directamente en dispositivos perimetrales (edge devices), sin depender de la nube. Se usa en cámaras inteligentes, vehículos autónomos o sensores IoT, mejorando la rapidez en la toma de decisiones al reducir la latencia.

Editor de Código.

Programa especializado para escribir, depurar y gestionar código fuente en diversos lenguajes de programación. Incluye funciones como resaltado de sintaxis, autocompletado y depuración. Ejemplos: Visual Studio Code, Sublime Text, Atom.

Editor de Texto.

Aplicación básica para crear y modificar texto plano, sin formato. Se usa principalmente para edición rápida o archivos de configuración. Ejemplos: Bloc de notas (Windows), Nano y Vim (Linux).

Editor Hexadecimal.

Software que permite visualizar y modificar datos en formato hexadecimal. Es usado en tareas de bajo nivel, como edición de firmware, depuración de memoria y análisis forense digital.

Eficiencia Algorítmica.

Medida del rendimiento de un algoritmo en términos de consumo de recursos como tiempo de ejecución y memoria. Se suele expresar mediante notación Big O (O(n), O (log n), etc.) para comparar soluciones.

ElasticSearch.

Motor de búsqueda y análisis de datos basado en Lucene. Permite indexar grandes volúmenes de información y realizar consultas rápidas y complejas. Es ampliamente usado en aplicaciones de monitoreo, análisis de logs y big data.

Electrónica Digital.

Rama de la electrónica que trabaja con señales discretas en formato binario (0 y 1). Es la base de la informática y los sistemas digitales, permitiendo el diseño de computadoras, microprocesadores y dispositivos electrónicos modernos.

Emulación de Hardware.

Técnica que permite simular el funcionamiento de un dispositivo físico mediante software. Es usada en pruebas de sistemas, videojuegos retro o cuando se quiere ejecutar aplicaciones diseñadas para hardware distinto.

Encriptación.

Proceso criptográfico que convierte información en un formato ilegible para quienes no tengan la clave de descifrado. Es fundamental en la seguridad informática para proteger datos en tránsito o almacenados.

EndPoint.

Dispositivo final que se conecta a una red, como computadoras, teléfonos móviles, impresoras o cámaras IP. Representa un punto vulnerable en ciberseguridad, por lo que requiere protección mediante antivirus, firewalls o gestión de dispositivos.

Ensayo de Penetración (Pentesting).

Práctica de seguridad que simula ataques reales contra un sistema para identificar vulnerabilidades y evaluar su nivel de protección. Se puede realizar de forma manual o automatizada, siendo parte clave de la ciberseguridad preventiva.

ERP (Enterprise Resource Planning).

Sistema de planificación de recursos empresariales que integra distintas áreas de una organización (finanzas, logística, recursos humanos, producción). Mejora la eficiencia al centralizar procesos en una sola plataforma.

Escalabilidad.

Capacidad de un sistema para manejar un incremento de carga de trabajo (usuarios, transacciones, procesos) sin perder rendimiento. Puede lograrse mediante escalabilidad vertical (mejor hardware) u horizontal (más servidores).

Escáner 3D.

Dispositivo que captura la geometría tridimensional de un objeto físico, generando un modelo digital. Se emplea en ingeniería, medicina, videojuegos y manufactura aditiva (impresión 3D).

Esquema XML.

Definición estructural que valida el formato y contenido de documentos XML, asegurando que cumplan con reglas establecidas. Facilita la interoperabilidad entre sistemas al estandarizar datos.

Ethernet.

Tecnología de red cableada que permite la interconexión de dispositivos en redes locales (LAN). Utiliza cables de par trenzado o fibra óptica y protocolos estandarizados (IEEE 802.3). Es la base de la mayoría de las redes corporativas y domésticas.

ETL (Extract, Transform, Load).

Proceso de extracción, transformación y carga de datos en un almacén de datos. Permite integrar información desde múltiples fuentes, depurarla y prepararla para análisis en sistemas de inteligencia de negocios.

Excel.

Aplicación de hojas de cálculo desarrollada por Microsoft. Permite realizar operaciones matemáticas, gráficas, análisis de datos y automatización mediante macros. Es ampliamente usada en entornos empresariales, académicos y científicos.

Extranet.

Red privada que extiende el acceso de una intranet a usuarios externos autorizados, como clientes o proveedores. Facilita la colaboración segura entre organizaciones sin exponer todos los recursos internos.

Facebook.

Red social diseñada para conectar a personas, facilitando la interacción entre amigos, familiares, compañeros de trabajo y personas con intereses comunes.

Factor de Autenticación.

Elemento utilizado para verificar la identidad de un usuario. Se clasifica en tres categorías: algo que sabes (contraseña), algo que tienes (token, tarjeta) y algo que eres (huella digital, reconocimiento facial).

Failover.

Mecanismo de alta disponibilidad que transfiere automáticamente funciones críticas a un sistema secundario cuando la principal falla. Es común en servidores, bases de datos y sistemas de telecomunicaciones.

FAQ Tecnológicas.

Listado de preguntas frecuentes y sus respuestas sobre productos o servicios tecnológicos. Sirven para resolver dudas comunes de los usuarios y mejorar la experiencia de soporte técnico.

Fast Ethernet.

Versión de Ethernet que soporta velocidades de transmisión de hasta 100 Mbps. Fue una evolución clave frente al Ethernet original (10 Mbps) y se convirtió en estándar en redes de los años 90 y 2000.

FAT32.

Sistema de archivos desarrollado por Microsoft que organiza y gestiona datos en dispositivos de almacenamiento. Es compatible con múltiples sistemas operativos, aunque tiene limitaciones como archivos de máximo 4 GB.

Fibra Óptica.

Medio de transmisión que utiliza hilos de vidrio o plástico para enviar datos mediante pulsos de luz. Ofrece gran velocidad y ancho de banda, siendo esencial en telecomunicaciones e internet de alta velocidad.

Figma.

Herramienta de diseño de interfaces colaborativa en la nube. Permite a varios usuarios trabajar en tiempo real en prototipos, wireframes y maquetas de aplicaciones o sitios web. Su gran ventaja es que no requiere instalación local y facilita la integración entre equipos de diseño y desarrollo.

Firmware.

Tipo de software de bajo nivel integrado en dispositivos electrónicos que controla su funcionamiento básico. Está almacenado en memoria no volátil (ROM, EEPROM o flash) y permite que el hardware opere correctamente.

Firewall.

Sistema de seguridad de red que controla y filtra el tráfico entrante y saliente entre dispositivos y redes. Puede estar basado en hardware o software, y se configura con reglas para bloquear accesos no autorizados y permitir solo comunicaciones seguras.

Flask.

Framework ligero y flexible en Python para el desarrollo de aplicaciones web. Se centra en la simplicidad, permitiendo construir APIs y sistemas escalables con pocas dependencias. Es muy usado en prototipado rápido y proyectos que no requieren estructuras tan grandes como Django.

Flujo de Trabajo (Workflow).

Conjunto de procesos y tareas organizados en una secuencia lógica, frecuentemente automatizada, para lograr un objetivo específico. Es ampliamente utilizado en la gestión empresarial y en plataformas digitales de productividad.

Fog Computing.

Modelo de computación distribuida que extiende la computación en la nube acercando el procesamiento de datos a la red local o dispositivos perimetrales. Reduce la latencia y mejora el rendimiento en IoT y sistemas en tiempo real.

Forense Digital.

Disciplina de la ciberseguridad que investiga incidentes informáticos mediante la recolección, preservación y análisis de evidencias digitales. Se aplica en casos legales, auditorías y respuesta a incidentes.

Formato JSON.

Estructura ligera de intercambio de datos basada en texto, fácil de leer y escribir por humanos y procesar por máquinas. Es ampliamente usado en aplicaciones web y APIs para transmitir información estructurada.

Formato XML.

Lenguaje de marcado utilizado para estructurar, almacenar e intercambiar datos de forma legible tanto para personas como para sistemas. Su flexibilidad lo hace útil en configuración de software, documentos y servicios web.

Formateo de Disco.

Proceso que prepara un dispositivo de almacenamiento para guardar datos, creando un sistema de archivos. Puede ser de bajo nivel (estructuración física) o alto nivel (organización lógica para el sistema operativo).

Framework de Desarrollo.

Conjunto de herramientas, librerías y convenciones que facilitan la creación de aplicaciones de software. Aporta una estructura predefinida que acelera el desarrollo y fomenta buenas prácticas. Ejemplos: Django, Angular, Spring.

Frecuencia de Reloj.

Velocidad a la que un procesador ejecuta instrucciones, medida en hertzios (Hz). Un mayor valor indica más operaciones por segundo, aunque no siempre implica mejor rendimiento, ya que influyen también la arquitectura y núcleos del CPU.

Frontend Developer.

Profesional especializado en desarrollar la parte visual e interactiva de una aplicación o sitio web. Utiliza tecnologías como HTML, CSS y JavaScript, además de frameworks (React, Angular, Vue). Su objetivo es garantizar una buena experiencia de usuario (UX/UI).

FTP (File Transfer Protocol).

Protocolo de red que permite la transferencia de archivos entre computadoras a través de Internet o redes locales. Funciona en un modelo cliente-servidor y puede usar versiones seguras como FTPS o SFTP.

Full-Dúplex.

Modo de transmisión de datos que permite la comunicación simultánea en ambos sentidos entre dos dispositivos. Se diferencia del half-dúplex, donde solo se transmite en una dirección a la vez.

Función Hash.

Algoritmo que transforma datos de entrada en una cadena de longitud fija, generalmente en formato alfanumérico. Es ampliamente usado en seguridad informática para almacenar contraseñas, verificar integridad de archivos y en criptomonedas.

Gamificación.

La gamificación es la aplicación de dinámicas, mecánicas y elementos propios de los juegos en contextos no lúdicos, como educación, marketing o gestión empresarial. Su objetivo es aumentar la motivación, la participación y el compromiso de los usuarios. Se utilizan recompensas, niveles, puntos o retos como estímulos.

Gateway.

Un gateway es un dispositivo o software que conecta redes con diferentes protocolos de comunicación, permitiendo la interoperabilidad entre ellas. Actúa como traductor de información entre sistemas heterogéneos. Es ampliamente usado en telecomunicaciones, IoT y en la interconexión de sistemas empresariales.

Geolocalización.

La geolocalización es el proceso de identificar la ubicación geográfica exacta de un dispositivo o persona a través de tecnologías como GPS, torres celulares o redes Wi-Fi. Se utiliza en aplicaciones móviles, logística, marketing digital y seguridad.

GIMP.

GIMP (GNU Image Manipulation Program) es un editor de imágenes gratuito y de código abierto. Ofrece herramientas avanzadas para retoque fotográfico, ilustración digital y composición de gráficos. Es una alternativa popular a programas privativos como Adobe Photoshop, con soporte para extensiones y plugins.

Git.

Git es un sistema de control de versiones distribuido creado por Linus Torvalds en 2005. Permite registrar los cambios en el código fuente, coordinar el trabajo en equipo y mantener un historial de versiones del proyecto. Su arquitectura distribuida garantiza que cada usuario tenga una copia completa del repositorio.

GitHub.

GitHub es una plataforma de colaboración y hospedaje de repositorios Git en la nube. Ofrece herramientas para gestión de proyectos, control de versiones y desarrollo colaborativo. Permite a los equipos trabajar en ramas, revisar cambios mediante pull requests y automatizar flujos de integración.

GitHub Actions.

GitHub Actions es una herramienta de automatización nativa de GitHub que permite implementar flujos de integración y entrega continua (CI/CD). Con ella, los desarrolladores pueden configurar pipelines para compilar, probar y desplegar aplicaciones automáticamente. Su integración con repositorios facilita la creación de entornos DevOps eficientes. Además, soporta acciones personalizadas y reutilizables en YAML.

GitLab.

GitLab es una plataforma de desarrollo colaborativo basada en Git que integra control de versiones, gestión de proyectos y CI/CD en un solo entorno. A diferencia de GitHub, ofrece mayor flexibilidad para despliegues en servidores propios o en la nube.

Glitch.

Un glitch es un fallo técnico menor o inesperado en un sistema, software o hardware, que interrumpe momentáneamente su funcionamiento. A diferencia de los errores críticos, suele ser transitorio y no compromete el sistema en su totalidad. En videojuegos, los glitches a veces son explotados como trucos por los usuarios. En ingeniería, su análisis ayuda a mejorar la estabilidad de los sistemas.

GNU.

GNU (GNU’s Not Unix) es un proyecto iniciado en 1983 por Richard Stallman con el objetivo de desarrollar un sistema operativo completamente libre. Incluye compiladores, bibliotecas, utilidades y herramientas que constituyen el ecosistema del software libre.

GNU/Linux.

GNU/Linux es un sistema operativo libre que combina el kernel Linux con las herramientas del proyecto GNU. Es ampliamente utilizado en servidores, supercomputadoras, dispositivos móviles (Android) y entornos de escritorio. Su flexibilidad permite múltiples distribuciones, como Ubuntu, Fedora o Debian, cada una adaptada a diferentes necesidades.

Gob. TI (Gobernanza TI).

La Gobernanza de TI se refiere al marco de políticas, procesos y estructuras que garantizan que las Tecnologías de la Información estén alineadas con los objetivos estratégicos de la organización. Su finalidad es maximizar el valor del negocio, gestionar riesgos tecnológicos y cumplir con regulaciones. Se apoya en marcos como COBIT e ITIL para su implementación.

Google Analytics.

Google Analytics es una herramienta de análisis web gratuita que permite medir el tráfico, la interacción de usuarios y el rendimiento de sitios web o aplicaciones. Ofrece métricas clave como conversiones, tiempo de permanencia y origen del tráfico. Facilita la toma de decisiones estratégicas para marketing digital y mejora de la experiencia del usuario.

Google BigQuery.

Google BigQuery es un servicio de análisis de datos en la nube que permite realizar consultas SQL sobre grandes volúmenes de información. Está diseñado para escalabilidad y velocidad, gestionando automáticamente recursos de almacenamiento y cómputo. Se utiliza en proyectos de Big Data, Business Intelligence y aprendizaje automático.

Google Cloud Platform (GCP).

Google Cloud Platform es un conjunto de servicios en la nube ofrecidos por Google. Proporciona soluciones de computación, almacenamiento, inteligencia artificial y análisis de datos. Es ampliamente usado por empresas que buscan escalar aplicaciones, optimizar costos y garantizar seguridad en entornos digitales. Compite con plataformas como AWS y Azure.

Google Drive.

Google Drive es un servicio de almacenamiento en la nube que permite guardar, sincronizar y compartir archivos en tiempo real. Ofrece integración con herramientas como Google Docs, Sheets y Slides, facilitando la colaboración en línea. Su accesibilidad multiplataforma lo hace esencial para entornos educativos y empresariales.

Google Fonts.

Google Fonts es una biblioteca gratuita de tipografías para uso en proyectos web y de diseño. Permite integrar fácilmente estilos de texto mediante enlaces o APIs, garantizando compatibilidad y rendimiento en diferentes navegadores. Su amplia colección contribuye a mejorar la estética y accesibilidad en aplicaciones digitales.

Google Maps API.

Google Maps API es un conjunto de servicios que permiten integrar mapas interactivos y funciones de geolocalización en aplicaciones y sitios web. Proporciona herramientas para calcular rutas, mostrar ubicaciones y personalizar datos cartográficos. Es ampliamente utilizado en sectores como logística, transporte y turismo.

Google Meet.

Google Meet es una aplicación de videoconferencias desarrollada por Google, diseñada para entornos profesionales y educativos. Permite reuniones en línea seguras, con opciones de grabación, subtítulos automáticos y uso compartido de pantalla. Está integrada con Google Workspace, facilitando la colaboración en equipo.

Google Play Store.

Google Play Store es la tienda oficial de aplicaciones para dispositivos Android. Además de apps, ofrece libros, música, películas y otros contenidos digitales. Incluye sistemas de seguridad como Google Play Protect para garantizar la confiabilidad de las descargas. Es la principal vía de distribución de software en el ecosistema Android.

Google Search Console.

Google Search Console es una herramienta gratuita que permite a los administradores de sitios web monitorear y optimizar la presencia de sus páginas en los resultados de búsqueda de Google. Proporciona información sobre rendimiento, indexación, usabilidad móvil y errores técnicos.

Google Workspace.

Google Workspace es una suite de productividad basada en la nube que integra aplicaciones como Gmail, Docs, Sheets, Drive y Meet. Está diseñada para la colaboración en tiempo real, el trabajo remoto y la gestión empresarial. Ofrece seguridad avanzada, control administrativo y escalabilidad para organizaciones de todos los tamaños.

GPU (Unidad de Procesamiento Gráfico).

Una GPU es un procesador especializado en realizar cálculos de manera paralela, originalmente diseñado para renderizar gráficos en 2D y 3D. Actualmente se utiliza también en inteligencia artificial, minería de criptomonedas y simulaciones científicas. Su arquitectura permite procesar grandes volúmenes de datos con alta eficiencia.

GPU Cloud Computing.

El GPU Cloud Computing se refiere al uso de unidades de procesamiento gráfico en entornos de computación en la nube. Permite ejecutar aplicaciones intensivas en datos, como entrenamiento de modelos de machine learning, sin necesidad de hardware físico dedicado. Empresas como Google, Amazon y Microsoft ofrecen este servicio.

GPU Virtualizada.

La GPU Virtualizada es una tecnología que permite dividir una unidad de procesamiento gráfico física en múltiples instancias virtuales. Esto optimiza los recursos en entornos con múltiples usuarios o aplicaciones. Es usada en escritorios virtuales, aplicaciones 3D y simulaciones de ingeniería en la nube.

Grafana.

Grafana es una plataforma de código abierto para la monitorización y visualización de métricas. Se integra con múltiples fuentes de datos como Prometheus, Elasticsearch o MySQL. Ofrece paneles interactivos en tiempo real, útiles en DevOps, análisis de sistemas y observabilidad empresarial.

GraphQL.

GraphQL es un lenguaje de consulta de datos desarrollado por Facebook que permite a los clientes solicitar exactamente la información que necesitan desde una API. Mejora la eficiencia frente a REST al reducir la sobrecarga de datos innecesarios. Se utiliza ampliamente en el desarrollo web y móvil moderno.

Green IT.

Green IT se refiere a las prácticas y estrategias que buscan reducir el impacto ambiental del uso de las tecnologías de la información. Incluye eficiencia energética, reciclaje de equipos, reducción de emisiones y optimización de recursos en centros de datos. Contribuye a la sostenibilidad y responsabilidad corporativa.

Grey Hat Hacker.

Un Grey Hat Hacker es un experto en ciberseguridad que actúa en un punto intermedio entre los hackers éticos (White Hat) y los maliciosos (Black Hat). Puede identificar vulnerabilidades sin autorización, pero sin fines destructivos, buscando a veces reconocimiento o recompensas. Su accionar plantea dilemas éticos y legales.

Grid Computing.

El Grid Computing es un modelo de computación distribuida que conecta múltiples equipos para trabajar como un superordenador virtual. Se utiliza en proyectos que requieren gran capacidad de procesamiento, como simulaciones científicas, análisis genómico y modelado climático. Aprovecha recursos subutilizados en diferentes ubicaciones.

Grupo de Usuarios de Linux (LUG).

Un Grupo de Usuarios de Linux (LUG) es una comunidad organizada de entusiastas y profesionales que promueven el uso del sistema operativo Linux. Estos grupos ofrecen talleres, charlas, soporte técnico y espacios de colaboración. Su propósito es fomentar el aprendizaje, la cooperación y la adopción de software libre.

GUI (Graphical User Interface).

La GUI es la interfaz gráfica de usuario que permite interactuar con un sistema o aplicación a través de elementos visuales como ventanas, iconos, botones y menús. Reemplazó la necesidad de comandos exclusivamente de texto, mejorando la accesibilidad y experiencia de usuario. Es fundamental en sistemas operativos modernos y software interactivo.

Gestión de Activos de TI.

La Gestión de Activos de TI es el proceso de supervisión y optimización de los recursos tecnológicos dentro de una organización. Incluye hardware, software, licencias y servicios asociados. Su propósito es maximizar el valor de los activos, reducir costos, mejorar la seguridad y cumplir normativas de control.

Gestión de Cambios.

La Gestión de Cambios es una metodología enfocada en controlar las modificaciones de los sistemas de TI. Su objetivo es implementar mejoras o correcciones sin afectar la estabilidad de los servicios. Se basa en planificación, evaluación de riesgos y autorización antes de ejecutar cambios en producción.

Gestión de Configuración.

La Gestión de Configuración asegura que los sistemas de TI mantengan una configuración definida y controlada. Se centra en identificar componentes, registrar cambios y garantizar consistencia en entornos tecnológicos. Es esencial en entornos de desarrollo, DevOps y operaciones críticas.

Gestión de Incidentes.

La Gestión de Incidentes comprende los procedimientos destinados a restaurar rápidamente los servicios de TI tras una interrupción. Se enfoca en minimizar el impacto en el negocio, documentar causas y mejorar la respuesta futura. Es una de las prácticas centrales de ITIL para garantizar continuidad operativa.

Gestión de Incidencias.

La Gestión de Incidencias se ocupa de identificar, registrar y resolver problemas técnicos en los sistemas informáticos. Aunque a menudo se confunde con la gestión de incidentes, su enfoque incluye la trazabilidad, categorización y análisis de tendencias para prevenir problemas recurrentes.

Gestión de Parches.

La Gestión de Parches es el proceso de identificar, probar e implementar actualizaciones en software y sistemas operativos. Su objetivo principal es corregir vulnerabilidades de seguridad, mejorar el rendimiento y garantizar la estabilidad. Es una práctica crítica en ciberseguridad, ya que evita que atacantes exploten fallos conocidos.

Gestión de Proveedores de TI.

La Gestión de Proveedores de TI consiste en supervisar contratos, acuerdos y servicios que terceros ofrecen a una organización. Busca asegurar que los proveedores cumplan con niveles de servicio (SLA), costos acordados y estándares de calidad. También ayuda a reducir riesgos asociados a la dependencia de servicios externos.

Gestión del Conocimiento.

La Gestión del Conocimiento es una estrategia organizacional que recopila, organiza y distribuye información útil dentro de una empresa. Su propósito es facilitar la toma de decisiones, mejorar la innovación y preservar el know-how. Utiliza herramientas tecnológicas como intranets, bases de datos de experiencias y sistemas colaborativos.

Gestión Documental.

La Gestión Documental es el conjunto de procesos y herramientas que permiten capturar, organizar, almacenar y acceder a documentos en formato digital. Mejora la eficiencia administrativa, facilita la trazabilidad de la información y cumple con normativas de archivo. Se aplica en sectores como gobierno, salud y educación.

Gestión de Redes Sociales.

La Gestión de Redes Sociales es la administración estratégica de cuentas en plataformas como Facebook, Instagram, LinkedIn o X (Twitter). Implica la creación de contenidos, análisis de métricas, interacción con usuarios y campañas publicitarias. Su objetivo es fortalecer la presencia digital de marcas y organizaciones.

Hackathon.

Un hackathon es un evento intensivo y colaborativo donde programadores, diseñadores y otros profesionales trabajan en equipos para desarrollar soluciones tecnológicas en un periodo corto, generalmente entre 24 y 72 horas.

Hacker Ético.

El hacker ético es un especialista en ciberseguridad que utiliza sus conocimientos avanzados en sistemas, redes y programación para identificar vulnerabilidades antes de que puedan ser explotadas por atacantes maliciosos.

Hadoop.

Hadoop es un marco de software de código abierto diseñado para almacenar y procesar grandes volúmenes de datos distribuidos en clusters de servidores. Su núcleo incluye HDFS (Hadoop Distributed File System) para almacenamiento y MapReduce para procesamiento paralelo.

Handover en Redes.

El handover es el proceso mediante el cual una conexión móvil (como una llamada telefónica o sesión de datos) se transfiere de una celda a otra dentro de una red de telecomunicaciones.

Hardware.

El hardware es el conjunto de componentes físicos y tangibles que forman parte de un sistema informático. Incluye dispositivos de entrada (teclado, ratón), salida (pantalla, impresora), almacenamiento (discos duros, SSD) y procesamiento (CPU, GPU, RAM). Su correcto funcionamiento depende de la integración con el software. Sin hardware no existiría la base material para ejecutar programas ni interactuar con los sistemas digitales.

Hardware Abierto.

El hardware abierto (open hardware) es aquel cuyos diseños, esquemas y especificaciones son de acceso público, permitiendo a cualquier persona estudiarlos, modificarlos, fabricarlos y distribuirlos.

Hardware de Red.

El hardware de red comprende los dispositivos físicos que permiten la interconexión y comunicación entre equipos en una red informática. Incluye routers, switches, hubs, firewalls, módems y puntos de acceso Wi-Fi.

Hardware Híbrido.

El hardware híbrido es una combinación de tecnologías analógicas y digitales dentro de un mismo dispositivo o sistema. Este enfoque se utiliza para aprovechar lo mejor de ambos mundos: la precisión y continuidad de lo analógico junto con la velocidad y flexibilidad de lo digital.

Hash.

Un hash es el resultado de aplicar una función criptográfica que transforma un bloque de datos de cualquier tamaño en una cadena alfanumérica de longitud fija.

Hashcat.

Hashcat es una de las herramientas más populares para la recuperación de contraseñas mediante técnicas como fuerza bruta, ataques de diccionario y reglas personalizadas.

Haskell.

Lenguaje de programación funcional puro, de alto nivel y fuertemente tipado. Se caracteriza por el uso de funciones inmutables, evaluación perezosa y un sistema de tipos avanzado. Es muy usado en investigación, banca y aplicaciones críticas donde la corrección es esencial.

HDMI.

Interfaz digital para transmitir video y audio en alta definición entre dispositivos. Sustituyó a conexiones analógicas como VGA y RCA. Es el estándar en televisores, monitores, consolas y equipos multimedia por su calidad y soporte de múltiples formatos.

Headless CMS.

Sistema de gestión de contenidos que separa el backend del frontend. Proporciona la información vía APIs, permitiendo usarla en aplicaciones web, móviles o IoT. Aporta flexibilidad frente a los CMS tradicionales como WordPress.

Heatmap.

Representación gráfica que usa colores para mostrar intensidad de uso, interacción o distribución de datos. En UX se aplica para analizar clics, desplazamientos o zonas de mayor atención en una interfaz. Es útil en optimización web y marketing digital.

Heurística.

Método de resolución de problemas basado en aproximaciones prácticas, no en soluciones exactas. Se usa en inteligencia artificial, optimización y búsqueda de soluciones complejas. Ayuda a reducir tiempos y recursos en problemas NP-completos.

Heurística de Usabilidad.

Conjunto de principios que guían el diseño de interfaces fáciles de usar, propuestos por Jakob Nielsen. Incluyen consistencia, visibilidad del estado del sistema y prevención de errores. Son fundamentales en pruebas de usabilidad.

Heurística en Algoritmos.

Técnica empleada en algoritmos para encontrar soluciones aproximadas a problemas difíciles. Se aplica en áreas como búsqueda en grafos, planificación y optimización. Aunque no garantiza la mejor solución, logra resultados eficientes en menos tiempo.

Hibernate.

Framework de Java para mapeo objeto-relacional (ORM). Permite trabajar con bases de datos mediante objetos, evitando escribir consultas SQL manuales. Mejora la productividad en aplicaciones empresariales y asegura portabilidad entre motores de BD.

HIPAA.

Ley estadounidense que regula la seguridad y privacidad de datos médicos electrónicos. Establece estándares para proteger información sensible en hospitales, aseguradoras y proveedores de salud. Su cumplimiento es obligatorio en sistemas sanitarios de EE. UU.

Hipervisor.

Software o firmware que permite ejecutar múltiples máquinas virtuales en un mismo hardware físico. Administra recursos como CPU, memoria y almacenamiento. Existen de tipo 1 (bare-metal) y tipo 2 (sobre un sistema operativo).

Hive.

Herramienta del ecosistema Hadoop que permite consultar y analizar grandes volúmenes de datos usando un lenguaje similar a SQL (HQL). Facilita la explotación de datos sin necesidad de programar en MapReduce.

Hosting.

Servicio que permite alojar sitios web y aplicaciones en servidores accesibles a través de Internet. Incluye opciones compartidas, VPS, dedicados y en la nube. Es esencial para la disponibilidad online de servicios digitales.

Hot Backup.

Copia de seguridad realizada mientras el sistema sigue en funcionamiento. Es común en bases de datos y servidores críticos donde no es posible detener la operación. Garantiza continuidad del servicio sin pérdida de datos.

Hotfix.

Parche rápido que corrige errores críticos en software. Generalmente se aplica sin necesidad de una actualización completa. Se utiliza para resolver vulnerabilidades o fallos graves en producción.

Hotspot Wi-Fi.

Punto de acceso inalámbrico que permite a los dispositivos conectarse a Internet. Puede estar en aeropuertos, cafeterías o compartido desde un móvil. Es clave para la conectividad en movilidad.

HTML.

Lenguaje de marcado estándar usado para estructurar contenido en la web. Define elementos como títulos, párrafos, enlaces e imágenes. Es la base de todas las páginas junto con CSS y JavaScript.

HTML5.

Versión más reciente de HTML que incluye soporte nativo para audio, video, gráficos vectoriales y almacenamiento local. Facilita el desarrollo de aplicaciones web interactivas sin necesidad de plugins externos.

HTTP.

Protocolo de comunicación que permite la transferencia de información en la web. Funciona bajo un modelo cliente-servidor y opera en el puerto 80. Es la base del acceso a sitios web.

HTTP/2.

Evolución de HTTP que mejora la eficiencia en la transferencia de datos. Permite multiplexación de solicitudes, compresión de cabeceras y mayor velocidad en la carga de páginas web.

HTTPS.

Versión segura de HTTP que utiliza cifrado SSL/TLS para proteger la comunicación entre cliente y servidor. Garantiza confidencialidad, integridad y autenticidad en transacciones online. Es esencial en comercio electrónico y banca digital.

Hub de Red.

Dispositivo de red que conecta múltiples equipos en una misma red local. Reenvía los datos recibidos a todos los puertos, lo que puede generar colisiones. Ha sido reemplazado en gran parte por switches.

Human-Computer Interaction (HCI).

Disciplina que estudia la interacción entre personas y sistemas informáticos. Se centra en mejorar la usabilidad, accesibilidad y experiencia de usuario en interfaces digitales. Combina informática, diseño y psicología cognitiva.

Hybrid App.

Aplicación que combina tecnologías web (HTML, CSS, JavaScript) con funciones nativas del dispositivo. Se ejecuta en múltiples plataformas usando un solo código base. Frameworks comunes son Ionic y React Native.

Hyperledger.

Proyecto de código abierto de la Linux Foundation enfocado en desarrollar tecnologías blockchain para empresas. Incluye plataformas como Hyperledger Fabric y Sawtooth. Se orienta a soluciones seguras y escalables en negocios.

HashMap.

Estructura de datos basada en pares clave-valor. Usa funciones hash para acceder rápidamente a los elementos almacenados. Es ampliamente utilizada en Java y otros lenguajes para optimizar búsquedas y almacenamiento.

Heap.

Estructura de datos en forma de árbol binario que cumple la propiedad de orden parcial. Se usa en algoritmos de ordenamiento (HeapSort) y en gestión de memoria dinámica. Permite acceso eficiente al valor máximo o mínimo.

Helm.

Herramienta de gestión de paquetes para Kubernetes. Facilita la instalación, actualización y administración de aplicaciones en contenedores mediante “charts”. Automatiza despliegues y reduce la complejidad en entornos cloud-native.

HDFS (Hadoop Distributed File System).

Hping.

Herramienta de línea de comandos usada en pruebas de seguridad y redes. Permite enviar paquetes TCP/IP personalizados para pruebas de firewall, detección de puertos y auditorías. Es popular en pentesting avanzado.

Honeypot.

Sistema de seguridad diseñado como trampa para atraer a atacantes. Simula vulnerabilidades para analizar sus técnicas y reforzar defensas. Se usa en investigación y detección de intrusiones.

HSL (Hue, Saturation, Lightness).

Modelo de color usado en gráficos digitales. Representa colores mediante matiz (tono), saturación (intensidad) y luminosidad (brillo). Es más intuitivo para el diseño que el modelo RGB.

HMAC (Hash-based Message Authentication Code).

Mecanismo criptográfico que combina una función hash con una clave secreta. Se utiliza para verificar integridad y autenticidad de mensajes en protocolos como TLS e IPsec.

Horizontal Scaling.

Estrategia de escalabilidad en sistemas distribuidos que consiste en añadir más nodos o servidores en lugar de aumentar recursos de un solo equipo. Mejora rendimiento y tolerancia a fallos.

HPCC (High-Performance Computing Cluster).

Infraestructura de cómputo formada por múltiples servidores trabajando en paralelo. Se usa en simulaciones científicas, análisis de datos masivos y modelado complejo. Optimiza potencia y velocidad de cálculo.

HQL (Hive Query Language).

Lenguaje de consulta similar a SQL usado en Apache Hive. Permite realizar consultas y análisis sobre grandes volúmenes de datos almacenados en Hadoop. Facilita el trabajo de analistas sin experiencia en MapReduce.

Hyper-V.

Hipervisor desarrollado por Microsoft que permite ejecutar máquinas virtuales en sistemas Windows. Ofrece aislamiento, administración centralizada y soporte para múltiples sistemas operativos invitados.

Hot Reload.

Técnica de desarrollo que actualiza cambios en el código sin necesidad de reiniciar toda la aplicación. Es común en frameworks como Flutter o React, acelerando la productividad de los desarrolladores.

Handshake.

Proceso de negociación en protocolos de red (como TLS o TCP) donde cliente y servidor acuerdan parámetros de comunicación. Garantiza sincronización y seguridad antes de transmitir datos.

Head-Mounted Display (HMD).

Dispositivo de visualización montado en la cabeza que proyecta imágenes directamente frente a los ojos. Se usa en realidad virtual (VR) y aumentada (AR), proporcionando experiencias inmersivas en simulación, juegos y capacitación.

IaaS (Infrastructure as a Service).

Es un modelo de servicio en la nube que ofrece recursos de infraestructura como servidores, almacenamiento y redes a través de internet. Permite a las empresas escalar fácilmente sus sistemas sin invertir en hardware físico.

IA Conversacional.

IA Explicable (XAI).

Es un campo de la inteligencia artificial enfocado en hacer comprensibles las decisiones tomadas por modelos complejos como redes neuronales. Busca transparencia y confianza al mostrar las razones detrás de una predicción o clasificación.

IA Fuerte.

Concepto teórico de inteligencia artificial que tendría capacidades cognitivas comparables a las humanas. Implica la habilidad de razonar, aprender de manera autónoma y adaptarse a contextos desconocidos.

IA Predictiva.

Rama de la inteligencia artificial que analiza datos históricos y patrones para anticipar comportamientos futuros. Utiliza algoritmos de machine learning y minería de datos para generar pronósticos.

IBM Cloud.

Es la plataforma de servicios en la nube de IBM, que integra modelos IaaS, PaaS y SaaS. Ofrece soluciones para inteligencia artificial, blockchain, IoT y análisis de datos. Está orientada a empresas que requieren seguridad, cumplimiento normativo y escalabilidad.

ICMP (Internet Control Message Protocol).

Icono de Aplicación.

Identidad Digital.

IEEE (Institute of Electrical and Electronics Engineers).

Organización internacional dedicada al avance de la tecnología eléctrica, electrónica y de telecomunicaciones. Desarrolla estándares globales como Wi-Fi (802.11) y Ethernet (802.3).

IFTTT (If This Then That).

Plataforma que permite automatizar acciones entre diferentes aplicaciones y servicios. Funciona mediante reglas condicionales: si ocurre un evento (This), entonces se ejecuta una acción (That).

IGMP (Internet Group Management Protocol).

Protocolo de comunicación utilizado en redes IP para gestionar grupos de multidifusión (multicast). Permite que dispositivos se unan o abandonen transmisiones grupales, optimizando el uso de ancho de banda.

IIS (Internet Information Services).

Servidor web desarrollado por Microsoft que permite alojar sitios, servicios y aplicaciones. Soporta protocolos como HTTP, HTTPS, FTP y SMTP. Es ampliamente usado en entornos empresariales que trabajan con tecnologías Windows y ASP.NET.

ILM (Information Lifecycle Management).

Estrategia que gestiona la información digital desde su creación hasta su eliminación. Clasifica datos según su valor, uso y cumplimiento normativo. Permite optimizar almacenamiento, reducir costos y garantizar seguridad de la información.

Imagen ISO.

Archivo que contiene una copia exacta de un sistema de archivos de un disco óptico, como CD o DVD. Se utiliza para distribuir sistemas operativos, software o copias de seguridad. Puede montarse en máquinas virtuales o grabarse en medios físicos.

Imagen Vectorial.

Representación gráfica basada en fórmulas matemáticas que definen puntos, líneas y curvas. Es escalable infinitamente sin pérdida de calidad, lo que la hace ideal para logotipos, ilustraciones y diseño gráfico.

Gráficos compuestos por una matriz de píxeles, donde cada píxel representa un color. Son comunes en fotografía digital y en formatos como JPEG, PNG y BMP.

IMAP (Internet Message Access Protocol).

Es un protocolo de correo electrónico que permite acceder y gestionar mensajes directamente en el servidor. A diferencia de POP3, no descarga automáticamente los correos, sino que sincroniza en tiempo real entre dispositivos.

Impresión 3D.

Impresión en la Nube.

Servicio que permite enviar documentos a impresoras a través de internet sin necesidad de estar en la misma red local. Es útil en entornos empresariales y móviles. Facilita la impresión remota desde diferentes dispositivos.

Impresora.

Dispositivo de salida que produce documentos físicos a partir de información digital. Puede ser de diferentes tipos: láser, inyección de tinta (inkjet) o térmica. Es un periférico básico en oficinas y hogares.

Impresora 3D.

Dispositivo especializado en fabricar objetos tridimensionales a partir de modelos digitales. Funciona aplicando la técnica de impresión 3D, depositando capas sucesivas de material.

Impresora Multifuncional.

Equipo que combina varias funciones en un solo dispositivo: impresión, escaneo, copiado y fax. Reduce costos y espacio en entornos de oficina. Puede conectarse en red para uso compartido.

Implementación de Software.

Proceso que consiste en instalar, configurar y poner en marcha un sistema informático dentro de una organización. Incluye migración de datos, pruebas de funcionamiento, capacitación de usuarios y ajustes de integración.

Infografía Digital.

Ingeniería de Requisitos.

Ingeniería de Telecomunicaciones.

Campo de la ingeniería que estudia, diseña y optimiza sistemas de transmisión y recepción de información. Incluye redes móviles, satélites, fibra óptica e internet.

Instancia en la Nube.

Instantánea de Sistema.

Integración Continua (CI).

Interactividad Web.

Internet de las Cosas (IoT).

Internet de las Cosas Médicas (IoMT).

Internet de los Vehículos (IoV).

Internet Explorer.

Fue un navegador web desarrollado por Microsoft y lanzado en 1995, incluido en el sistema operativo Windows. Durante años dominó el mercado, aunque posteriormente perdió terreno frente a competidores como Chrome y Firefox.

Internet Exchange Point (IXP).

Es una infraestructura física donde diferentes proveedores de servicios de internet e ISPs intercambian tráfico entre sus redes. Su función es optimizar la conectividad, reducir la latencia y mejorar la eficiencia del enrutamiento.

Internet Protocol (IP).

Es el protocolo fundamental que rige el direccionamiento y envío de datos en redes de computadoras. Cada dispositivo conectado posee una dirección IP que lo identifica de manera única.

Internet Protocol Security (IPsec).

Es un conjunto de protocolos diseñado para asegurar las comunicaciones IP mediante cifrado y autenticación. Funciona en la capa de red y protege la confidencialidad, integridad y autenticidad de los datos.

Internet Protocol Security VPN (IPsec VPN).

Es una red privada virtual que utiliza el protocolo IPsec para crear canales seguros de comunicación sobre internet. Proporciona cifrado de extremo a extremo, autenticación de usuarios y protección contra ataques de intermediarios.

Interoperabilidad.

Intrusión Cibernética.

Intranet.

Es una red privada interna utilizada dentro de una organización para compartir información y recursos. Funciona de manera similar a internet, pero con acceso restringido a los miembros autorizados.

IoT Analytics.

Es el proceso de análisis de los datos generados por dispositivos conectados en el Internet de las Cosas. Permite extraer información valiosa para optimizar procesos, predecir comportamientos y tomar decisiones basadas en datos.

IoT Edge.

Es un modelo de procesamiento de datos del IoT en la periferia de la red, cerca de los dispositivos que los generan. Reduce la dependencia de la nube al procesar información localmente, disminuyendo la latencia.

IOS.

Es el sistema operativo móvil desarrollado por Apple para dispositivos como iPhone y iPad. Destaca por su seguridad, estabilidad, rendimiento y ecosistema de aplicaciones optimizado.

IP Dinámica.

Es una dirección IP que cambia cada vez que un dispositivo se conecta a la red o después de cierto periodo de tiempo. Los proveedores de servicios de internet suelen asignarla automáticamente mediante DHCP.

IP Estática.

Es una dirección IP fija asignada permanentemente a un dispositivo dentro de una red. A diferencia de la dinámica, no cambia con el tiempo, lo que facilita el acceso remoto y la configuración de servicios.

IP Multicast.

Es una técnica de transmisión en la que los datos se envían a múltiples receptores de manera simultánea y eficiente. Permite optimizar el uso del ancho de banda en aplicaciones como videoconferencias, IPTV y distribución de software.

IP Pública.

Es la dirección IP que identifica de manera única a un dispositivo o red en internet y es accesible desde el exterior. Permite la comunicación entre sistemas ubicados en diferentes partes del mundo, siendo necesaria para servidores web, correos electrónicos y servicios en línea.

IPv4.

Es la cuarta versión del Protocolo de Internet y la más usada durante décadas para direccionar dispositivos en la red. Utiliza direcciones de 32 bits, lo que permite aproximadamente 4.3 mil millones de combinaciones.

IPv6.

Es la sexta versión del Protocolo de Internet y sucesora de IPv4, diseñada para solventar la escasez de direcciones. Utiliza direcciones de 128 bits, lo que permite una cantidad casi ilimitada de dispositivos conectados.

IRC (Internet Relay Chat).

Es un protocolo de comunicación en tiempo real creado en 1988 que permite conversaciones en grupo mediante canales de chat. Aunque su uso ha disminuido, fue pionero en el desarrollo de comunidades en línea y mensajería instantánea.

IRDA (Infrared Data Association).

Es un estándar de comunicación inalámbrica que utiliza luz infrarroja para transmitir datos entre dispositivos. Fue muy usado en los años noventa en laptops, teléfonos móviles y periféricos antes de la masificación del Bluetooth.

ISO 27001.

Es una norma internacional que establece requisitos para implementar y mantener un sistema de gestión de seguridad de la información. Define procesos, controles y buenas prácticas para proteger la confidencialidad, integridad y disponibilidad de los datos.

ISO 9001.

Es una norma internacional enfocada en la gestión de la calidad que establece criterios para optimizar procesos organizacionales. Busca garantizar que los productos y servicios cumplan consistentemente con las expectativas del cliente.

ISP (Proveedor de Servicios de Internet).

Iteración.

Jabber (XMPP).

JAR (Java Archive).

Un archivo JAR es un contenedor comprimido en formato ZIP que agrupa clases de Java, metadatos y recursos en un único paquete. Facilita la distribución y ejecución de aplicaciones completas o bibliotecas reutilizables.

Jasmine.

Jasmine es un framework de pruebas unitarias para JavaScript, diseñado para aplicaciones web. Utiliza una sintaxis simple y legible, basada en el enfoque de behavior-driven development (BDD).

JasperReports.

JasperReports es una biblioteca de Java usada para la generación de reportes dinámicos en distintos formatos como PDF, HTML, XLS, CSV y más.

Javadoc.

Javadoc es una herramienta incluida en el JDK que permite generar documentación automática de proyectos Java a partir de comentarios en el código fuente. Utiliza etiquetas específicas para describir clases, métodos, parámetros y excepciones.

Java.

Java es un lenguaje de programación orientado a objetos, creado por Sun Microsystems en 1995 (actualmente mantenido por Oracle). Su característica principal es la portabilidad, gracias a que se ejecuta sobre la JVM, bajo el lema Write Once, Run Anywhere.

JavaBeans.

JavaBeans son componentes de software reutilizables desarrollados en Java bajo un conjunto de convenciones estandarizadas. Incluyen atributos privados, métodos getters y setters, un constructor sin parámetros y soporte para serialización.

Java Card.

Java EE (Jakarta EE).

JavaFX es una plataforma para desarrollar interfaces gráficas avanzadas en Java. Ofrece componentes modernos, soporte multimedia, gráficos 2D/3D y FXML para diseño declarativo.

Java ME.

Java ME es una plataforma diseñada para dispositivos con recursos limitados como teléfonos móviles, PDAs y sistemas embebidos. Proporciona un entorno reducido de APIs y una JVM optimizada para correr en hardware de bajo consumo.

Java SE.

Java SE es la edición estándar del lenguaje Java, que incluye el núcleo de bibliotecas, el compilador, la máquina virtual y herramientas de desarrollo. Es la base sobre la cual se construyen otras ediciones como Java EE y Java ME.

Java Virtual Machine (JVM).

La JVM es la máquina virtual que ejecuta programas Java compilados en bytecode, lo que hace posible la portabilidad entre diferentes sistemas operativos.

JavaServer Faces (JSF).

JSF es un framework de Java para crear interfaces gráficas basadas en componentes en aplicaciones web. Está orientado al desarrollo de aplicaciones empresariales con soporte para validación, manejo de eventos y navegación entre vistas.

JavaServer Pages (JSP).

JSP es una tecnología Java que permite generar páginas web dinámicas embebiendo código Java dentro de HTML. Se compilan en servlets y son ejecutadas en el servidor, generando contenido que luego es enviado al navegador.

JavaScript.

JavaScript es un lenguaje de programación interpretado, esencial en el desarrollo web para añadir interactividad y dinamismo a las páginas.

JAX-RS (Java API for RESTful Web Services).

JAX-RS es una especificación de Java para desarrollar servicios web RESTful de forma sencilla. Utiliza anotaciones para mapear recursos a URLs, métodos HTTP y formatos de datos como JSON o XML.

JAX-WS (Java API for XML Web Services).

JAX-WS es una API de Java para crear y consumir servicios web basados en SOAP. Facilita la comunicación entre aplicaciones distribuidas usando XML como formato de intercambio de datos.

JDBC (Java Database Connectivity).

JDBC es una API de Java que permite conectar aplicaciones con bases de datos relacionales. Define interfaces estándar para ejecutar consultas SQL, actualizar registros y manejar transacciones.

JDK (Java Development Kit).

El JDK es el kit de desarrollo oficial para programar en Java, que incluye el compilador javac, la JVM, bibliotecas estándar y herramientas como Javadoc. Existen distintas distribuciones como OpenJDK y Oracle JDK, utilizadas en desarrollo profesional y académico.

Jenkins.

Jenkins es una herramienta de automatización de código abierto enfocada en integración continua (CI) y entrega continua (CD). Permite compilar, probar y desplegar software automáticamente mediante pipelines.

JHipster.

JIRA.

JIRA es una herramienta de Atlassian usada para la gestión de proyectos y seguimiento de incidencias. Está orientada a metodologías ágiles como Scrum y Kanban, facilitando la organización de tareas, sprints y reportes.

Joomla!.

Joomla! es un sistema de gestión de contenidos (CMS) de código abierto, desarrollado en PHP y MySQL. Permite crear y administrar sitios web dinámicos sin necesidad de conocimientos avanzados de programación.

JPA (Java Persistence API).

JPA es una especificación de Java para la gestión de datos en aplicaciones usando mapeo objeto-relacional (ORM). Permite trabajar con entidades Java en lugar de escribir directamente consultas SQL.

JRuby.

JRuby es una implementación del lenguaje Ruby sobre la JVM. Permite ejecutar aplicaciones Ruby en entornos Java y aprovechar la interoperabilidad entre ambos ecosistemas.

JSON (JavaScript Object Notation).

JSON es un formato ligero de intercambio de datos basado en texto, fácil de leer para humanos y procesar por máquinas. Representa estructuras de datos mediante pares clave-valor y listas ordenadas.

JTAG es un estándar para la verificación y depuración de hardware digital, especialmente microcontroladores y procesadores. Permite realizar pruebas a nivel de pines, cargar firmware y depurar software embebido directamente en el hardware.

JUnit.

JUnit es un framework de pruebas unitarias para Java. Permite automatizar la ejecución de pruebas, verificar resultados esperados y detectar errores en etapas tempranas del desarrollo.

Jupyter Notebook.

Jupyter Notebook es una herramienta interactiva usada en ciencia de datos, inteligencia artificial y aprendizaje automático. Permite combinar código ejecutable, visualizaciones y texto descriptivo en un mismo documento.

Jython.

Jython es una implementación del lenguaje Python sobre la JVM. Permite ejecutar código Python en entornos Java y acceder a librerías del ecosistema Java directamente desde Python.

Joystick.

La JVM es la máquina virtual encargada de ejecutar bytecode Java, garantizando la portabilidad de las aplicaciones en múltiples plataformas.

Kahoot.

Kahoot es una plataforma educativa en línea que permite crear y responder cuestionarios interactivos. Se utiliza en entornos académicos y empresariales para fomentar el aprendizaje colaborativo mediante dinámicas de juego.

Kafka (Apache Kafka).

Apache Kafka es una plataforma distribuida de mensajería y transmisión de datos en tiempo real. Está diseñada para manejar grandes volúmenes de eventos de manera rápida, escalable y tolerante a fallos.

Kali Linux.

Kali Linux es una distribución de Linux basada en Debian, especializada en pruebas de penetración y auditorías de seguridad informática.

Kali NetHunter es una plataforma de pruebas de penetración para dispositivos móviles basada en Android y desarrollada sobre Kali Linux.

Kanban.

Kanban es un método ágil de gestión de proyectos que utiliza tableros visuales para representar el flujo de trabajo. Cada tarea se organiza en tarjetas que avanzan por columnas.

Kdenlive.

Keras.

Keras es una biblioteca de deep learning de alto nivel para Python que proporciona una interfaz simple para crear redes neuronales. Inicialmente funcionaba sobre frameworks como TensorFlow y Theano, y hoy está integrada oficialmente en TensorFlow.

Kerberos.

Kerberos es un protocolo de autenticación en redes que permite a los usuarios y servicios comunicarse de manera segura en entornos distribuidos. Utiliza criptografía simétrica y un sistema de tickets emitidos por un servidor de autenticación central (KDC).

Kernel.

El Kernel es el núcleo del sistema operativo encargado de gestionar los recursos de hardware y proporcionar servicios básicos al software.

Un KPI es una métrica clave usada para evaluar el desempeño y éxito de un proceso, proyecto o sistema tecnológico. En el ámbito de software y TICS, pueden medir velocidad de despliegue, calidad del código, tiempo de respuesta de sistemas o satisfacción del usuario.

Un keylogger es un software o dispositivo malicioso diseñado para registrar las pulsaciones del teclado de un usuario. Se utiliza comúnmente en ataques de ciberseguridad para robar credenciales, datos bancarios o información confidencial.

Kick Off.

El Kick Off es la reunión inicial de un proyecto, donde se presentan los objetivos, roles, responsabilidades y el plan de trabajo. Es una práctica común en la gestión de proyectos siguiendo metodologías tradicionales o ágiles.

Kismet.

Kismet es una herramienta de código abierto utilizada para la detección, monitoreo y análisis de redes inalámbricas. Puede identificar puntos de acceso WiFi, dispositivos conectados y detectar intrusiones o vulnerabilidades.

Kit de Desarrollo de Software (SDK).

Un SDK es un conjunto de herramientas, bibliotecas, documentación y utilidades que permiten a los desarrolladores crear aplicaciones para una plataforma específica.

Klocwork.

Klocwork es una herramienta de análisis estático de código que permite identificar errores de programación, vulnerabilidades de seguridad y problemas de calidad en aplicaciones. Es utilizada en entornos de desarrollo de software crítico como aeroespacial, automotriz y financiero.

Knoppix.

Knoppix es una distribución de Linux que se ejecuta directamente desde un CD, DVD o USB sin necesidad de instalación en el disco duro. Está diseñada principalmente como sistema de rescate y diagnóstico, pero también puede usarse como entorno portátil.

KornShell (ksh).

Kotlin.

Kotlin es un lenguaje de programación moderno desarrollado por JetBrains y diseñado para ejecutarse en la Java Virtual Machine (JVM). Es totalmente interoperable con Java, lo que facilita su adopción en proyectos existentes.

Kubernetes.

Kubernetes es un sistema de orquestación de contenedores de código abierto creado por Google. Permite automatizar la implementación, escalado y gestión de aplicaciones basadas en contenedores como Docker.

Kubernetes Dashboard.

El Kubernetes Dashboard es la interfaz gráfica oficial de Kubernetes. Permite a los usuarios gestionar clústeres, desplegar aplicaciones, monitorear recursos y resolver incidencias desde un entorno visual.

Kubernetes Operator.

Un Operator en Kubernetes es una extensión que permite automatizar la gestión de aplicaciones complejas dentro de un clúster. Está basado en el patrón de controlador, donde un operador gestiona recursos personalizados.

Kuiper.

Kuiper es un motor de procesamiento de flujo ligero y de código abierto diseñado para IoT. Puede ejecutarse en dispositivos de borde (edge computing) con recursos limitados.

KVM (Kernel-based Virtual Machine).

KVM es una tecnología de virtualización incluida en el kernel de Linux que permite ejecutar máquinas virtuales utilizando hardware de virtualización (Intel VT o AMD-V).

LAMP.

LAMP es un acrónimo que representa un conjunto de tecnologías usadas en el desarrollo web: Linux (sistema operativo), Apache (servidor web), MySQL (base de datos) y PHP (lenguaje de programación). Esta pila tecnológica es de código abierto y ampliamente utilizada para aplicaciones web dinámicas.

LAN (Local Area Network).

Una LAN es una red de computadoras que conecta dispositivos dentro de un área geográfica limitada, como una oficina, hogar o campus. Se caracteriza por su alta velocidad y baja latencia en comparación con redes de mayor alcance como WAN.

Landing Page.

Una landing page o página de aterrizaje es una página web diseñada con un propósito específico: captar la atención de los usuarios y convertirlos en clientes o suscriptores. Suele estar optimizada con llamadas a la acción (CTA), formularios y elementos visuales persuasivos.

Laravel.

Laravel es un framework de PHP de código abierto que facilita el desarrollo de aplicaciones web modernas. Está basado en el patrón MVC (Modelo-Vista-Controlador) y ofrece herramientas como migraciones de base de datos, sistema de plantillas (Blade) y gestión de dependencias con Composer.

Latencia en Redes.

La latencia es el tiempo que tarda un paquete de datos en viajar desde su origen hasta su destino a través de una red.

LDAP (Lightweight Directory Access Protocol).

LDAP es un protocolo abierto y estándar utilizado para acceder y gestionar servicios de directorio distribuidos. Permite autenticar y organizar información como usuarios, grupos, dispositivos y recursos en redes empresariales.

Lenguaje de Alto Nivel.

Los lenguajes de alto nivel son aquellos diseñados para ser fácilmente comprensibles por los humanos, utilizando palabras clave y estructuras cercanas al lenguaje natural.

Lenguaje de Bajo Nivel.

Lenguaje de Marcado.

Lenguaje de Programación.

Lenguaje de Programación Visual.

Un lenguaje de programación visual es aquel que permite construir programas a través de diagramas o bloques gráficos en lugar de código textual.

Lenguaje Natural en IA.

El procesamiento de lenguaje natural (PLN) es un área de la inteligencia artificial que busca que los sistemas comprendan, interpreten y generen lenguaje humano.

Lenguajes de Consultas.

Los lenguajes de consultas permiten interactuar con bases de datos para obtener, insertar, modificar o eliminar información. El más usado es SQL, que trabaja con bases de datos relacionales como MySQL, PostgreSQL y SQL Server.

Librería en Programación.

LibreELEC.

LibreOffice.

LibreOffice es una suite ofimática gratuita y de código abierto que incluye procesador de textos, hojas de cálculo, presentaciones, bases de datos y más.

Licencia de Software Libre.

Licencias de Software.

Lifelong Learning en TIC.

LimeSurvey.

Linux.

Linux Mint.

Logging.

Loop en Programación.

Low Code.

LTE (Long Term Evolution).

Lua.

Macro en Software.

MacOS.

MacOS es el sistema operativo desarrollado por Apple para sus computadoras Mac. Se caracteriza por una interfaz gráfica amigable, un alto nivel de seguridad y un ecosistema optimizado con hardware propio.

Magento.

Magento es una plataforma de comercio electrónico de código abierto muy utilizada a nivel mundial. Permite a las empresas crear tiendas online con gestión de inventarios, pagos, envíos y marketing digital.

Mainframe.

Un mainframe es un tipo de computadora de gran tamaño y potencia diseñada para manejar grandes volúmenes de datos y procesos críticos de negocio.

Malware.

El término malware hace referencia a cualquier tipo de software malicioso diseñado para dañar sistemas, robar información o interrumpir operaciones. Existen distintos tipos como virus, troyanos, gusanos, spyware y ransomware.

Manejo de Excepciones.

El manejo de excepciones es una técnica de programación que permite controlar errores en tiempo de ejecución. En lugar de que el programa se detenga abruptamente, se definen bloques de código que gestionan estas situaciones (try, catch, finally en Java, Python, C#, etc.).

Mantenimiento Correctivo.

El mantenimiento correctivo es la acción reactiva de reparar o restaurar un equipo o sistema después de que se ha producido una falla o avería. Incluye diagnóstico, reparación de la causa raíz y verificación de funcionalidad para devolver el activo a condiciones operativas.

Mantenimiento Predictivo.

El mantenimiento predictivo utiliza monitorización en tiempo real, sensores y análisis de datos para anticipar fallos antes de que ocurran, basándose en el estado real del equipo en lugar de calendarios rígidos.

Mantenimiento Preventivo.

El mantenimiento preventivo consiste en intervenciones programadas (inspecciones, lubricación, reemplazo de piezas) realizadas en intervalos fijos de tiempo o uso para prevenir fallos futuros.

MapReduce.

MapReduce es un modelo de programación y procesamiento distribuido diseñado para procesar grandes volúmenes de datos en clústeres.

Mapa de Sitio Web.

Un mapa de sitio web es una representación estructural de todas las páginas y secciones de un sitio. Puede ser visual (diagramas jerárquicos) o en formato XML para ayudar a los motores de búsqueda a indexar mejor el contenido.

Marca de Agua Digital.

La marca de agua digital es una técnica para incrustar información (visible o imperceptible) dentro de contenidos multimedia (imágenes, audio, video, documentos) con fines de autenticidad, trazabilidad o protección de derechos.

Marcado (Markup).

El marcado es un sistema de anotaciones que define la estructura y presentación de documentos digitales. Su ejemplo más conocido es HTML, que usa etiquetas para organizar texto, imágenes y otros elementos.

Marketing Automation.

El marketing automation es el uso de software para automatizar tareas repetitivas como envíos de correos, segmentación de clientes y gestión de leads.

Marketing Digital.

El marketing digital abarca estrategias para promocionar productos o servicios en canales online como redes sociales, buscadores, correos y sitios web.

Marketing en Redes Sociales.

Es una rama del marketing digital que usa plataformas como Facebook, Instagram o LinkedIn para conectar con audiencias.

Markup Validation Service.

Servicio del W3C que, valida la estructura de documentos HTML, XHTML y CSS para garantizar cumplimiento de estándares web.

Marketplace Online.

Un marketplace online es una plataforma digital que conecta vendedores y compradores en un mismo entorno.

Matplotlib.

Matplotlib es una librería de Python para crear gráficos y visualizaciones 2D en ciencia de datos. Permite generar diagramas de barras, líneas, histogramas y dispersión, con gran personalización.

Maven.

Apache Maven es una herramienta de gestión de proyectos Java que automatiza procesos como compilación, pruebas y despliegue. Usa un archivo POM para definir dependencias, configuraciones y plugins.

Memcached.

Memcached es un sistema de almacenamiento en caché distribuido en memoria que mejora la velocidad de aplicaciones web. Guarda temporalmente resultados de consultas y datos frecuentes, reduciendo la carga en bases de datos.

Memoria Caché.

La memoria caché es un tipo de almacenamiento rápido que guarda datos temporales de acceso frecuente. Puede encontrarse en CPUs (niveles L1, L2 y L3), discos o aplicaciones web.

Memoria Flash.

La memoria flash es un tipo de memoria no volátil usada en dispositivos como USB, SSD y tarjetas SD. Conserva los datos sin necesidad de energía y permite lecturas rápidas con bajo consumo.

Memoria RAM.

La memoria RAM (Random Access Memory) es una memoria volátil que almacena datos temporales mientras el sistema está encendido. Permite un acceso rápido a la información necesaria para ejecutar programas.

Memoria ROM.

La memoria ROM (Read Only Memory) es un tipo de memoria no volátil que almacena instrucciones permanentes esenciales para el arranque del sistema.

Memoria Secundaria.

La memoria secundaria es el almacenamiento no volátil destinado a guardar grandes volúmenes de información, como discos duros, SSD o DVDs.

Memoria USB.

La memoria USB es un dispositivo portátil de almacenamiento basado en memoria flash. Permite transferir, almacenar y respaldar datos de manera rápida y sencilla.

Memoria Virtual.

La memoria virtual es una técnica que extiende la capacidad de la RAM usando parte del disco duro como apoyo. Permite que los sistemas ejecuten más aplicaciones de las que físicamente cabrían en la RAM.

Mensajería Instantánea.

La mensajería instantánea es un servicio de comunicación digital en tiempo real que permite intercambiar mensajes, archivos, voz y video.

Meta Tags.

Las meta tags son etiquetas HTML que aportan información adicional sobre un sitio web a los navegadores y motores de búsqueda. Incluyen descripciones, palabras clave, autor y configuraciones de idioma.

Metadatos.

Los metadatos son datos que describen otros datos, proporcionando contexto como autor, fecha de creación, formato o ubicación. Se utilizan en bases de datos, archivos multimedia y sistemas de gestión documental.

Metadatos en Imágenes.

Son datos ocultos en archivos gráficos que incluyen información como cámara usada, fecha de captura, resolución o coordenadas GPS. Usan estándares como EXIF y XMP.

Metaverso.

El metaverso es un entorno digital inmersivo que combina realidad virtual, aumentada y espacios sociales en línea. Permite la interacción mediante avatares en entornos tridimensionales persistentes.

Método Ágil.

El método ágil es un conjunto de principios de desarrollo de software basado en iteraciones cortas, flexibilidad y colaboración con el cliente. Se centra en entregar valor rápidamente, adaptarse a cambios y fomentar la comunicación constante.

Middleware.

El middleware es un software intermediario que conecta aplicaciones, sistemas y servicios. Facilita la comunicación, interoperabilidad y gestión de datos entre diferentes entornos.

Migración de Datos.

La migración de datos es el proceso de transferir información entre sistemas, formatos o plataformas. Involucra extracción, transformación y carga (ETL).

Minado de Criptomonedas.

El minado de criptomonedas valida transacciones y asegura redes blockchain mediante cálculos matemáticos complejos. Requiere hardware especializado como ASICs o GPUs.

Minería de Datos.

La minería de datos consiste en extraer patrones, tendencias y conocimiento útil de grandes volúmenes de información. Usa técnicas estadísticas, algoritmos de machine learning y bases de datos.

Mini PC.

Una mini PC es una computadora compacta que ofrece funcionalidades similares a una de escritorio en menor tamaño. Se usa en oficinas, entretenimiento multimedia y entornos educativos.

Miniaturización en TIC.

La miniaturización en TIC es el proceso de reducir el tamaño de componentes electrónicos sin perder potencia. Ha permitido dispositivos más ligeros, portátiles y con mayor capacidad.

Mirror Site.

Un mirror site es una copia exacta de un sitio web alojada en otro servidor. Se utiliza para mejorar la disponibilidad, reducir la carga del servidor original y garantizar accesibilidad en distintas regiones.

MIPS (Million Instructions Per Second).

MIPS es una unidad de medida del rendimiento de un procesador que indica cuántas instrucciones puede ejecutar por segundo.

MITRE ATT&CK.

MITRE ATT&CK es un marco de conocimiento que documenta tácticas, técnicas y procedimientos usados por atacantes en ciberseguridad.

Modelo de Datos.

Es una representación abstracta que describe cómo se organizan, almacenan y relacionan los datos. Permite estructurar información en bases de datos de manera lógica y coherente.

Modelo de Datos Relacional.

Se basa en tablas (relaciones) que almacenan datos en filas y columnas. Fue propuesto por Edgar F. Codd en los años 70.

Modelo de Red OSI.

Es un marco de referencia de 7 capas que describe cómo se transmiten los datos en una red. Incluye capas como física, enlace, red, transporte, sesión, presentación y aplicación.

Modelo Predictivo.

Es un enfoque analítico que usa algoritmos estadísticos y de machine learning para anticipar resultados futuros. Se entrena con datos históricos y busca patrones que permitan tomar decisiones informadas.

Módem 5G.

Es un dispositivo que conecta equipos a internet mediante la red móvil de quinta generación. Ofrece mayor velocidad, menor latencia y soporte para múltiples dispositivos en IoT.

Modulación Digital.

Es la técnica de transformar datos binarios en señales adecuadas para su transmisión. Incluye métodos como ASK, PSK y QAM.

MongoDB.

Es una base de datos NoSQL orientada a documentos que almacena información en formato BSON (similar a JSON). Destaca por su escalabilidad, flexibilidad y alto rendimiento. Es ampliamente utilizada en aplicaciones web modernas y big data.

Monitor.

Es un dispositivo de salida que muestra información visual de un sistema. Existen monitores LCD, LED y OLED, con distintas resoluciones y tasas de refresco. Son esenciales en entornos de oficina, diseño gráfico, programación y videojuegos.

Monitor de Red.

Es una herramienta que analiza el tráfico y rendimiento de una red. Permite detectar fallos, cuellos de botella y amenazas de seguridad. Es clave en la administración de infraestructuras de TI y ciberseguridad.

Monitorización de Servidores.

Proceso de supervisar en tiempo real el estado y recursos de los servidores. Incluye control de CPU, memoria, disco, servicios y seguridad.

Monitoreo en la Nube.

Es el proceso de supervisar el rendimiento, disponibilidad y seguridad de servicios alojados en entornos cloud. Usa herramientas que permiten rastrear métricas en tiempo real.

Monte Carlo en Computación.

Método de simulación que utiliza números aleatorios para resolver problemas matemáticos o de modelado. Se aplica en finanzas, inteligencia artificial, redes y optimización.

Moodle.

Es un sistema de gestión del aprendizaje (LMS) de código abierto. Facilita la creación de cursos en línea, evaluaciones, foros y seguimiento del progreso de estudiantes.

Moquette.

Es un servidor ligero de mensajería basado en el protocolo MQTT, orientado a aplicaciones IoT. Está desarrollado en Java y destaca por su bajo consumo de recursos.

Motorola 68000.

Es una familia de microprocesadores de 16/32 bits introducida en 1979. Fue ampliamente usada en computadoras como Macintosh, Amiga y consolas de videojuegos. Su arquitectura influyó en posteriores diseños de CPU.

Motores de Búsqueda.

Son plataformas en línea que rastrean, indexan y presentan información de la web según consultas de los usuarios. Ejemplos son Google, Bing y DuckDuckGo. Utilizan algoritmos avanzados y técnicas de SEO para ofrecer resultados relevantes.

Mouse.

Es un dispositivo de entrada que permite interactuar con interfaces gráficas mediante un puntero en pantalla. Existen modelos ópticos, láser, inalámbricos y ergonómicos.

Movimiento Maker.

Es una comunidad global que impulsa la innovación tecnológica bajo la filosofía “hazlo tú mismo” (DIY). Promueve el uso de herramientas como impresoras 3D, placas Arduino, Raspberry Pi y robótica. Fomenta la creatividad, el aprendizaje práctico y la cultura colaborativa.

MP3 (MPEG-1 Audio Layer 3).

Es un formato de compresión de audio digital desarrollado en los años 90. Reduce significativamente el tamaño de los archivos sin perder mucha calidad percibida. Fue clave en la masificación de la música digital y dispositivos portátiles.

MPEG (Moving Picture Experts Group).

Es un grupo de expertos que desarrolla estándares de compresión de audio y video. De ellos surgieron formatos como MPEG-2, MPEG-4 y H.264. Son fundamentales en transmisión digital, DVD, Blu-ray y streaming.

MPLS (Multiprotocol Label Switching).

Es una tecnología de redes que enruta datos usando etiquetas en lugar de direcciones largas. Permite mayor eficiencia, calidad de servicio y seguridad en la transmisión.

MQ (Message Queue).

Es un sistema de comunicación que organiza mensajes en colas para que diferentes aplicaciones o procesos los consuman de forma asincrónica.

MQTT (Message Queuing Telemetry Transport).

Es un protocolo ligero de mensajería usado en IoT. Optimiza el envío de datos entre dispositivos con bajo consumo de energía y ancho de banda.

MS-DOS.

Fue un sistema operativo de línea de comandos lanzado en 1981 por Microsoft. Permitía gestionar archivos y ejecutar programas en computadoras personales. Sentó las bases de los sistemas posteriores como Windows.

MS Project.

Es una herramienta de Microsoft para la gestión de proyectos. Permite planificar tareas, asignar recursos, estimar costos y controlar cronogramas.

Microsoft Office.

Es una suite de productividad que incluye aplicaciones como Word, Excel, PowerPoint y Outlook. Facilita la creación de documentos, hojas de cálculo, presentaciones y gestión de correos.

Microsoft Teams.

Es una plataforma de colaboración desarrollada por Microsoft. Integra chat, videollamadas, almacenamiento de archivos y trabajo en equipo en la nube.

Multicloud.

Es una estrategia que utiliza servicios de múltiples proveedores de nube (como AWS, Azure o Google Cloud). Permite mayor flexibilidad, reducción de riesgos y optimización de costos.

Multitarea.

Es la capacidad de un sistema operativo para ejecutar varios procesos de manera simultánea. Puede ser real (con múltiples núcleos) o simulada mediante la división del tiempo del procesador.

Multiprocesamiento.

Es el uso de dos o más procesadores en un mismo sistema para ejecutar tareas en paralelo. Aumenta el rendimiento, la disponibilidad y la tolerancia a fallos. Es común en servidores y sistemas de alto desempeño.

Multithreading.

Es una técnica de programación que permite ejecutar múltiples hilos dentro de un proceso. Optimiza el uso del CPU y mejora la velocidad en aplicaciones concurrentes.

Muestreo Digital.

Es el proceso de convertir una señal analógica en datos digitales mediante la toma de muestras periódicas. Su calidad depende de la frecuencia de muestreo y la resolución. Es esencial en audio, video y telecomunicaciones.

MuleSoft.

Es una plataforma de integración de aplicaciones y datos adquirida por Salesforce. Facilita la conexión entre sistemas locales y servicios en la nube mediante API y conectores. Se utiliza ampliamente en empresas que requieren interoperabilidad.

Multiplexación por División de Frecuencia (FDM).

Es una técnica que transmite múltiples señales en un mismo canal dividiendo el espectro en diferentes frecuencias. Se usa en radio, televisión por cable y telecomunicaciones.

Multiplexación por División de Tiempo (TDM).

Es una técnica que divide un canal en intervalos de tiempo asignados a distintas señales. Es ampliamente usada en redes digitales y telecomunicaciones. Asegura que múltiples usuarios compartan el mismo medio sin interferencias.

MVP (Producto Mínimo Viable).

Es una versión inicial de un producto con las características esenciales para validar su viabilidad en el mercado. Permite obtener retroalimentación rápida de los usuarios. Es un enfoque clave en metodologías ágiles y startups.

MySQL.

Es un sistema de gestión de bases de datos relacional de código abierto. Utiliza SQL para consultar y manipular datos. Es muy usado en aplicaciones web, siendo la base de plataformas como WordPress y parte del stack LAMP.

Mozilla Firefox.

Es un navegador web de código abierto desarrollado por Mozilla Foundation. Destaca por su seguridad, personalización y soporte a estándares web. Ha sido un competidor relevante frente a navegadores como Chrome y Edge.

N-API.

Es una interfaz estable de programación para Node.js que permite a los desarrolladores crear módulos nativos en C o C++. La gran ventaja de N-API es que abstrae los cambios internos del motor de JavaScript (V8, Chakra, etc.), garantizando compatibilidad entre versiones.

NaaS (Network as a Service).

Es un modelo de servicio en la nube que permite consumir recursos de red bajo demanda, como firewalls, balanceadores de carga, VPN o SD-WAN.

Nagios.

Es una herramienta de código abierto para la supervisión de sistemas, aplicaciones, servicios y redes. Su función principal es detectar problemas antes de que impacten al usuario final, mediante alertas y notificaciones en tiempo real.

NAND Flash.

Es un tipo de memoria no volátil muy utilizada en unidades SSD, memorias USB, tarjetas SD y dispositivos móviles. Al no necesitar energía para retener datos, resulta ideal para almacenamiento masivo portátil.

Nanotecnología en TIC.

Es la aplicación de la nanotecnología en el desarrollo de dispositivos electrónicos y sistemas de información. Permite fabricar transistores más pequeños, procesadores más rápidos y materiales con propiedades avanzadas.

NAS (Network Attached Storage).

Es un sistema de almacenamiento conectado a la red que permite a múltiples usuarios y dispositivos acceder a los mismos archivos de manera centralizada.

NAT (Network Address Translation).

Es una técnica de redes que traduce direcciones IP privadas en públicas, permitiendo que múltiples dispositivos internos compartan una misma conexión a Internet.

Navegación Segura.

Incluye un conjunto de medidas tecnológicas y hábitos que protegen al usuario al interactuar con Internet. Se basa en protocolos como HTTPS, herramientas anti-malware, firewalls y prácticas como evitar enlaces sospechosos.

Navegación Web Anónima.

Es el uso de técnicas y herramientas que permiten acceder a la web sin exponer la identidad digital del usuario. Se logra mediante redes como Tor, proxys o VPN que ocultan la dirección IP y el rastro de navegación.

Navegador Web.

Es un software que permite acceder, visualizar e interactuar con páginas y aplicaciones en Internet mediante protocolos como HTTP y HTTPS. Ejemplos destacados incluyen Google Chrome, Mozilla Firefox, Safari y Microsoft Edge.

Namespace.

Es un contenedor lógico que organiza e identifica de forma única elementos en programación o sistemas. Se utiliza en lenguajes como C++, Java o Python para evitar conflictos de nombres entre variables, funciones o clases.

NAT64.

Es un mecanismo de traducción que permite la comunicación entre redes IPv6 e IPv4. Es fundamental en la transición hacia IPv6, ya que muchas aplicaciones y servidores todavía dependen de IPv4.

NCA.

Es la automatización de la configuración de redes mediante herramientas como Ansible, Puppet o Terraform. Permite gestionar dispositivos de red de forma centralizada, reduciendo errores humanos y aumentando la eficiencia.

NFC (Near Field Communication).

Es una tecnología de comunicación inalámbrica de corto alcance (hasta 10 cm) que permite el intercambio de datos entre dispositivos. Se usa en pagos móviles, transporte público y autenticación de acceso.

Nebula Graph.

Es una base de datos orientada a grafos de código abierto, diseñada para manejar relaciones complejas en grandes volúmenes de datos. Se utiliza en sistemas de recomendación, detección de fraudes y análisis de redes sociales.

Neo4j.

Es una de las bases de datos de grafos más populares a nivel mundial. Está optimizada para consultas de relaciones complejas mediante su lenguaje Cypher. Se utiliza en aplicaciones de inteligencia artificial, gestión de redes, motores de recomendación y análisis de fraude.

Net Neutrality (Neutralidad de la Red).

Es el principio que garantiza que todos los datos en Internet sean tratados por igual, sin discriminación ni priorización de tráfico por parte de los proveedores de servicios.

NetApp.

Es una empresa especializada en soluciones de almacenamiento en red y gestión de datos. Sus productos abarcan sistemas NAS, SAN y herramientas de backup en la nube.

NetBeans.

Es un entorno de desarrollo integrado (IDE) de código abierto, principalmente orientado a Java, aunque soporta otros lenguajes como PHP, HTML y C++. Incluye herramientas para edición, depuración, pruebas y despliegue de aplicaciones.

NetFlow.

Es un protocolo desarrollado por Cisco para recopilar información de tráfico en redes IP. Permite a los administradores analizar patrones de comunicación, identificar cuellos de botella y detectar anomalías de seguridad.

Netlify.

Es una plataforma moderna para desplegar y alojar aplicaciones web, especialmente optimizada para sitios estáticos y proyectos JAMstack. Ofrece integración con Git, CI/CD automatizado y funciones serverless.

Netmask.

Es una máscara de red que se utiliza para dividir una dirección IP en dos partes: la porción de red y la porción de host. Ayuda a determinar a qué subred pertenece un dispositivo y cuáles son los rangos válidos de direcciones dentro de esa red.

Neuralink.

Es un proyecto impulsado por Elon Musk que busca desarrollar interfaces cerebro-computadora. Su objetivo es permitir la comunicación directa entre el cerebro humano y dispositivos electrónicos.

Neurona Artificial.

Es la unidad básica de una red neuronal artificial, inspirada en las neuronas biológicas. Recibe entradas, aplica una función de activación y produce una salida que influye en las siguientes capas.

Neurocomputación.

Es un campo de la informática que estudia el diseño de sistemas inspirados en el cerebro humano para procesar información de manera paralela y adaptativa. Integra conocimientos de neurociencia, inteligencia artificial y hardware especializado.

Networking.

Se refiere al conjunto de técnicas y actividades para establecer, mantener y gestionar conexiones entre dispositivos, sistemas y personas.

NFS (Network File System).

Es un protocolo desarrollado por Sun Microsystems que permite a los sistemas compartir archivos a través de una red. Facilita que los usuarios accedan a archivos remotos como si fueran locales.

NGINX.

Es un servidor web de alto rendimiento que también funciona como proxy inverso, balanceador de carga y caché de contenido. Destaca por su capacidad de manejar miles de conexiones simultáneas con bajo consumo de recursos.

NIC (Network Interface Card).

Es una tarjeta de red, integrada o independiente, que permite a un dispositivo conectarse a una red cableada o inalámbrica. Contiene una dirección MAC única para la identificación en la red.

NIST Cybersecurity Framework.

Es un conjunto de directrices desarrollado por el Instituto Nacional de Estándares y Tecnología (NIST) de Estados Unidos. Proporciona un marco estructurado para gestionar riesgos de ciberseguridad.

Nivel de Abstracción en Software.

Es el grado en el que se ocultan los detalles técnicos de implementación para facilitar la programación y comprensión de un sistema.

Nivel de Servicio (SLA).

Un Service Level Agreement (SLA) es un contrato entre un proveedor de servicios y un cliente que define los niveles mínimos de calidad esperados. Incluye métricas como disponibilidad, tiempo de respuesta y resolución de incidentes.

Nivel de Usuario en Sistemas Operativos.

Es el espacio de ejecución en un sistema operativo donde se desarrollan los programas y aplicaciones sin acceso directo al hardware.

NLP (Natural Language Processing).

Es una rama de la inteligencia artificial que permite a las computadoras comprender, interpretar y generar lenguaje humano. Se aplica en chatbots, traducción automática, análisis de sentimientos y asistentes virtuales.

NoSQL.

Es un modelo de bases de datos que no sigue el esquema relacional tradicional. Está diseñado para manejar datos no estructurados y distribuidos a gran escala.

Node.js.

Es un entorno de ejecución de JavaScript del lado del servidor, basado en el motor V8 de Google. Permite construir aplicaciones escalables y rápidas gracias a su modelo de I/O no bloqueante y orientado a eventos.

Noise en Redes.

Se refiere a las interferencias electromagnéticas o señales no deseadas que afectan la transmisión de datos en una red. Puede provocar pérdida de paquetes, baja velocidad y errores en la comunicación.

Normalización en Bases de Datos.

Es un proceso de organización estructurada de datos en una base de datos relacional. Su objetivo es reducir la redundancia y mejorar la integridad mediante la aplicación de formas normales (1FN, 2FN, 3FN, etc.). Una base de datos bien normalizada optimiza el almacenamiento y evita inconsistencias en la información.

Notebook Jupyter.

Es una aplicación de código abierto ampliamente usada en ciencia de datos, aprendizaje automático y análisis estadístico. Permite crear y compartir documentos que combinan código ejecutable, texto explicativo, fórmulas y visualizaciones.

Núcleo de Procesador.

Es la unidad de procesamiento independiente dentro de un chip que ejecuta instrucciones de manera autónoma. Los procesadores modernos incluyen múltiples núcleos, lo que permite realizar tareas en paralelo y mejorar el rendimiento.

Núcleo Microkernel.

Es una arquitectura de sistema operativo donde el kernel incluye solo las funciones esenciales, como comunicación entre procesos y gestión de memoria básica. Otros servicios se ejecutan en espacio de usuario.

Núcleo Monolítico.

Es una arquitectura de sistema operativo en la que todos los servicios del sistema (gestión de memoria, procesos, controladores, etc.) se ejecutan en el mismo espacio del kernel.

Número de Serie en Hardware.

Es un identificador único asignado a dispositivos electrónicos como computadoras, discos duros o teléfonos. Se utiliza para control de inventario, soporte técnico y verificación de autenticidad.

NumPy.

NVD (National Vulnerability Database).

NVMe (Non-Volatile Memory Express).

NVR (Network Video Recorder).

Es un dispositivo o software que graba video digital proveniente de cámaras IP en una red. A diferencia de los DVR, que trabajan con cámaras analógicas, los NVR manejan flujos digitales, lo que permite mejor calidad de imagen y funciones avanzadas como análisis inteligente de video.

OAuth.

Es un protocolo de autorización que permite que aplicaciones externas accedan a recursos de un usuario en otra plataforma sin necesidad de compartir sus credenciales (usuario y contraseña).

OAuth 2.0.

Es la segunda versión del protocolo OAuth, diseñado para mayor seguridad y flexibilidad en la autorización. Introduce roles como "cliente", "servidor de autorización" y "servidor de recursos", además de diferentes "flujos" de autenticación (authorization code, implicit, client credentials).

Objetivo SMART en Proyectos.

Objetos 3D en Computación Gráfica.

Son representaciones digitales tridimensionales de figuras o estructuras que se utilizan en simulación, videojuegos, cine, ingeniería y realidad aumentada.

Objetos Inmutables.

Son estructuras de datos que, una vez creadas, no pueden ser modificadas. En lugar de alterar un objeto, cualquier cambio genera una nueva instancia. Este concepto es fundamental en lenguajes funcionales y en programación concurrente porque evita errores relacionados con estados compartidos.

Objetos Inteligentes (Smart Objects).

Son dispositivos físicos conectados a internet (IoT) que interactúan entre sí o con usuarios. Incorporan sensores, software y conectividad para recopilar y transmitir datos. Ejemplos incluyen relojes inteligentes, termostatos conectados y electrodomésticos domóticos.

OBDC (Open Database Connectivity).

Es un estándar de acceso a bases de datos que permite que aplicaciones se conecten a distintos sistemas de gestión (MySQL, Oracle, SQL Server) usando una misma interfaz.

Observabilidad en Sistemas.

Es la capacidad de un sistema para ser monitoreado y comprendido a través de sus datos de salida (logs, métricas y trazas). Va más allá del monitoreo tradicional al permitir entender el estado interno del sistema y diagnosticar problemas complejos.

OCR (Optical Character Recognition).

Es una tecnología que convierte imágenes de texto (escaneos, fotos o documentos) en texto editable digitalmente. Se aplica en la digitalización de libros, automatización de facturas, reconocimiento de matrículas y accesibilidad.

Offshoring en TI.

Es la práctica de trasladar procesos o servicios tecnológicos a otro país, generalmente con el objetivo de reducir costos o acceder a talento especializado.

Oficina Virtual.

Es un entorno digital que permite a empleados y colaboradores trabajar de manera remota con acceso a herramientas de comunicación, colaboración y gestión de proyectos.

Ofimática.

Se refiere al conjunto de programas y herramientas diseñadas para automatizar y facilitar las tareas de oficina. Incluye procesadores de texto, h ojas de cálculo, presentaciones, gestores de correo y bases de datos.

OIDC (OpenID Connect).

Es un protocolo de autenticación construido sobre OAuth 2.0 que añade una capa de verificación de identidad. Permite que aplicaciones confirmen la identidad de un usuario de forma segura mediante tokens de acceso e ID.

OLAP (Online Analytical Processing).

Es una tecnología que permite analizar grandes volúmenes de datos de manera multidimensional. Facilita la exploración interactiva de información mediante operaciones como drill-down, roll-up y slice-and-dice.

OLTP (Online Transaction Processing).

Es un tipo de sistema diseñado para gestionar transacciones en tiempo real. Se caracteriza por manejar un gran número de operaciones cortas y rápidas como reservas, compras en línea o transferencias bancarias.

OLED.

Es una tecnología de pantalla donde cada píxel emite su propia luz gracias a compuestos orgánicos. Esto permite mayor contraste, colores más vivos y pantallas más delgadas que las LCD.

Omnicanalidad en TIC.

Es una estrategia de comunicación que integra múltiples canales (presenciales, telefónicos, web, redes sociales, aplicaciones móviles) en una experiencia unificada para el usuario.

On-Premise.

Hace referencia a soluciones tecnológicas implementadas y mantenidas dentro de las instalaciones físicas de la organización. A diferencia de la nube, en On-Premise la empresa tiene control total sobre la infraestructura y los datos, aunque con mayores costos de mantenimiento y menor flexibilidad.

Online Banking.

Es el servicio que ofrecen las instituciones financieras para realizar operaciones bancarias a través de internet. Permite a los usuarios consultar saldos, transferir dinero, pagar facturas y gestionar cuentas sin necesidad de acudir físicamente al banco.

Open Access.

Es un movimiento que promueve el acceso libre y gratuito a publicaciones académicas, artículos científicos y materiales educativos en línea. Busca eliminar barreras económicas y de copyright para fomentar la difusión del conocimiento.

OpenAPI.

Es una especificación estándar para definir, describir y documentar APIs REST de manera estructurada y legible tanto para humanos como para máquinas.

OpenBSD.

Open Hardware.

OpenID.

OpenShift.

OpenStack.

Es una plataforma de software libre para la creación y gestión de infraestructuras de nube públicas y privadas. Ofrece módulos para almacenamiento, redes, virtualización y cómputo.

OpenVPN.

Es una solución de red privada virtual (VPN) de código abierto que permite conexiones seguras y cifradas a través de internet. Soporta diversos protocolos de seguridad y autenticación, lo que lo hace flexible para distintos escenarios.

Operadores Lógicos en Programación.

Son símbolos o palabras reservadas que permiten construir condiciones en expresiones booleanas, como AND, OR y NOT. Se utilizan ampliamente en estructuras de control como bucles y condicionales.

Operador Ternario en Programación.

Es un operador condicional que evalúa una expresión booleana y devuelve un valor u otro según el resultado. Su sintaxis básica es condición ? valor_si_verdadero : valor_si_falso.

Optimización de Motores de Búsqueda (SEO).

Es el conjunto de técnicas aplicadas a sitios web para mejorar su visibilidad en buscadores como Google. Incluye estrategias de palabras clave, optimización de velocidad, arquitectura web y generación de contenido relevante.

Orquestación en TI.

Consiste en la automatización y coordinación de procesos tecnológicos, integrando múltiples sistemas y aplicaciones. Se aplica en entornos de nube, DevOps y microservicios para optimizar recursos y reducir errores manuales.

OSI (Modelo de Referencia OSI).

Es un modelo conceptual que describe cómo los sistemas de red se comunican en siete capas: Física, Enlace de Datos, Red, Transporte, Sesión, Presentación y Aplicación.

OSINT (Open Source Intelligence).

Es la práctica de recopilar información pública disponible en internet, medios digitales, redes sociales y documentos oficiales. Se utiliza en seguridad informática, inteligencia militar y ciberinvestigaciones.

OSS (Operational Support Systems).

Son sistemas que ayudan a los operadores de telecomunicaciones en la gestión de servicios, redes y clientes. Incluyen funciones como facturación, aprovisionamiento, gestión de fallos y monitoreo de red.

OTP (One Time Password).

Es una contraseña de un solo uso generada dinámicamente para procesos de autenticación. Su vigencia limitada la hace más segura que contraseñas estáticas, reduciendo riesgos de robo de credenciales.

Outsourcing en TI.

Es la práctica de externalizar servicios tecnológicos a proveedores especializados. Incluye soporte técnico, desarrollo de software, mantenimiento de redes y ciberseguridad.

Outsourcing Tecnológico.

Es una variante del outsourcing enfocada específicamente en infraestructura, desarrollo de software y servicios de TI. Se diferencia por estar orientado a la innovación y a la gestión de tecnología avanzada.

Overclocking.

Es la práctica de aumentar la velocidad de reloj de un procesador, tarjeta gráfica o memoria RAM más allá de las especificaciones del fabricante. Su objetivo es mejorar el rendimiento en aplicaciones exigentes como videojuegos o software de diseño.

OWASP (Open Web Application Security Project).

Es una organización sin fines de lucro dedicada a mejorar la seguridad del software. Proporciona recursos como guías, metodologías y herramientas para proteger aplicaciones web.

PaaS (Platform as a Service).

Es un modelo de servicios en la nube que proporciona plataformas listas para desarrollar, probar y desplegar aplicaciones sin necesidad de gestionar la infraestructura subyacente.

Packet Sniffer.

s una herramienta de análisis de red que captura y examina paquetes de datos que circulan por una red. Se utiliza en administración de sistemas, resolución de problemas y pruebas de seguridad, pero también puede emplearse con fines maliciosos (intercepción de información sensible).

Pair Programming.

Es una práctica ágil de desarrollo de software donde dos programadores trabajan juntos en el mismo código: uno actúa como “conductor” (escribe el código) y el otro como “navegador” (revisa y analiza en tiempo real).

Palo Alto Networks.

Es una empresa líder en ciberseguridad que ofrece soluciones avanzadas como firewalls de nueva generación, sistemas de prevención de intrusos, protección de endpoints y seguridad en la nube.

Panel de Control.

Es una interfaz gráfica que centraliza configuraciones del sistema operativo y aplicaciones. En Windows, permite administrar hardware, redes, usuarios, seguridad y programas.

Paquete de Datos.

Es la unidad básica de información transmitida a través de una red. Un paquete suele incluir una cabecera (con datos de control como direcciones IP de origen y destino), la carga útil (información real transmitida) y un pie de control.

Parámetros en Programación.

Son valores que se pasan a funciones o métodos para personalizar su ejecución. Existen parámetros por valor, por referencia, opcionales y con valores predeterminados, según el lenguaje de programación.

Paralelismo en Computación.

Es la capacidad de ejecutar múltiples tareas de manera simultánea dividiendo el trabajo entre varios procesadores o núcleos. Se utiliza en supercomputación, big data, inteligencia artificial y videojuegos.

Parseo (Parsing).

Es el proceso de analizar una cadena de texto o datos siguiendo reglas gramaticales definidas (normalmente mediante gramáticas formales). Es fundamental en compiladores, intérpretes, validadores de XML/JSON y procesamiento de datos.

Pascal.

Es un lenguaje de programación estructurado creado por Niklaus Wirth en 1970. Fue ampliamente utilizado en la enseñanza de programación por su claridad y enfoque en la programación estructurada. Inspiró a otros lenguajes como Modula-2 y Delphi.

Password Manager.

Es una aplicación diseñada para almacenar, generar y administrar contraseñas de manera segura. Emplea cifrado fuerte para proteger la información y suele integrar autenticación multifactor.

Patch en Software.

Es una actualización que corrige errores, vulnerabilidades de seguridad o añade mejoras a un programa existente. Los parches pueden ser menores (hotfixes) o acumulativos, dependiendo de la magnitud de los cambios.

Path.

Hace referencia a la dirección jerárquica que define la ubicación exacta de un archivo o directorio dentro de un sistema de archivos. Puede ser ruta absoluta (desde la raíz) o ruta relativa (en relación con la carpeta actual).

Patrón de Diseño.

Son soluciones reutilizables y probadas para problemas recurrentes en el diseño de software. Existen categorías como creacionales (Singleton, Factory), estructurales (Adapter, Composite) y de comportamiento (Observer, Strategy).

PayPal.

Es una plataforma de pagos en línea fundada en 1998 que permite realizar transacciones electrónicas de manera segura sin compartir datos financieros con los vendedores.

PCI DSS (Payment Card Industry Data Security Standard).

Es un estándar de seguridad creado por las principales marcas de tarjetas de crédito (Visa, MasterCard, American Express, etc.) para proteger la información de los titulares de tarjetas.

PCI Express.

Es un estándar de bus de expansión de alta velocidad utilizado en placas base para conectar tarjetas gráficas, discos SSD NVMe, tarjetas de red y otros periféricos.

Peer-to-Peer (P2P).

Es una arquitectura de red en la que los nodos actúan simultáneamente como clientes y servidores, compartiendo recursos directamente sin necesidad de un servidor central.

Penetration Testing (Pentesting).

Es la práctica de realizar pruebas controladas de seguridad para identificar vulnerabilidades en sistemas, redes o aplicaciones. Los pentesters simulan ataques reales para evaluar la resistencia de una infraestructura.

Pentium.

Es una familia de microprocesadores desarrollada por Intel, lanzada en 1993 como sucesora de la serie 486. Fue ampliamente utilizada en computadoras personales durante la década de 1990 y principios de 2000. Perceptrón.

Es el modelo más básico de red neuronal artificial, creado por Frank Rosenblatt en 1958. Consiste en un nodo que recibe múltiples entradas ponderadas, aplica una función de activación y genera una salida binaria.

Performance Testing (Pruebas de Rendimiento).

Son pruebas de software enfocadas en evaluar la capacidad de una aplicación bajo diferentes condiciones de carga, estrés y volumen de usuarios. Permiten medir tiempos de respuesta, uso de recursos y estabilidad del sistema.

Periféricos.

Son dispositivos externos conectados a una computadora para ampliar sus funciones. Se dividen en entrada (teclado, ratón, escáner), salida (monitor, impresora, altavoces) y almacenamiento (discos externos, memorias USB).

Perl.

Es un lenguaje de programación interpretado, creado en 1987 por Larry Wall, inicialmente enfocado en procesamiento de texto. Combina características de lenguajes como C, sed y awk. Es muy usado en administración de sistemas, desarrollo web y bioinformática.

Persistencia en Datos.

Es la propiedad que asegura que la información guardada permanezca disponible incluso después de que una aplicación finaliza su ejecución. Generalmente se implementa en bases de datos, archivos o sistemas de almacenamiento distribuido.

Persona Digital.

Es la representación virtual de un individuo en entornos digitales, que puede incluir avatares, perfiles en redes sociales o identidades en el metaverso. Estas representaciones permiten interacción en plataformas colaborativas, educativas o de entretenimiento.

Personalización de Software.

Consiste en la adaptación de una aplicación a las necesidades específicas de un usuario o empresa. Puede implicar cambios en la interfaz, funciones, idioma, flujos de trabajo o integración con otros sistemas.

PHP (Hypertext Preprocessor).

Es un lenguaje de programación interpretado, ampliamente usado en el desarrollo web del lado del servidor. Permite generar páginas dinámicas, interactuar con bases de datos y manejar sesiones de usuario.

Píxel.

Es la unidad mínima de una imagen digital, formada por un punto de color que al combinarse con otros conforma una representación visual. Cada píxel contiene información de color expresada generalmente en modelos como RGB o CMYK.

PKI (Public Key Infrastructure).

Es un sistema de gestión de claves y certificados digitales que permite asegurar la comunicación y la identidad en entornos electrónicos. Se basa en criptografía de clave pública y privada para garantizar confidencialidad, integridad y autenticidad.

Plataforma CI/CD.

Es un entorno de herramientas que facilita la Integración Continua (CI) y la Entrega/Despliegue Continuo (CD) en proyectos de software. Automatiza la construcción, pruebas y despliegues, reduciendo errores humanos y acelerando la entrega de valor.

Plataforma de Comercio Electrónico.

Son sistemas digitales que permiten la compra y venta de bienes o servicios en línea. Incluyen catálogos de productos, pasarelas de pago, gestión de inventarios y sistemas de envío.

Plataforma de Comunicación.

Se refiere a aplicaciones y sistemas que permiten el intercambio de mensajes, llamadas o videoconferencias entre usuarios. Pueden ser en tiempo real (WhatsApp, Slack, Microsoft Teams) o asincrónicas (correo electrónico, foros).

Plataforma de Entretenimiento Digital.

Son servicios en línea que ofrecen contenidos multimedia como música, series, películas o videojuegos. Funcionan generalmente bajo suscripción (Netflix, Spotify, PlayStation Network) y utilizan técnicas de personalización para recomendar contenido.

Plataforma Educativa.

Es un entorno digital diseñado para la enseñanza y el aprendizaje, que facilita cursos, clases virtuales y materiales didácticos. Incluyen herramientas de gestión académica, evaluación, foros y seguimiento de estudiantes.

Plataforma de Información.

Es un sistema que organiza, gestiona y distribuye datos a usuarios o empresas. Puede integrar fuentes diversas, ofrecer paneles de control y apoyar en la toma de decisiones.

Plataforma IoT.

Plataforma de Streaming.

Placa Base (Motherboard).

Plug and Play (PnP).

Es un estándar que permite que el sistema operativo detecte y configure automáticamente hardware conectado, como impresoras, cámaras o memorias USB. Su objetivo es facilitar la instalación sin necesidad de configuraciones manuales.

PoE (Power over Ethernet).

Es una tecnología que permite transmitir energía eléctrica junto con datos a través de un mismo cable Ethernet. Esto elimina la necesidad de cables de alimentación adicionales en dispositivos como cámaras IP, teléfonos VoIP y puntos de acceso WiFi.

Podcast.

Es un archivo o programa de audio digital distribuido por internet, generalmente en formato de episodios periódicos. Se pueden descargar o escuchar en streaming mediante plataformas como Spotify, Apple Podcasts o Google Podcasts.

Point of Sale (POS).

Es un sistema informático utilizado en comercios para gestionar ventas, inventarios y pagos. Incluye hardware como terminales, escáneres y cajas registradoras, junto con software de facturación y control de clientes.

Polimorfismo en Programación.

Es un principio de la programación orientada a objetos (POO) que permite a una misma función o método tener múltiples comportamientos según el contexto.

Política de Privacidad.

Es un documento legal que describe cómo una organización recopila, usa, protege y comparte los datos personales de los usuarios. Es obligatorio en muchos países para cumplir con normativas como el GDPR en Europa o la Ley de Habeas Data en América Latina.

Política de Seguridad Informática.

Conjunto de normas, procedimientos y controles establecidos por una organización para proteger sus sistemas, redes y datos. Define aspectos como acceso, copias de seguridad, cifrado, gestión de contraseñas y respuesta ante incidentes.

Pop-up.

Es una ventana emergente que aparece de forma automática en una aplicación o página web. Puede tener fines legítimos, como mostrar notificaciones o formularios, pero también se usa en publicidad intrusiva o ataques de phishing.

Portal Web.

Es un sitio en línea que centraliza el acceso a múltiples servicios, recursos o aplicaciones. Puede incluir noticias, foros, correo electrónico, herramientas de trabajo y bases de datos.

Portabilidad de Software.

Es la capacidad de un programa para ejecutarse en diferentes plataformas o sistemas operativos con mínimas modificaciones. Se logra mediante lenguajes multiplataforma, virtualización o contenedores.

PostgreSQL.

Es un sistema de gestión de bases de datos relacional de código abierto, altamente robusto y extensible. Soporta operaciones ACID, replicación, procedimientos almacenados y tipos de datos avanzados.

Power BI.

Es una herramienta de Microsoft para análisis de datos y generación de reportes interactivos. Permite conectarse a múltiples fuentes de datos, transformarlos y visualizarlos en dashboards dinámicos.

PowerPoint.

Es un programa de presentaciones desarrollado por Microsoft, parte de la suite Office. Permite crear diapositivas con texto, imágenes, animaciones, gráficos y multimedia.

PowerShell.

Es un lenguaje de scripting y línea de comandos desarrollado por Microsoft para automatización y administración de sistemas. Está basado en .NET y permite gestionar tareas tanto en Windows como en entornos multiplataforma (Linux, macOS).

PPP (Point-to-Point Protocol).

Es un protocolo de comunicación que permite la transmisión de datos entre dos nodos directamente conectados. Se utiliza en enlaces de red como conexiones seriales, DSL y VPN.

Procesador Cuántico.

Es un tipo de unidad de procesamiento que utiliza principios de la mecánica cuántica, como la superposición y el entrelazamiento, para realizar cálculos.

Procesador de Texto.

Procesamiento Batch.

Es un método de ejecución de programas en el que se procesan grandes volúmenes de datos en lotes sin interacción del usuario. Se utiliza en entornos empresariales para tareas repetitivas como generación de reportes, facturación o respaldo de datos.

Procesamiento Digital de Señales (DSP).

Consiste en la manipulación matemática de señales digitales, como audio, video o datos de sensores. Se utiliza en aplicaciones como reconocimiento de voz, telecomunicaciones, procesamiento de imágenes y dispositivos médicos.

Procesamiento Distribuido.

Es un modelo en el que múltiples computadoras trabajan de manera colaborativa para ejecutar procesos. Cada nodo contribuye con recursos de cómputo y comunicación, logrando mayor escalabilidad y tolerancia a fallos.

Procesamiento Paralelo.

Es la técnica de ejecutar múltiples instrucciones simultáneamente utilizando varios procesadores o núcleos. Se emplea en supercomputadoras, inteligencia artificial y simulaciones científicas que requieren alto rendimiento.

Proceso de Desarrollo de Software.

Es el conjunto de fases, metodologías y herramientas utilizadas para planificar, construir, probar y mantener aplicaciones. Incluye etapas como análisis de requisitos, diseño, programación, pruebas, despliegue y mantenimiento.

Proyecto de Software.

Es una iniciativa organizada para crear, mejorar o mantener un sistema informático dentro de un tiempo y presupuesto definidos. Requiere gestión de recursos humanos, tecnológicos y financieros.

Proyector.

Proyección Holográfica.

Protocolo POP3.

Es un protocolo de correo electrónico utilizado para descargar mensajes desde un servidor al dispositivo cliente. Generalmente, los mensajes se eliminan del servidor tras la descarga, aunque algunos clientes permiten conservar una copia.

Protocolo RDP.

Es un protocolo desarrollado por Microsoft que permite conectarse y controlar remotamente un equipo a través de una red. Proporciona transmisión de gráficos, entrada de teclado y ratón, así como redirección de recursos locales.

Protocolo SIP.

Es un protocolo de señalización usado en telecomunicaciones para establecer, modificar y finalizar sesiones multimedia como llamadas de voz y video sobre IP.

Protocolo SMTP.

Es el protocolo estándar para el envío de correos electrónicos a través de internet. Funciona en conjunto con POP3 o IMAP para la recepción de mensajes.

Protocolo SNMP.

Es un protocolo utilizado para la supervisión y gestión de dispositivos en redes, como routers, switches, servidores e impresoras. Permite recolectar métricas, enviar alertas y modificar configuraciones de forma remota.

Protocolo SSH.

Es un protocolo de administración remota que cifra las comunicaciones entre cliente y servidor. Reemplaza a Telnet al ofrecer autenticación segura y transferencia de datos protegida.

Protocolo SSL/TLS.

Son protocolos de seguridad que cifran la comunicación en internet, garantizando confidencialidad, integridad y autenticidad de los datos. Se utilizan en sitios web con HTTPS, correo electrónico y aplicaciones críticas.

Protocolo TCP/IP.

Es el conjunto de protocolos fundamentales para la comunicación en internet. TCP gestiona la transmisión fiable de datos, mientras que IP se encarga de direccionar y enrutar los paquetes.

Protocolo Telnet.

Es un protocolo de red que permite controlar dispositivos de forma remota mediante una interfaz de línea de comandos. Fue ampliamente usado en sus inicios, pero carece de cifrado, lo que lo hace inseguro frente a ataques de interceptación.

Prototipo de Alta Fidelidad.

Es una representación avanzada y casi funcional de un sistema o aplicación, usada para pruebas de usabilidad y validación de requisitos. Incluye diseño visual detallado, interactividad y simulación cercana al producto final.

Prototipo de Baja Fidelidad.

Representación preliminar y esquemática de un sistema o aplicación, normalmente en papel o con herramientas simples. Se enfoca en la estructura, navegación y disposición de los elementos sin considerar diseño visual detallado.

Prototipo de Software.

Versión inicial de un sistema o aplicación que simula funcionalidades básicas para evaluar requerimientos. Ayuda a identificar errores, ambigüedades y necesidades del usuario antes de la programación definitiva.

Prototipado Rápido.

Técnica de desarrollo que busca crear prototipos funcionales en poco tiempo, generalmente usando herramientas que permiten ajustes inmediatos. Es clave en innovación, diseño de productos y metodologías ágiles.

Proxy.

Servidor o servicio que actúa como intermediario entre un cliente y otro servidor. Se utiliza para filtrar peticiones, mejorar seguridad, balancear carga o almacenar en caché.

Pseudoclase CSS.

Conjunto de palabras clave en CSS que definen un estado especial de un elemento HTML. Ejemplo: :hover aplica estilos cuando el usuario pasa el cursor sobre un elemento. Permiten enriquecer la interactividad sin necesidad de JavaScript.

Pseudocódigo.

Lenguaje informal que combina estructuras lógicas y descripciones en lenguaje natural para representar algoritmos. No sigue una sintaxis estricta de programación, lo que lo hace accesible para programadores y no programadores.

Pseudoelemento CSS.

Herramienta en CSS que permite aplicar estilos a una parte específica de un elemento, como ::before o ::after. A diferencia de las pseudoclases, crean elementos virtuales dentro del DOM visual.

Puerto de Red.

Identificador numérico asociado a un proceso o servicio de comunicación en una red. Por ejemplo, el puerto 80 corresponde a HTTP y el 443 a HTTPS. Los puertos permiten que múltiples servicios funcionen simultáneamente en una misma máquina.

Python.

Lenguaje de programación interpretado, de alto nivel y multiparadigma, creado por Guido van Rossum en 1991. Es reconocido por su sintaxis clara, legibilidad y amplia comunidad. Se utiliza en desarrollo web, inteligencia artificial, análisis de datos, automatización y más.

Python Flask.

Framework minimalista de Python para desarrollo web. Proporciona lo esencial para crear aplicaciones rápidas y escalables, dejando flexibilidad para añadir extensiones según la necesidad. Es muy usado en APIs REST, prototipos y microservicios.

Python Pandas.

Biblioteca de Python enfocada en análisis y manipulación de datos. Ofrece estructuras como DataFrame y Series para manejar datos tabulares y temporales de manera eficiente.

Python TensorFlow.

Framework de código abierto desarrollado por Google para el desarrollo y entrenamiento de modelos de inteligencia artificial. Está diseñado para machine learning y deep learning, con soporte para redes neuronales complejas.

QBasic.

Lenguaje de programación desarrollado por Microsoft en los años 90 como una evolución de BASIC. Su sencillez lo convierte en una excelente herramienta para aprender lógica y fundamentos de programación. Aunque hoy en día está en desuso, marcó a toda una generación de programadores.

Q-learning.

Es un algoritmo de aprendizaje por refuerzo que permite a un agente aprender decisiones óptimas a través de la interacción con su entorno. Utiliza una tabla de valores para evaluar acciones y estados, mejorando con la retroalimentación.

Qubit.

Es la unidad básica de información en la computación cuántica. A diferencia de un bit clásico (0 o 1), un qubit puede estar en una superposición de ambos estados gracias a las propiedades cuánticas. También puede estar entrelazado con otros qubits, lo que multiplica exponencialmente la capacidad de procesamiento.

Query.

Es una instrucción usada en bases de datos para recuperar, insertar, modificar o eliminar información. Se redacta en lenguajes como SQL, siendo el SELECT el más común para consultar datos. Es un concepto esencial en la gestión y análisis de datos.

Quicksort.

Algoritmo de ordenamiento muy eficiente basado en la estrategia de divide y vencerás. Consiste en elegir un pivote, dividir los elementos en mayores o menores a él y ordenar recursivamente.

RAT (Remote Access Trojan).

Es un tipo de software malicioso que permite a un atacante controlar remotamente un sistema infectado. Suelen instalarse de forma oculta y pueden registrar pulsaciones de teclado, acceder a archivos, encender cámaras o instalar más malware.

Realidad Aumentada (AR).

Es una tecnología que superpone información digital (gráficos, audio, texto) sobre el mundo real en tiempo real. Funciona mediante cámaras, sensores y software de procesamiento.

Realidad Mixta (MR).

Combina elementos de realidad aumentada y realidad virtual para crear entornos donde objetos físicos y virtuales coexisten e interactúan. A diferencia de la AR, en la MR los objetos digitales se anclan en el espacio físico y reaccionan a cambios en el entorno.

Realidad Virtual (VR).

Es una simulación completamente digital e inmersiva que sustituye la percepción del mundo físico por un entorno generado por computadora. Se accede mediante visores de VR y, en ocasiones, guantes hápticos o sensores de movimiento.

Reciclaje de Electrónicos (E-Waste).

Es el proceso de gestión responsable de dispositivos electrónicos al final de su vida útil. Incluye la recuperación de materiales reutilizables (oro, cobre, aluminio) y la eliminación segura de sustancias peligrosas como plomo o mercurio.

Recuperación ante Desastres (Disaster Recovery - DR).

Es el conjunto de políticas y procedimientos que garantizan la restauración de sistemas críticos después de un fallo, ataque cibernético o catástrofe natural.

Red Ad Hoc.

Es una red temporal y descentralizada formada por dispositivos que se conectan directamente entre sí sin necesidad de un router o punto de acceso central.

Red de Área Amplia (WAN).

Es una red de telecomunicaciones que cubre grandes distancias geográficas, conectando múltiples redes locales (LAN). Ejemplos son Internet y redes privadas de empresas multinacionales.

Red de Área Metropolitana (MAN).

Es una red que abarca una ciudad o área urbana completa, conectando varias LAN a lo largo de una región metropolitana. Se utiliza para interconectar universidades, empresas y organismos gubernamentales.

Red de Área Personal (PAN).

Es una red de corto alcance que conecta dispositivos cercanos de una misma persona, como smartphones, tablets, auriculares o relojes inteligentes. Normalmente utiliza tecnologías inalámbricas como Bluetooth, NFC o infrarrojo.

Red de Distribución de Contenido (CDN).

Es una red de servidores distribuidos geográficamente que almacenan y entregan contenido web (imágenes, videos, páginas) de forma más rápida a los usuarios. Reduce la latencia al acercar los datos al lugar donde se realiza la petición.

Red de Sensores Inalámbricos (WSN).

Es un conjunto de sensores interconectados que recopilan datos del entorno (temperatura, humedad, movimiento) y los transmiten de manera inalámbrica.

Red de Telecomunicaciones.

Es la infraestructura que permite la transmisión de información entre usuarios o sistemas a distancia. Incluye cables, antenas, satélites, routers, switches y protocolos de comunicación.

Red Híbrida.

Es una arquitectura de red que combina distintos tipos de tecnologías, como cableado Ethernet, fibra óptica y Wi-Fi. Permite aprovechar las ventajas de cada medio: la estabilidad de la conexión cableada y la flexibilidad de la conexión inalámbrica.

Red Mesh.

Es un tipo de red donde cada nodo está conectado a múltiples nodos, creando una topología de malla. No depende de un único router, lo que mejora la resiliencia y cobertura. Se usa en Wi-Fi doméstico, IoT industrial y entornos militares.

Red Privada Virtual (VPN).

Es una tecnología que crea un túnel cifrado entre un dispositivo y una red, generalmente a través de Internet. Se usa para proteger la privacidad, evitar censura, acceder a recursos corporativos y ocultar la dirección IP real.

Redes Definidas por Software (SDN).

Es un enfoque de gestión de redes que separa el plano de control (decisiones de enrutamiento) del plano de datos (tráfico real). Esto permite programar, automatizar y optimizar redes de manera centralizada.

Redes Inalámbricas (Wi-Fi).

Es una tecnología que permite conectar dispositivos a través de ondas de radio, eliminando la necesidad de cables. Opera en bandas de 2.4 GHz, 5 GHz y recientemente 6 GHz (Wi-Fi 6E).

Redes Neuronales Artificiales.

Son modelos computacionales inspirados en el cerebro humano, formados por capas de nodos (neuronas artificiales) interconectados. Se utilizan en inteligencia artificial para resolver problemas complejos como reconocimiento de voz, visión por computadora y predicciones financieras.

Redes Neuronales Convolucionales (CNN).

Son un tipo de red neuronal artificial especializada en el procesamiento de datos con estructuras en forma de rejilla, como imágenes.

Redes Neuronales Recurrentes (RNN).

Son un tipo de red neuronal diseñada para trabajar con datos secuenciales, como texto, audio o series temporales.

Redis.

Es una base de datos en memoria, de tipo clave-valor, que ofrece acceso ultrarrápido a la información. Soporta estructuras avanzadas como listas, conjuntos, hashes y streams. E

Redundancia en Redes.

Es la implementación de elementos adicionales (enlaces, routers, switches) para garantizar continuidad en caso de fallas. Se usa en infraestructuras críticas donde la disponibilidad es prioritaria.

Redundancia de Datos.

Es la duplicación de información dentro de una base de datos o sistema de almacenamiento. Puede ser intencional, como en copias de seguridad o sistemas distribuidos, o innecesaria, generando inconsistencias.

Refactorización (Refactoring).

Es la práctica de modificar el código para mejorar su estructura, legibilidad y mantenimiento sin alterar su comportamiento externo. Permite reducir la deuda técnica y facilitar futuras ampliaciones.

Regex (Expresiones Regulares).

Son patrones formales que describen cadenas de texto para facilitar búsquedas, validaciones y transformaciones. Se utilizan en validación de correos, extracción de datos en logs y análisis de documentos.

Registro CPU.

Son pequeñas unidades de almacenamiento ultrarrápidas integradas en el procesador. Guardan datos temporales, direcciones de memoria e instrucciones que la CPU necesita de inmediato.

Registro de Actividad (Log).

Es un archivo donde se almacenan eventos generados por sistemas, aplicaciones o dispositivos. Incluye información como errores, accesos, procesos iniciados o finalizados.

Registro DNS.

Es un elemento de la base de datos DNS que asocia un dominio con información como direcciones IP, alias o servidores de correo.

Registro MX (Mail Exchange).

Es un tipo de registro DNS que identifica qué servidores de correo electrónico están autorizados a recibir mensajes en nombre de un dominio. Cada registro MX tiene una prioridad que define el orden en que los servidores deben ser utilizados: primero el de menor valor numérico.

Registro SPF.

Es un mecanismo de autenticación de correo electrónico que especifica qué servidores pueden enviar mensajes en nombre de un dominio. Funciona como un registro DNS de tipo TXT que lista las direcciones IP y dominios autorizados.

Regulación de Datos (GDPR).

Es la normativa de la Unión Europea sobre protección de datos personales, conocida como Reglamento General de Protección de Datos (General Data Protection Regulation).

Reingeniería de Procesos.

Es la revisión y rediseño radical de procesos de negocio para lograr mejoras significativas en productividad, calidad y eficiencia. Se apoya en las TIC para automatizar y transformar tareas.

Reinicio en Caliente (Hot Restart).

Es la capacidad de reiniciar un sistema, aplicación o servicio sin apagar completamente el hardware o interrumpir a los usuarios. Es común en servidores, dispositivos de red y aplicaciones críticas donde la continuidad del servicio es esencial.

Reinicio en Frío (Cold Boot).

Se refiere al arranque de un sistema desde un estado de apagado completo. Ocurre cuando el equipo se enciende desde cero, cargando el firmware (BIOS/UEFI), inicializando hardware y cargando el sistema operativo.

Repositorio de Código.

Es un espacio centralizado donde se almacena, organiza y versiona el código fuente de proyectos de software. Puede estar en servidores locales o en plataformas en la nube como GitHub, GitLab o Bitbucket.

Repositorio Git.

Es un tipo de repositorio específico que utiliza el sistema de control de versiones Git. Permite gestionar cambios en el código fuente, trabajar en ramas, fusionar desarrollos y mantener un historial detallado.

Repositorio Maven.

Es un almacén de dependencias para proyectos basados en Java que utilizan Apache Maven como herramienta de gestión y construcción.

Repositorio de Software.

Es un lugar donde se almacenan y distribuyen paquetes de software, librerías y aplicaciones. Puede ser público (ejemplo: PyPI para Python, npm para JavaScript, apt en Linux) o privado en entornos empresariales.

Reproducción en Streaming.

Es la transmisión de contenido multimedia (audio, video, juegos) en tiempo real a través de Internet, sin necesidad de descargarlo por completo. Plataformas como Netflix, Spotify o Twitch utilizan esta tecnología.

Requerimientos Funcionales.

Son especificaciones que describen las funciones y comportamientos que un sistema de software debe realizar. Incluyen entradas, procesos, salidas e interacciones con usuarios o con otros sistemas.

Requerimientos No Funcionales.

Son condiciones que definen cómo debe comportarse un sistema, en lugar de lo que debe hacer. Incluyen rendimiento, seguridad, escalabilidad, disponibilidad y usabilidad.

Resiliencia en TI.

Es la capacidad de los sistemas tecnológicos para resistir, adaptarse y recuperarse rápidamente de fallos, ataques o interrupciones. Incluye redundancia, planes de contingencia, copias de seguridad y monitoreo constante.

Residencia de Datos.

Hace referencia a la ubicación física o geográfica donde se almacenan los datos en entornos locales o en la nube. Tiene importancia legal y de cumplimiento, ya que diferentes países imponen regulaciones sobre dónde pueden residir los datos.

Resolución de Pantalla.

Es la cantidad de píxeles que puede mostrar un dispositivo de visualización en ancho y alto, expresada en formato “ancho × alto” (ejemplo: 1920×1080 píxeles, Full HD).

Resource Allocation en Cloud.

Es la asignación y distribución de recursos informáticos (CPU, memoria, almacenamiento, red) en entornos de computación en la nube. Puede hacerse de manera estática o dinámica, según la demanda del usuario.

REST API.

Es una interfaz de programación de aplicaciones que sigue los principios arquitectónicos REST (Representational State Transfer). Permite la comunicación entre cliente y servidor mediante operaciones estándar HTTP (GET, POST, PUT, DELETE).

RESTful Services.

Son servicios web que implementan las reglas de la arquitectura REST. Se caracterizan por ser sin estado (stateless), usar URLs como identificadores de recursos y soportar múltiples formatos (JSON, XML).

Restauración de Sistemas.

Es el proceso de devolver un sistema a un estado funcional anterior tras un fallo, ataque o error. Puede realizarse mediante copias de seguridad, puntos de restauración o snapshots en máquinas virtuales.

Retargeting Digital.

Es una estrategia de marketing que muestra anuncios personalizados a usuarios que previamente visitaron un sitio web o interactuaron con una marca. Utiliza cookies, píxeles de seguimiento y plataformas de publicidad.

Retroalimentación del Usuario.

Es la información que los usuarios proporcionan sobre el desempeño, usabilidad y calidad de un sistema o aplicación. Puede recibirse mediante encuestas, métricas de uso o pruebas beta.

Retrocompatibilidad.

Es la capacidad de un sistema o software para ejecutar aplicaciones o soportar archivos de versiones anteriores. Ejemplo: una consola de videojuegos que permite jugar títulos de la generación pasada.

Reverse Engineering.

Es el análisis de un sistema, programa o hardware con el fin de entender su diseño, funcionamiento o estructura interna. Se utiliza para depuración, interoperabilidad, auditorías de seguridad y recuperación de software sin documentación.

RFID (Radio Frequency Identification).

Es una tecnología que utiliza ondas de radio para identificar y rastrear objetos mediante etiquetas electrónicas (tags). A diferencia de los códigos de barras, no requiere contacto visual. Se usa en logística, control de inventarios, transporte y seguridad.

RGB.

Es un modelo de color aditivo basado en la combinación de los tres colores primarios de la luz: rojo, verde y azul. Se utiliza en pantallas, cámaras, gráficos digitales y diseño web.

Rich Text Format (RTF).

Es un formato de archivo desarrollado por Microsoft que permite almacenar texto enriquecido con estilos como negritas, cursivas, colores y tablas. Es multiplataforma y puede ser leído por diversos procesadores de texto.

Riesgo Cibernético.

Es la posibilidad de que una organización o individuo sufra daños por amenazas digitales como ataques de malware, robo de información, fraudes o interrupciones de servicios.

RISC (Reduced Instruction Set Computer).

Es una arquitectura de procesadores basada en un conjunto reducido y optimizado de instrucciones simples. Cada instrucción se ejecuta en un ciclo de reloj, lo que aumenta la eficiencia y el rendimiento.

RMM (Remote Monitoring and Management).

Es una tecnología que permite supervisar y administrar sistemas informáticos a distancia. Es ampliamente usada por proveedores de servicios gestionados (MSP) para controlar servidores, redes y dispositivos de clientes.

Robótica Colaborativa (Cobot).

Se refiere a robots diseñados para trabajar junto a humanos en entornos compartidos. Son seguros, adaptables y fáciles de programar. A diferencia de los robots industriales tradicionales, no requieren cercas de seguridad.

Robótica en IA.

Es la integración de algoritmos de inteligencia artificial en robots, permitiéndoles aprender, adaptarse y tomar decisiones autónomas. Incluye visión por computadora, procesamiento de lenguaje natural y aprendizaje por refuerzo.

Robótica Industrial.

Es el uso de robots en procesos de manufactura y producción. Estos robots suelen ser grandes, precisos y especializados en tareas repetitivas como soldadura, ensamblaje, pintura o empaquetado.

Robotic Process Automation (RPA).

Es una tecnología que utiliza robots de software para automatizar tareas repetitivas y basadas en reglas dentro de aplicaciones digitales. Se emplea en finanzas, recursos humanos y atención al cliente para mejorar la eficiencia.

ROM (Read Only Memory).

Es un tipo de memoria no volátil que almacena datos de forma permanente, incluso cuando el dispositivo está apagado. Se utiliza para guardar firmware y programas esenciales para el arranque del sistema.

Root.

Es el usuario con privilegios administrativos completos en sistemas operativos basados en Unix/Linux. Tiene control total sobre archivos, procesos y configuraciones del sistema. Su uso indebido puede causar fallos graves o comprometer la seguridad.

Rootkit.

Es un tipo de malware diseñado para ocultar la presencia de otros programas maliciosos en un sistema. Modifica el sistema operativo para evadir la detección de antivirus y administradores. Puede otorgar acceso remoto a atacantes y persistir incluso tras reinicios.

Router.

Es un dispositivo de red que dirige el tráfico de datos entre diferentes redes, seleccionando la mejor ruta para los paquetes. Conecta redes locales (LAN) con redes externas como Internet.

Router Inteligente.

Es una versión avanzada de los routers tradicionales que incorpora funciones de gestión inteligente como priorización de tráfico, seguridad reforzada, control parental y optimización automática del Wi-Fi.

RSS (Really Simple Syndication).

Es un formato basado en XML que permite distribuir y recibir contenido actualizado de manera automática, como noticias, blogs o podcasts. Los usuarios se suscriben a fuentes RSS mediante lectores o aplicaciones.

RTC (Real-Time Clock).

Es un componente de hardware que mantiene la fecha y hora actual incluso cuando el dispositivo está apagado, gracias a una batería integrada. Se utiliza en computadoras, servidores y sistemas embebidos.

RTC (Real-Time Communication).

Es la transmisión de voz, video o datos en tiempo real entre dispositivos a través de Internet. Se utiliza en aplicaciones como videollamadas, mensajería instantánea y juegos en línea.

Ruby.

Es un lenguaje de programación interpretado, orientado a objetos y de alto nivel, creado por Yukihiro Matsumoto en 1995. Su filosofía prioriza la simplicidad y la productividad del desarrollador.

untime Environment.

Es el conjunto de componentes de software que permiten que un programa funcione, incluyendo librerías, intérpretes y configuraciones del sistema.

Renderizado.

Es el proceso de generar una imagen, video o interfaz visual a partir de datos o código. En desarrollo web, se refiere a cómo el navegador interpreta HTML, CSS y JavaScript para mostrar una página.

SaaS (Software as a Service).

Es un modelo de distribución de software en la nube donde las aplicaciones se alojan en servidores externos y los usuarios acceden a ellas a través de Internet, normalmente mediante suscripción.

Safari.

Es el navegador web desarrollado por Apple, introducido en 2003. Está diseñado para sistemas macOS y iOS, con integración profunda en el ecosistema Apple. Se caracteriza por su eficiencia energética, motor de renderizado WebKit y soporte de estándares web modernos.

Salto de Dirección (Jump Instruction).

En ensamblador, una instrucción de salto permite alterar el flujo de ejecución de un programa, dirigiéndolo a una dirección específica de memoria. Puede ser incondicional (ej. JMP) o condicional, dependiendo del resultado de una operación lógica o aritmética.

Samba.

Es un software libre que implementa el protocolo SMB (Server Message Block), permitiendo compartir archivos e impresoras entre sistemas Linux/Unix y Windows. Facilita la interoperabilidad en entornos mixtos de red y puede actuar como controlador de dominio en Active Directory.

Sandbox.

Es un entorno de ejecución aislado y controlado que permite probar o ejecutar aplicaciones sin que afecten al sistema principal. Se utiliza en desarrollo de software, ciberseguridad y análisis de malware.

SAP.

Es una de las principales suites de software empresarial (ERP), diseñada para gestionar procesos de negocio como finanzas, logística, recursos humanos y producción. Fundada en Alemania, SAP ofrece soluciones modulares altamente integradas.

SATA (Serial ATA).

Es una interfaz de conexión usada para discos duros y unidades SSD, reemplazando al antiguo estándar PATA. Introducida en 2003, ofrece mayor velocidad de transferencia, cables más delgados y soporte para hot-swapping.

Scala.

Es un lenguaje de programación híbrido, que combina paradigmas de orientación a objetos y programación funcional. Ejecuta sobre la JVM (Java Virtual Machine), lo que le da compatibilidad con el ecosistema Java.

Scanner 3D.

Es un dispositivo de hardware que digitaliza objetos físicos para generar un modelo tridimensional. Funciona mediante técnicas como luz estructurada, láser o fotogrametría. Estos modelos se usan en diseño industrial, medicina, arqueología y animación.

Schema en Bases de Datos.

Es la estructura lógica que organiza y define los objetos dentro de una base de datos, como tablas, vistas, índices y relaciones. Actúa como un contenedor que establece cómo se almacenan y se relacionan los datos.

Schema XML.

Es un conjunto de reglas escritas en un lenguaje de definición (XSD) que especifica la estructura, tipos de datos y restricciones de un documento XML. Su propósito es validar que un archivo XML cumpla con un formato predefinido.

Scrum.

Es un marco de trabajo ágil para la gestión y desarrollo de proyectos complejos. Se basa en ciclos cortos llamados sprints y en la colaboración continua entre los miembros del equipo.

Scrum Master.

Es el rol dentro de Scrum encargado de asegurar que el equipo siga los principios y prácticas del marco ágil. Actúa como facilitador, elimina obstáculos y protege al equipo de distracciones externas. No gestiona al equipo, sino que fomenta la autoorganización.

SDK (Software Development Kit).

Es un conjunto de herramientas de desarrollo que incluye bibliotecas, documentación, ejemplos de código y entornos de prueba para crear aplicaciones específicas.

SDR (Software Defined Radio).

Es un sistema de radio en el que funciones tradicionalmente implementadas en hardware, como modulación y filtrado, se realizan mediante software. Esto brinda flexibilidad para soportar múltiples estándares de comunicación en un mismo dispositivo.

SDSL (Symmetric Digital Subscriber Line).

Es una tecnología de transmisión de datos que ofrece la misma velocidad tanto en descarga como en subida. Utiliza líneas telefónicas de cobre, pero no permite simultaneidad con llamadas de voz.

Secure Boot.

Es un mecanismo de seguridad en el proceso de arranque de un sistema operativo que garantiza que solo se cargue software verificado y firmado digitalmente. Su objetivo es impedir la ejecución de malware a nivel de arranque, como rootkits.

Secure Socket Layer (SSL).

Es un protocolo criptográfico diseñado para proporcionar comunicaciones seguras a través de Internet. Funciona mediante el cifrado de los datos transmitidos entre cliente y servidor.

SSL Certificate.

Es un archivo digital que autentica la identidad de un sitio web y habilita el cifrado de las comunicaciones mediante HTTPS. Es emitido por una Autoridad de Certificación (CA) confiable.

Seguridad Informática.

Es el conjunto de prácticas, políticas, tecnologías y procesos diseñados para proteger sistemas informáticos, redes y datos frente a accesos no autorizados, ciberataques y daños.

Servidor Dedicado.

Es un equipo físico completo destinado exclusivamente a un cliente, aplicación o servicio. Ofrece control total sobre hardware y software, así como mayor rendimiento y seguridad que soluciones compartidas. .¿

Servidor DNS.

Es un sistema que traduce nombres de dominio legibles por humanos (ej. www.ejemplo.com) en direcciones IP numéricas utilizadas por las computadoras. Actúa como una guía telefónica de Internet, permitiendo que los usuarios accedan a servicios web de forma sencilla. Es esencial para el funcionamiento de la red global.

Servidor Proxy.

Es un intermediario entre un cliente y un servidor de destino. Su función principal es reenviar solicitudes y respuestas, pudiendo además ofrecer anonimato, control de accesos, filtrado de contenido y optimización de tráfico.

Servidor Virtual Privado (VPS).

Es un entorno de servidor virtualizado que simula el comportamiento de un servidor físico dentro de un hardware compartido. Ofrece independencia en configuración, recursos dedicados y escalabilidad a un costo más bajo que un servidor dedicado.

Service Mesh.

Es una capa de infraestructura que gestiona la comunicación entre microservicios dentro de una arquitectura distribuida. Proporciona seguridad, balanceo de carga, monitoreo y control del tráfico sin modificar el código de las aplicaciones.

Shader.

Es un programa especializado en gráficos que se ejecuta en la GPU para controlar cómo se procesan píxeles, vértices y efectos visuales.

SharePoint.

Es una plataforma de colaboración desarrollada por Microsoft que facilita la gestión documental, el trabajo en equipo y la automatización de flujos de trabajo.

Shell.

Es una interfaz que permite al usuario interactuar con el sistema operativo mediante comandos. Puede ser de tipo textual (CLI) o gráfico (GUI). En sistemas Unix/Linux destacan Bash y Zsh, mientras que en Windows existe PowerShell.

SIEM.

Es una solución de seguridad que recopila, correlaciona y analiza en tiempo real los eventos generados por sistemas, aplicaciones y dispositivos de red.

Simulación por Computadora.

Es la creación de modelos digitales que representan sistemas reales para estudiar su comportamiento bajo diferentes condiciones. Se utiliza en ingeniería, medicina, meteorología, economía y ciencias sociales.

Smart Contracts.

Son programas autoejecutables almacenados en una blockchain que cumplen automáticamente las condiciones establecidas en su código. Garantizan transparencia, inmutabilidad y eliminación de intermediarios en transacciones.

Smart Grid.

Es una red eléctrica digital inteligente que integra tecnologías de comunicación y control para optimizar la generación, distribución y consumo de energía.

Smart Home.

Es un sistema de hogar conectado que integra dispositivos inteligentes como luces, cámaras, termostatos y electrodomésticos, controlados de forma remota mediante aplicaciones o asistentes virtuales.

SMTP Relay.

Es un servicio o servidor que retransmite correos electrónicos desde un remitente hasta el servidor de destino. Se utiliza para asegurar la entrega de correos masivos, boletines o notificaciones de aplicaciones.

Sniffer de Red.

Es una herramienta de software o hardware que captura y analiza el tráfico de red en tiempo real. Se utiliza en administración de sistemas para diagnóstico y en ciberseguridad para detección de intrusiones.

Software Embebido.

Es el software diseñado específicamente para controlar dispositivos de hardware dedicados. Está optimizado para funcionar en recursos limitados, como microcontroladores, sensores y electrodomésticos inteligentes.

Software Libre.

Es aquel que garantiza a los usuarios las libertades de uso, estudio, modificación y distribución. Su filosofía se basa en la transparencia y la colaboración, permitiendo acceder al código fuente. Ejemplos destacados son Linux, LibreOffice y GIMP.

Software Propietario.

Es aquel cuyo código fuente está restringido y bajo control exclusivo de su autor o empresa. Generalmente requiere licencia de pago y limita las modificaciones, redistribución o uso fuera de los términos establecidos.

Software RAID.

Es la implementación de la tecnología RAID mediante software en lugar de controladores de hardware dedicados. Permite crear arreglos de discos para redundancia o rendimiento utilizando el sistema operativo.

Spam Bot.

Es un programa automatizado que envía grandes cantidades de mensajes basura, principalmente a correos electrónicos, foros o redes sociales. Se utiliza en campañas de publicidad invasiva, distribución de malware o fraudes.

SQL Injection.

Es una vulnerabilidad de seguridad en aplicaciones que interactúan con bases de datos SQL. Permite a un atacante insertar o manipular consultas maliciosas a través de formularios o parámetros de entrada.

Standardization (Estandarización).

Es el proceso de establecer normas y criterios comunes para tecnologías, sistemas y procesos. Busca garantizar compatibilidad, interoperabilidad y calidad en productos y servicios.

Stateful Firewall.

Es un tipo de cortafuegos que monitorea activamente el estado de las conexiones de red y permite o bloquea el tráfico en función de reglas y contexto. A diferencia de un firewall estático, analiza el flujo completo de paquetes, lo que mejora la seguridad contra ataques sofisticados.

Stateless Firewall.

Es un cortafuegos que filtra paquetes de red basándose únicamente en reglas predefinidas, sin tener en cuenta el estado de las conexiones. Su funcionamiento es más rápido y ligero que un stateful firewall, pero menos seguro frente a ataques complejos como DoS o tráfico fragmentado.

Storage Area Network (SAN).

Es una red de alta velocidad dedicada al almacenamiento de datos. Conecta servidores a dispositivos de almacenamiento centralizados, mejorando el rendimiento y la escalabilidad.

Structured Data.

Son datos organizados en un formato predefinido, como tablas en bases de datos relacionales. Facilitan la búsqueda, análisis y almacenamiento mediante lenguajes como SQL.

Subnet (Subred).

Es una división lógica dentro de una red IP que organiza los dispositivos en segmentos más pequeños. Mejora la seguridad, administración y eficiencia en el enrutamiento.

Supply Chain Attack.

Es un tipo de ciberataque que compromete proveedores o terceros de confianza para infiltrarse en la cadena de suministro tecnológica de una organización. Los atacantes aprovechan software, actualizaciones o servicios legítimos para propagar malware.

Switch de Red.

Es un dispositivo de hardware que conecta múltiples equipos dentro de una red local (LAN) y gestiona el tráfico de datos enviando paquetes solo al destinatario correspondiente. A diferencia de un hub, optimiza el rendimiento y reduce colisiones.

Symbolic AI.

Es un enfoque de la inteligencia artificial basado en reglas lógicas, símbolos y conocimiento explícito. Se centra en la representación del razonamiento humano mediante sistemas expertos.

Syn Flood.

Es un tipo de ataque de denegación de servicio (DoS) que explota el protocolo TCP. El atacante envía múltiples solicitudes de conexión (SYN) a un servidor, pero nunca completa el proceso con el mensaje ACK.

Sysadmin./h2>

Es el profesional encargado de instalar, configurar, administrar y mantener los sistemas informáticos de una organización. Su trabajo abarca servidores, redes, seguridad, copias de respaldo y actualizaciones.

T-SQL (Transact-SQL).

Es una extensión del lenguaje SQL desarrollada por Microsoft para sus bases de datos SQL Server. Incluye características adicionales como manejo de transacciones, control de flujo con sentencias condicionales, procedimientos almacenados y funciones definidas por el usuario.

Tabla Hash.

Es una estructura de datos que organiza información en pares clave-valor, utilizando una función hash para determinar la posición de cada elemento. Su principal ventaja es la rapidez en operaciones de búsqueda, inserción y eliminación, que en promedio ocurren en tiempo constante (O(1)).

Tablet.

Es un dispositivo electrónico portátil con pantalla táctil de tamaño intermedio entre un teléfono inteligente y un ordenador portátil. Permite consumir contenido multimedia, navegar por internet, trabajar con aplicaciones de productividad y realizar actividades educativas o creativas.

Tag HTML.

Es un elemento del lenguaje de marcado HTML utilizado para estructurar y presentar contenido en la web. Existen etiquetas de apertura y cierre, como

...

para párrafos, y etiquetas auto-contenidas, como para imágenes.

TeamViewer.

Es un software multiplataforma que permite la conexión remota y el control de dispositivos a través de internet. Se utiliza ampliamente en soporte técnico, administración de servidores y colaboración en tiempo real.

Tecnología Blockchain.

Es un sistema de registro descentralizado e inmutable que almacena información en bloques enlazados criptográficamente. Cada bloque contiene un conjunto de transacciones verificadas por una red de nodos, eliminando la necesidad de intermediarios.

Tecnología NFC (Near Field Communication).

Es una tecnología de comunicación inalámbrica de corto alcance (hasta 10 cm) que permite el intercambio de datos entre dispositivos compatibles. Se utiliza principalmente en pagos móviles, tarjetas de transporte, sistemas de autenticación y dispositivos IoT.

Tecnología OCR (Optical Character Recognition).

Es una tecnología que permite convertir imágenes de texto impreso o manuscrito en datos digitales editables y procesables.

Tecnología OLED.

Es una tecnología de visualización basada en diodos orgánicos emisores de luz. Cada píxel emite su propia iluminación, lo que permite negros profundos, mayor contraste y pantallas ultradelgadas.

Tecnología Quantum Dot.

Consiste en el uso de nanocristales semiconductores que emiten colores muy precisos cuando reciben luz. Se aplica principalmente en televisores y monitores para mejorar la calidad de imagen.

Tecnología RFID.

Identificación por radiofrecuencia que utiliza etiquetas y lectores para transmitir datos sin contacto. Se emplea en logística, control de inventarios, transporte y sistemas de seguridad.

Tecnología VoIP (Voice over IP).

Permite transmitir voz a través de internet en lugar de redes telefónicas tradicionales. Funciona convirtiendo la señal de voz en paquetes de datos digitales.

Telnet.

Protocolo de red que permite la comunicación remota con otro dispositivo mediante texto plano. Fue muy usado en la administración de servidores y equipos de red.

TensorFlow Lite.

Es una versión ligera del framework TensorFlow diseñada para dispositivos móviles y embebidos. Permite ejecutar modelos de machine learning optimizados en smartphones, IoT y microcontroladores. Reduce el consumo de recursos y mantiene buena precisión.

Terminal Linux.

Interfaz de línea de comandos que permite a los usuarios interactuar con el sistema operativo Linux. A través de comandos, se gestionan archivos, procesos, redes y configuraciones del sistema.

Terraform.

Herramienta de infraestructura como código desarrollado por HashiCorp. Permite definir, aprovisionar y gestionar recursos en la nube de forma automatizada.

Test A/B.

Es una técnica experimental usada para comparar dos versiones de un producto, interfaz o campaña. Se divide la audiencia en dos grupos, cada uno expuesto a una variante.

Testing de Integración.

Consiste en validar que diferentes módulos o componentes de software funcionen correctamente al combinarse. Detecta errores de comunicación entre subsistemas. Se puede realizar de manera incremental o en un solo conjunto.

Testing de Rendimiento.

Evalúa la velocidad, capacidad de respuesta y estabilidad de un sistema bajo diferentes cargas de trabajo. Incluye pruebas de estrés, volumen y escalabilidad.

Thunderbolt.

Es una interfaz de hardware desarrollada por Intel y Apple que combina transmisión de datos, video y energía. Ofrece altas velocidades (hasta 40 Gbps en versiones recientes). Se utiliza en dispositivos de almacenamiento, monitores y estaciones de trabajo.

Tiempo de Respuesta.

Es el tiempo que tarda un sistema en procesar una solicitud y devolver un resultado. Es un indicador clave de rendimiento en aplicaciones y redes. Un tiempo bajo mejora la experiencia de usuario y la eficiencia operativa.

Transistor.

Es un componente electrónico que controla el flujo de corriente en un circuito. Actúa como interruptor o amplificador, siendo la base de los microprocesadores modernos. Su invención revolucionó la electrónica al sustituir a las válvulas de vacío.

Transporte de Capa (OSI).

Es la cuarta capa del modelo OSI, responsable de garantizar la entrega confiable de datos entre aplicaciones. Ofrece servicios como control de flujo, corrección de errores y segmentación.

Trojan (Troyano).

Es un tipo de malware que se disfraza de software legítimo para infectar un sistema. Una vez activo, permite el acceso remoto al atacante o roba información confidencial. Suele propagarse a través de descargas no seguras o correos electrónicos maliciosos.

Tunelización (Tunneling).

Es el proceso de encapsular datos dentro de otro protocolo para transmitirlos de forma segura o compatible. Se utiliza en redes privadas virtuales (VPN) y en transiciones de IPv4 a IPv6.

Unidad de Medida DPI.

Significa “dots per inch” o puntos por pulgada, y mide la resolución de impresión o pantallas. Un valor mayor de DPI implica más detalle y nitidez.

Unidad de Red Mapeada.

Es una carpeta o recurso compartido en red que se presenta como una unidad de disco local. Facilita el acceso a archivos y datos de servidores en entornos corporativos.

URL (Uniform Resource Locator).

Es la dirección única que identifica recursos en internet, como páginas web, imágenes o archivos. Está compuesta por protocolo, dominio y ruta del recurso.

Validación de Datos.

La validación de datos es el proceso mediante el cual se verifican la exactitud, coherencia y formato de los datos ingresados en un sistema.

Varias Capas de Seguridad.

La estrategia de varias capas de seguridad, también llamada "defense in depth", consiste en implementar múltiples barreras para proteger sistemas informáticos. Incluye medidas como firewalls, antivirus, autenticación multifactor, cifrado y monitoreo constante.

Vector de Ataque.

Un vector de ataque es el método o camino utilizado por un atacante para vulnerar un sistema, red o aplicación. Puede incluir técnicas como phishing, malware, explotación de vulnerabilidades o ataques de fuerza bruta.

Versionado Semántico.

El versionado semántico (SemVer) es un sistema estandarizado para asignar números de versión a software, siguiendo el formato Mayor.Menor.Parche (X.Y.Z).

Virtual Desktop Infrastructure (VDI).

VDI es una tecnología que permite ejecutar escritorios virtuales en un servidor centralizado y acceder a ellos de forma remota. Ofrece ventajas como seguridad de datos, administración centralizada y reducción de costos en hardware.

Virtual Machine (VM).

Una máquina virtual es una emulación de un sistema operativo y hardware dentro de otro sistema anfitrión. Permite ejecutar varios sistemas en paralelo, cada uno aislado del otro, con su propio entorno de ejecución.

Virtualización de Servidores.

La virtualización de servidores consiste en dividir un servidor físico en múltiples servidores virtuales independientes mediante software. Esto mejora la utilización de recursos, reduce costos de hardware y facilita la administración.

Virus Informático.

Un virus informático es un programa malicioso diseñado para alterar el funcionamiento normal de un sistema. Se propaga al insertarse en archivos ejecutables o documentos, pudiendo dañar datos, robar información o ralentizar equipos.

Visual Studio Code.

Visual Studio Code (VS Code) es un editor de código fuente gratuito y multiplataforma desarrollado por Microsoft. Soporta múltiples lenguajes mediante extensiones, ofrece depuración integrada y control de versiones con Git.

Visualización de Datos.

La visualización de datos es el proceso de representar información en gráficos, diagramas o dashboards para facilitar su análisis. Permite identificar patrones, tendencias y anomalías de manera más intuitiva que con datos en bruto.

VPN (Virtual Private Network).

Una VPN es una red privada virtual que cifra la comunicación entre un dispositivo y un servidor remoto, protegiendo la privacidad en internet. Permite acceder de forma segura a redes corporativas, ocultar la dirección IP y evitar rastreos.

VRML (Virtual Reality Modeling Language).

VRML es un lenguaje de modelado que permite crear y describir entornos y objetos tridimensionales interactivos en la web. Fue uno de los primeros intentos de estandarizar la realidad virtual en internet en los años 90.

Vulnerabilidad Cero Día.

Una vulnerabilidad cero día es una falla de seguridad desconocida por el fabricante del software o hardware. Al no existir parches ni soluciones oficiales, los atacantes pueden explotarla de inmediato con gran efectividad.

Webhook / Webhooks.

Un webhook es un método que permite a una aplicación enviar información en tiempo real a otra aplicación mediante HTTP. A diferencia de las APIs tradicionales que requieren consultas periódicas, los webhooks transmiten datos de manera inmediata cuando ocurre un evento.

WiMAX.

WiMAX es una tecnología de telecomunicaciones que proporciona acceso a internet de banda ancha inalámbrica a larga distancia.

Windows.

Windows es un sistema operativo desarrollado por Microsoft, ampliamente usado en computadoras personales, servidores y dispositivos móviles en el pasado.

Word.

Microsoft Word es un procesador de texto desarrollado por Microsoft, parte del paquete Office. Permite crear, editar y dar formato a documentos, integrando herramientas de corrección, colaboración y diseño.

XML (Extensible Markup Language).

Lenguaje de marcado estándar definido por el W3C para estructurar y almacenar datos de manera legible tanto por humanos como por máquinas.

YUV (Formato de color).

Es un modelo de representación de color usado en procesamiento de video digital. Separa la información en un componente de luminancia (Y) y dos de crominancia (U y V), lo que permite reducir ancho de banda sin perder calidad visual percibida.

Zabbix.

Es una plataforma de monitoreo de código abierto que permite supervisar redes, servidores, aplicaciones y servicios en tiempo real. Ofrece alertas proactivas, paneles personalizables y compatibilidad con distintos protocolos como SNMP y JMX. Es muy usado en empresas por su escalabilidad y porque admite tanto infraestructura física como virtual.

Zapier.

Es una herramienta de automatización en la nube que conecta aplicaciones y servicios web sin necesidad de programar. Funciona mediante “Zaps”, flujos de trabajo automáticos que se activan por un evento y ejecutan una acción en otra aplicación.

Zappa.

Framework serverless para Python que facilita el despliegue de aplicaciones en AWS Lambda y API Gateway. Permite transformar proyectos en microservicios sin necesidad de gestionar servidores, mejorando la escalabilidad y reduciendo costos.

ZBrush.

Software de escultura y modelado digital 3D desarrollado por Pixologic (ahora parte de Maxon). Destaca por su motor de “pixol” que guarda información de color, profundidad e iluminación de cada píxel.

ZCache.

Tecnología del kernel de Linux que mejora la eficiencia de la memoria mediante compresión de páginas RAM. Almacena más información en menos espacio, reduciendo la necesidad de usar swap en disco.

Zombie botnet.

Red de dispositivos comprometidos (computadoras, IoT, servidores) controlados de forma remota por un atacante. Se utilizan para lanzar ataques distribuidos como DDoS, envío masivo de spam o minería de criptomonedas.

Zope.

Servidor de aplicaciones web de código abierto escrito en Python. Fue uno de los primeros frameworks orientados a objetos en la web, destacando por su flexibilidad y soporte para desarrollo rápido.

Zorin OS.

Distribución de Linux diseñada para ser amigable con principiantes y usuarios que migran desde Windows o macOS. Su interfaz gráfica está basada en GNOME, pero adaptada para ofrecer una experiencia similar a otros sistemas.

Zotero.

Herramienta gratuita para la gestión de referencias bibliográficas y citas académicas. Permite organizar fuentes, generar bibliografías y sincronizar colecciones en la nube.

Zoom.

Plataforma de videoconferencias en línea ampliamente usada en educación, empresas y comunicación remota. Ofrece funciones de chat, grabación de reuniones, compartición de pantalla y colaboración en tiempo real.

Zookeeper.

Servicio centralizado de coordinación para aplicaciones distribuidas, desarrollado dentro del ecosistema Hadoop. Gestiona la configuración, sincronización y nombres de servicios en clústeres.

Zune.

Reproductor multimedia desarrollado por Microsoft y lanzado en 2006 como competencia del iPod. Incluía integración con una tienda de música en línea y sincronización con PC.

Zynq.

Familia de SoC (System on Chip) de Xilinx que combina procesadores ARM Cortex con lógica programable FPGA. Ofrece gran flexibilidad para aplicaciones embebidas, inteligencia artificial, visión por computadora y telecomunicaciones. Permite diseñar sistemas personalizados de alto rendimiento con bajo consumo.
